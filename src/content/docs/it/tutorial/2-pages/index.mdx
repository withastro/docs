Ora che hai un sito funzionante sul web, è il momento di aggiungere pagine e articoli!

## Guardando avanti

In questa unità, farai:
- Creare le tue prime pagine Astro con la sintassi `.astro`
- Aggiungere articoli del blog con file Markdown (`.md`)
- Stilizzare una singola pagina con `<style>`
- Applicare stili globali attraverso le pagine

 Lungo il percorso, imparerai come le **due sezioni di un file `.astro`**  lavorano insieme per creare una pagina e come usare variabili e rendering condizionale sulle tue pagine.
