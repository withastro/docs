Ora che il tuo sito è attivo sul web, è il momento di aggiungere pagine e post!

## Cosa ti aspetta

In questa unità, imparerai a:
- Creare le tue prime pagine Astro con la sintassi `.astro`
- Aggiungere post al blog con file Markdown (`.md`)
- Stilizzare una pagina individuale con `<style>`
- Applicare stili globali su più pagine

Durante il percorso, scoprirai come le **due sezioni di un file `.astro`** lavorano insieme per creare una pagina, e come utilizzare variabili e rendering condizionale nelle tue pagine.
