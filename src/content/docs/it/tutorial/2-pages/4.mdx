## Stila una pagina singola

Usando il tag `<style></style>` di Astro, puoi stilizzare gli elementi della tua pagina. Aggiungere **attributi** e **direttive** a questi tag ti dà ancora più possibilità di stilizzare.

<Steps>
1. Copia il seguente codice e incollalo in `src/pages/about.astro`:

    ```astro title="src/pages/about.astro" ins={6-11}
    <html lang="en">
      <head>
        <meta charset="utf-8" />
        <meta name="viewport" content="width=device-width" />
        <title>{pageTitle}</title>
        <style>
          h1 {
            color: purple;
            font-size: 4rem;
          }
        </style>
      </head> 
    ```

    Controlla tutte e tre le pagine nella tua anteprima nel browser.
    
    - Di che colore è il titolo della pagina:

        - La tua Home page?  <Spoiler>nero</Spoiler>
        - La tua About page? <Spoiler>viola</Spoiler>
        - La tua Blog page? <Spoiler>nero</Spoiler>

    - La pagina con il titolo più grande è? <Spoiler>La tua About page</Spoiler>

    :::tip
    Se non riesci a determinare i colori visivamente, puoi usare gli strumenti per sviluppatori nel tuo browser per ispezionare gli elementi `<h1>` e verificare il colore del testo applicato.
    :::

2. Aggiungi il nome della classe `skill` agli elementi `<li>` generati nella tua About page, così potrai stilizzarli. Il tuo codice dovrebbe apparire ora così:

    ```astro title="src/pages/about.astro" 'class="skill"'
    <p>Le mie competenze sono:</p>
    <ul>
      {skills.map((skill) => <li class="skill">{skill}</li>)}
    </ul>
    ```

3. Aggiungi il seguente codice al tuo tag `<style>` esistente:

    ```astro title="src/pages/about.astro" ins={6-9}
    <style>
      h1 {
        color: purple;
        font-size: 4rem;
      }
      .skill {
        color: green;
        font-weight: bold;
      }
    </style>
    ```

4. Visita di nuovo la tua About page nel browser e verifica, attraverso l'ispezione visiva o gli strumenti per sviluppatori, che ogni elemento nella lista delle tue competenze sia ora di colore verde e in grassetto.
</Steps>

## Usa la tua prima variabile CSS
Il tag `<style>` di Astro può anche fare riferimento a qualsiasi variabile definita nel tuo script di frontmatter usando la direttiva `define:vars={ {...} }`. Puoi **definire variabili all'interno del tuo codice** e poi **usarle come variabili CSS nel tuo tag `<style>`**.

<Steps>
1. Definisci una variabile `skillColor` aggiungendola allo script di frontmatter di `src/pages/about.astro` così:

    ```astro title="src/pages/about.astro" ins={17}
    ---
    const pageTitle = "About Me";

    const identity = {
      firstName: "Sarah",
      country: "Canada",
      occupation: "Technical Writer",
      hobbies: ["photography", "birdwatching", "baseball"],
    };

    const skills = ["HTML", "CSS", "JavaScript", "React", "Astro", "Writing Docs"];

    const happy = true;
    const finished = false;
    const goal = 3;
  
    const skillColor = "navy";
    ---
    ```

2. Aggiorna il tuo tag `<style>` esistente per prima definire, e poi usare questa variabile `skillColor` all'interno delle parentesi graffe doppie.

    ```astro title="src/pages/about.astro" "define:vars={{skillColor}}" "var(--skillColor)" del={7} ins={8}
    <style define:vars={{skillColor}}> 
      h1 {
        color: purple;
        font-size: 4rem;
      }
      .skill {
        color: green;
        color: var(--skillColor);
        font-weight: bold;
      }
    </style>
    ```

3. Controlla di nuovo la tua About page nel browser. Dovresti vedere che le competenze sono ora di colore blu navy, come definito dalla variabile `skillColor` passata alla direttiva `define:vars`.
</Steps>

## Prova tu stesso - Definisci variabili CSS
 
<Steps>
 1. Aggiorna il tag `<style>` nella tua About page affinché corrisponda a quello qui sotto:

    ```astro title="src/pages/about.astro"
    <style define:vars={{skillColor, fontWeight, textCase}}>
      h1 {
        color: purple;
        font-size: 4rem;
      }
      .skill {
        color: var(--skillColor);
        font-weight: var(--fontWeight);
        text-transform: var(--textCase);
      }
    </style>
    ```
 
 2. Aggiungi le definizioni di variabili mancanti nel tuo script di frontmatter affinché il tuo nuovo tag `<style>` applichi correttamente questi stili alla tua lista di competenze:
    - Il colore del testo è blu navy
    - Il testo è in grassetto
    - Gli elementi della lista sono in maiuscolo (tutte le lettere maiuscole)
</Steps>
