Ora che sai che i file `.astro` sono responsabili per le pagine del tuo sito web, è il momento di crearne uno!

<PreCheck>
  - Crea due nuove pagine nel tuo sito: About e Blog
  - Aggiungi i link di navigazione alle tue pagine
  - Distribuisci una versione aggiornata del tuo sito sul web
</PreCheck>

## Crea un nuovo file `.astro`

<Steps>
1. Nel pannello dei file del tuo editor di codice, naviga nella cartella `src/pages/`, dove vedrai il file esistente `index.astro`.

2. Nella stessa cartella, crea un nuovo file chiamato `about.astro`.

3. Copia, o riscrivi, il contenuto di `index.astro` nel tuo nuovo file `about.astro`.

    :::tip
    Il tuo editor potrebbe mostrare un cerchio bianco solido sull'etichetta della scheda per questo file. Questo significa che il file non è ancora salvato. Nel menu File in VS Code, abilita "Auto Save" e non dovrai più salvare manualmente i file.
    :::

4. Aggiungi `/about` alla fine dell'URL di anteprima del tuo sito nella barra degli indirizzi e verifica che la pagina venga caricata correttamente. (ad esempio `http://localhost:4321/about`)
</Steps>

In questo momento, la tua pagina "About" dovrebbe apparire esattamente come la prima pagina, ma la modificheremo!

## Modifica la tua pagina

Modifica il contenuto HTML per rendere questa pagina su di te.

Per cambiare o aggiungere altro contenuto alla tua pagina "About", aggiungi più tag di elementi HTML contenenti del testo. Puoi copiare e incollare il codice HTML qui sotto tra i tag esistenti `<body></body>`, oppure crearne uno tutto tuo.

```astro title="src/pages/about.astro" ins={3-8} del={2}
<body>
  <h1>Il mio sito Astro</h1>
  <h1>Chi sono</h1>
  <h2>... e il mio nuovo sito Astro!</h2>

  <p>Sto seguendo il tutorial introduttivo di Astro. Questa è la seconda pagina del mio sito, ed è la prima che ho costruito da solo!</p>

  <p>Questo sito si aggiornerà man mano che completerò più parti del tutorial, quindi continua a controllare per vedere come sta andando il mio percorso!</p>
</body>
```

Ora, visita di nuovo la tua pagina `/about` nella tua scheda del browser e dovresti vedere il contenuto aggiornato.

## Aggiungi i link di navigazione

Per facilitare la visualizzazione di tutte le tue pagine, aggiungi i link di navigazione HTML prima del tuo `<h1>` in cima a entrambe le tue pagine (`index.astro` e `about.astro`):

```astro title="src/pages/about.astro" ins={1-2}
<a href="/">Home</a>
<a href="/about/">Chi sono</a>

<h1>Chi sono</h1>
<h2>... e il mio nuovo sito Astro!</h2>
```

Verifica che puoi cliccare su questi link per spostarti avanti e indietro tra le pagine del tuo sito.

:::note
A differenza di molti framework, Astro utilizza i tradizionali elementi HTML `<a>` per navigare tra le pagine (chiamate anche _route_), con i tradizionali aggiornamenti di pagina.
:::

## Provalo tu stesso - Aggiungi una pagina Blog

Aggiungi una terza pagina `blog.astro` al tuo sito, seguendo [gli stessi passaggi di sopra](#crea-un-nuovo-file-astro).

(Non dimenticare di aggiungere un terzo link di navigazione a ogni pagina.)

<details>
<summary>Mostrami i passaggi.</summary>
<Steps>
1. Crea un nuovo file in `src/pages/blog.astro`.
2. Copia l'intero contenuto di `index.astro` e incollalo in `blog.astro`.
3. [Aggiungi un terzo link di navigazione](#aggiungi-i-link-di-navigazione) in cima a ogni pagina:
</Steps>

```astro title="src/pages/index.astro" ins={4}
<body>
  <a href="/">Home</a>
  <a href="/about/">Chi sono</a>
  <a href="/blog/">Blog</a>

  <h1>Il mio sito Astro</h1>
</body>
```
</details>
</Box>

Ora dovresti avere un sito web con tre pagine che si linkano l'una all'altra. È il momento di aggiungere un po' di contenuto alla pagina Blog.

Aggiorna il contenuto della pagina in `blog.astro` con:

```astro astro title="src/pages/blog.astro" ins={7-8} del={6}
<body>
  <a href="/">Home</a>
  <a href="/about/">Chi sono</a>
  <a href="/blog/">Blog</a>

  <h1>Il mio sito Astro</h1>
  <h1>Il mio blog di apprendimento Astro</h1>
  <p>Qui scriverò del mio percorso nell'apprendere Astro.</p>
</body>
```

Visualizza in anteprima l'intero sito visitando tutte e tre le pagine nella tua anteprima del browser e verifica che:
- Ogni pagina linka correttamente a tutte e tre le pagine
- Le tue due nuove pagine hanno ciascuna un proprio titolo descrittivo
- Le tue due nuove pagine hanno ciascuna un paragrafo di testo

## Pubblica le tue modifiche sul web
Se hai seguito la configurazione nell'Unità 1, puoi pubblicare le tue modifiche sul tuo sito live tramite Netlify.

Quando sei soddisfatto dell'aspetto della tua anteprima, **esegui il commit** delle modifiche nel tuo repository online su GitHub. 

<Steps>
1. In VS Code, visualizza i file che sono cambiati dall'ultimo commit su GitHub. 

    - Vai nella **scheda Controllo Sorgente** nel menu a sinistra. Dovrebbe essere visualizzato un piccolo "3".

    - Dovresti vedere `index.astro`, `about.astro` e `blog.astro` elencati come file cambiati.

2. Inserisci un messaggio di commit (ad esempio "Aggiunte due nuove pagine - about e blog") nella casella di testo e premi <kbd>Ctrl + Enter</kbd> (macOS: <kbd>Cmd ⌘ + Enter</kbd>) per eseguire il commit del cambiamento nel tuo spazio di lavoro corrente.

3. Clicca il pulsante per <kbd>Sincronizzare le modifiche</kbd> su GitHub.

4. Dopo aver atteso qualche minuto, visita l'URL di Netlify per verificare che le tue modifiche siano pubblicate live.
</Steps>

:::tip[Commetti e distribuisci regolarmente]
Segui questi passaggi ogni volta che fai una pausa! Le tue modifiche saranno aggiornate nel tuo repository GitHub. Se hai distribuito il sito su Netlify, verrà ricostruito e ripubblicato.
:::
