## Come ottenere gli strumenti di sviluppo necessari

### Terminale

Userai un **terminale a linea di comando** per creare il tuo progetto Astro e per eseguire i comandi più importanti per assemblare, sviluppare e testare il tuo sito.

Potrai accedere alla linea di comando usando un programma cosiddetto "Terminale" per il tuo sistema operativo. I terminali più diffusi sono **Terminale** (MacOS/Linux), **Prompt dei comandi** (Windows) o **Termux** (Android). Probabilmente uno di questi è già presente sulla tua macchina.

### Node.js

Per eseguire Astro sul tuo computer devi avere installato [**Node.js**](https://nodejs.org/en/), versione `v16.12.0` o successive.

Per verificare se è già presente una versione compatibile sul tuo sistema, esegui il comando seguente nel terminale:

```sh
node -v

// Output di esempio
v16.14.0
```

Se il comando restituisce la versione `v16.12.0` o superiore, tutto a posto, puoi continuare.

Se invece il comando restituisce un errore simile a `Command not found` o `Comando non trovato`, oppure una versione inferiore a `v16.12.0`, allora dovrai [installare una versione di Node.js compatibile](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)

### Editor di codice

Dovrai inoltre scaricare e installare un **editor di codice** per scrivere il tuo codice.

:::tip[noi useremo...]
In questo tutorial useremo **VS Code**, tuttavia è possibile usare qualsiasi editor disponibile per il tuo sistema operativo.
:::

1. [Scarica e installa VS Code](https://code.visualstudio.com/#alt-downloads) o qualsiasi altro editor di codice di tuo gradimento.
