## Ottieni gli strumenti di sviluppo di cui hai bisogno

### Terminale

Userai una **riga di comando (terminale)** per creare il tuo progetto Astro e per eseguire comandi fondamentali per costruire, sviluppare e testare il tuo sito.

Puoi accedere alla riga di comando tramite un programma di terminale locale per il tuo sistema operativo. Le applicazioni comuni includono **Terminale** (MacOS/Linux), **Prompt dei comandi** (Windows) e **Termux** (Android). Uno di questi dovrebbe già essere installato sul tuo computer.

### Node.js

Per far funzionare Astro sul tuo sistema, dovrai anche avere installato [**Node.js**](https://nodejs.org/en/) versione `v18.17.1` o `v20.3.0` o successiva. (La versione `v19` non è supportata.)

Per verificare se hai già una versione compatibile installata, esegui il seguente comando nel tuo terminale:

```sh
node -v

// Esempio di output
v18.17.1
```

Se il comando restituisce un numero di versione maggiore di `v18.17.1` o `v20.3.0` (esclusa la `v19`), sei a posto!

Se il comando restituisce un messaggio di errore come `Command 'node' not found`, o un numero di versione inferiore a quello richiesto, dovrai [installare una versione compatibile di Node.js](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm).

### Editor di codice

Inoltre, dovrai scaricare e installare un **editor di codice** per scrivere il tuo codice.

:::tip[Useremo...]
In questo tutorial useremo **VS Code**, ma puoi usare qualsiasi editor compatibile con il tuo sistema operativo.
:::

<Steps>
1. [Scarica e installa VS Code](https://code.visualstudio.com/#alt-downloads) o un altro editor di codice a tua scelta.
</Steps>
