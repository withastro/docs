Qui, collegherai il tuo repository GitHub a Netlify. Netlify utilizzerà quel progetto per costruire e distribuire il tuo sito live sul web ogni volta che esegui il commit di una modifica al tuo codice.

:::tip[Useremo...]
Questo tutorial utilizzerà **Netlify**, ma sei libero di usare il tuo servizio di hosting preferito per distribuire il tuo sito su internet.
:::

## Crea un nuovo sito Netlify

<Steps>
1. Crea un account gratuito su [Netlify](https://netlify.com) se non ne hai già uno.

    Prendi nota del tuo username. Visualizzerai la tua dashboard e tutti i siti che crei su `https://app.netlify.com/teams/username`

2. Clicca <kbd>Add new site</kbd> > <kbd>Import an existing project</kbd>.

    Ti verrà chiesto di connetterti a un provider Git. Scegli GitHub e segui i passaggi sullo schermo per autenticare il tuo account GitHub. Quindi, scegli il repository GitHub del tuo progetto Astro dalla lista fornita.

3. Al passaggio finale, Netlify ti mostrerà le impostazioni del sito della tua app. Le impostazioni predefinite dovrebbero essere corrette per il tuo progetto Astro, quindi puoi scorrere verso il basso e cliccare <kbd>Deploy site</kbd>.
</Steps>

Congratulazioni, hai un sito web Astro!

## Cambia il nome del tuo progetto

Nella pagina di panoramica del tuo sito in Netlify, vedrai il nome del tuo progetto generato casualmente e l'URL del tuo sito web nella forma `https://project-name-123456.netlify.app`. Puoi cambiare il nome del tuo progetto in qualcosa di più memorabile e questo aggiornerà automaticamente il tuo URL.

## Visita il tuo nuovo sito web

Clicca sull'URL nelle impostazioni del tuo sito, o digitalo in una finestra del browser per visualizzare il tuo nuovo sito web.
