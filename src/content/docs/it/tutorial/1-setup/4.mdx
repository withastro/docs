Questo tutorial utilizzerà GitHub per archiviare il nostro repository e collegarlo a un host web. Puoi usare il fornitore di git online che preferisci.

:::note
Se sei già familiare con git e hai un tuo flusso di lavoro, crea un nuovo repository GitHub per il tuo progetto utilizzando il metodo che preferisci. Salta alla pagina successiva: ["Distribuisci il tuo sito sul web"](/en/tutorial/1-setup/5/).
:::

## Crea un repository su GitHub

Sebbene ci siano diversi modi per archiviare il tuo codice locale su GitHub, questo tutorial ti guiderà in un metodo che non richiede l'uso di git dalla riga di comando. 

<Steps>
1. Accedi a GitHub.com tramite il tuo browser e clicca sul <kbd>+</kbd> nell'angolo in alto a destra dello schermo per creare un nuovo repository. 

2. Scegli un nome per il tuo repository. Questo non deve essere lo stesso nome della tua cartella di progetto.

3. Ti verranno presentate alcune opzioni, ma non è necessario modificare nulla dei valori predefiniti. Scorri verso il basso e clicca il pulsante <kbd>Crea repository</kbd>.

4. Ti verranno presentati i vari passaggi di configurazione successivi, ma non sarà necessario utilizzarli. Prendi nota dell'URL del tuo repository. Ora puoi uscire da questa pagina senza fare nulla.
</Steps>

## Commit del tuo codice locale su GitHub

Nella sezione precedente, hai fatto una modifica al contenuto della tua pagina. Questo significa che i file del tuo progetto sono cambiati, e VS Code dovrebbe mostrarti un numero sopra l'icona del menu "Source". Questa scheda è dove andrai regolarmente per aggiornare i tuoi file su GitHub.

<Steps>
1. Clicca sulla scheda Controllo sorgente in VS Code per vedere un elenco dei file modificati. Se vedi un messaggio che ti chiede di installare `git`, segui le istruzioni fornite, poi ricarica VS Code.

2. Clicca sul menu <kbd>•••</kbd> "3 puntini" sopra il messaggio di commit e scegli <kbd>Remote</kbd> > <kbd>Aggiungi Remote</kbd>.

3. Seleziona <kbd>Aggiungi remote da GitHub</kbd>. Se necessario, segui i passaggi di autenticazione, quindi torna su VS Code e ripeti questa azione.

4. Dovresti vedere un elenco di tutti i tuoi repository su GitHub. Scegli quello che hai creato per questo progetto. Se non vedi il tuo progetto, incolla direttamente l'URL del repository di GitHub. Potresti anche essere chiesto di dare a questo repository un nome locale. Puoi scegliere qualsiasi nome.

5. In cima al pannello del menu, ci sarà un campo dove inserire un **messaggio di commit** (descrizione delle modifiche ai tuoi file). Scrivi `commit iniziale` e premi il pulsante <kbd>Commit</kbd> per registrare queste modifiche.

6. Potresti vedere un messaggio che ti dice che non hai commit "staggiati" e ti chiede se desideri stagiarli. Clicca su <kbd>Sempre</kbd> e continua.

7. Infine, l'elenco dei file modificati dovrebbe essere sostituito dal pulsante <kbd>Pubblica</kbd>. Clicca su questo per inviare le tue modifiche registrate su GitHub.
</Steps>

### Vedi il tuo progetto su GitHub

Per verificare che il tuo progetto sia stato archiviato correttamente su GitHub, visita GitHub.com e guarda sotto il tuo account per un elenco dei tuoi repository. Scegli quello nuovo che hai creato e verifica che contenga i file del tuo progetto Astro.
