## Modifica la tua home page

<Steps>
1. Nel tuo editor di codice, naviga nel pannello dei file Explorer fino a `src/pages/index.astro` e cliccaci sopra per aprire il contenuto del file in una scheda modificabile.

    Il contenuto del tuo file `index.astro` dovrebbe apparire così:

    ```astro title="src/pages/index.astro"
    ---
    ---

    <html lang="it">
      <head>
        <meta charset="utf-8" />
        <link rel="icon" type="image/svg+xml" href="/favicon.svg" />
        <meta name="viewport" content="width=device-width" />
        <meta name="generator" content={Astro.generator} >
        <title>Astro</title>
      </head>
      <body>
        <h1>Astro</h1>
      </body>
    </html>
    ```

2. Modifica il contenuto del `<body>` della tua pagina.

    Digita nell'editor per cambiare il testo dell'intestazione sulla tua pagina e salva la modifica.

    ```astro title="src/pages/index.astro" del={2} ins={3}
    <body>
      <h1>Astro</h1>
      <h1>Il mio sito Astro</h1>
    </body>
    ```

3. Controlla l'anteprima del browser e dovresti vedere il contenuto della tua pagina aggiornato con il nuovo testo.
</Steps>

Congratulazioni! Ora sei uno sviluppatore Astro!

Il resto di questa unità ti preparerà per il successo con il controllo di versione e un sito web pubblicato che puoi mostrare.
