Ora che hai alcuni post nel blog, è il momento di usare l'API di Astro per lavorare con i tuoi file!

## Cosa ti aspetta

In questa unità, potenzierai il tuo blog con una pagina indice, pagine per i tag e un feed RSS.

Nel frattempo, imparerai a usare:
- `import.meta.glob()` per accedere ai dati dai file nel tuo progetto
- `getStaticPaths()` per creare più pagine (rotte) contemporaneamente
- Il pacchetto RSS di Astro per creare un feed RSS
