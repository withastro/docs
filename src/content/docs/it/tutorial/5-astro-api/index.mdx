Ora che hai alcuni articoli del blog, è il momento di usare l'API di Astro per lavorare con i tuoi file!

## Guardando avanti

In questa unità, potenzierai il tuo blog con una pagina indice, pagine di tag e un feed RSS.

Lungo il percorso, imparerai come usare:
- `import.meta.glob()` per accedere ai dati dai file nel tuo progetto
- `getStaticPaths()` per creare più pagine (rotte) contemporaneamente
- Il pacchetto RSS di Astro per creare un feed RSS
