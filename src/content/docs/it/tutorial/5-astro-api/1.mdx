Ora che hai alcuni post del blog da collegare, è il momento di configurare la pagina del Blog per creare un elenco di essi automaticamente!

<PreCheck>
  - Accedi ai dati di tutti i tuoi post contemporaneamente usando `import.meta.glob()`
  - Visualizza un elenco generato dinamicamente di post nella tua pagina Blog
  - Rifattorizza per usare un componente `<BlogPost />` per ogni elemento dell'elenco
</PreCheck>

## Visualizzare dinamicamente un elenco di post

<Steps>
1. Aggiungi il seguente codice a `blog.astro` per restituire informazioni su tutti i tuoi file Markdown. `import.meta.glob()` restituirà un array di oggetti, uno per ogni post del blog.

    ```astro title="src/pages/blog.astro" ins={3}
    ---
    import BaseLayout from '../layouts/BaseLayout.astro'
    const allPosts = Object.values(import.meta.glob('./posts/*.md', { eager: true }));
    const pageTitle = "Il mio blog di apprendimento Astro";
    ---
    <BaseLayout pageTitle={pageTitle}>
      <p>Qui pubblicherò il mio percorso di apprendimento di Astro.</p>
      <ul>
        <li><a href="/posts/post-1/">Post 1</a></li>
        <li><a href="/posts/post-2/">Post 2</a></li>
        <li><a href="/posts/post-3/">Post 3</a></li>
      </ul>
    </BaseLayout>
    ```

2. Per generare dinamicamente l'intero elenco dei post, usando i titoli e gli URL dei post, sostituisci i tuoi singoli tag `<li>` con il seguente codice Astro:

    ```astro title="src/pages/blog.astro" del={9,10,11} ins={13}
    ---
    import BaseLayout from '../layouts/BaseLayout.astro'
    const allPosts = Object.values(import.meta.glob('./posts/*.md', { eager: true }));
    const pageTitle = "Il mio blog di apprendimento Astro";
    ---
    <BaseLayout pageTitle={pageTitle}>
      <p>Qui pubblicherò il mio percorso di apprendimento di Astro.</p>
      <ul>
        {allPosts.map((post: any) => <li><a href={post.url}>{post.frontmatter.title}</a></li>)}
      </ul>
    </BaseLayout>
    ```

    Ora, l'intero elenco dei post del blog viene generato dinamicamente utilizzando il supporto TypeScript integrato di Astro, mappando l'array restituito da `import.meta.glob()`.

3. Aggiungi un nuovo post del blog creando un nuovo file `post-4.md` in `src/pages/posts/` e aggiungendo del contenuto Markdown. Assicurati di includere almeno le proprietà frontmatter usate qui sotto.

    ```markdown
    ---
    layout: ../../layouts/MarkdownPostLayout.astro
    title: Il mio quarto post del blog
    author: Astro Learner
    description: "Questo post apparirà da solo!"
    image: 
        url: "https://docs.astro.build/default-og-image.png"
        alt: "La parola astro contro un'illustrazione di pianeti e stelle."
    pubDate: 2022-08-08
    tags: ["astro", "successi"]
    ---
    Questo post dovrebbe apparire con gli altri post del mio blog, perché `import.meta.glob()` sta restituendo un elenco di tutti i miei post per creare l'elenco.
    ```

4. Ritorna alla tua pagina del blog nella tua anteprima del browser su `http://localhost:4321/blog` e cerca un elenco aggiornato con quattro elementi, incluso il tuo nuovo post del blog!
</Steps>

## Sfida: Crea un componente BlogPost

Prova a fare da solo tutte le modifiche necessarie nel tuo progetto Astro in modo da poter usare il seguente codice per generare il tuo elenco di post del blog:

```astro title="src/pages/blog.astro" del={2} ins={3}
<ul>
  {allPosts.map((post: any) => <li><a href={post.url}>{post.frontmatter.title}</a></li>)}
  {allPosts.map((post: any) => <BlogPost url={post.url} title={post.frontmatter.title} />)}
</ul>
```

    <details>
