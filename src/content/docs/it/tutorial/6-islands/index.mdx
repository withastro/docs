Ora che hai un blog completamente funzionante, è il momento di aggiungere alcune isole interattive al tuo sito!

## Guardando avanti

In questa unità, utilizzerai le **isole Astro** per portare i componenti dei framework frontend nel tuo sito Astro.

Farai:
- Aggiungerai un framework UI, **Preact**, al tuo progetto Astro
- Usi Preact per creare un componente di saluto interattivo
- Imparerai quando **non** scegliere le isole per l'interattività
