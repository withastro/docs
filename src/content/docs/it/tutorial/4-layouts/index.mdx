Ora che sai come costruire con i componenti, è il momento di creare dei layout personalizzati!

## Guardando al futuro

In questa unità, costruirai layout per condividere elementi e stili comuni tra le tue pagine e i post del blog.

Per farlo, farai:

- Creare componenti di layout riutilizzabili
- Passare il contenuto ai tuoi layout con `<slot />`
- Passare i dati dal frontmatter di Markdown ai tuoi layout
- Annidare layout multipli
