Ora che puoi costruire con i componenti, è il momento di creare alcuni layout personalizzati!

## Guardando avanti

In questa unità, costruirai layout per condividere elementi e stili comuni tra le tue pagine e articoli del blog.

Per fare questo, farai:

- Creare componenti layout riutilizzabili
- Passare contenuto ai tuoi layout con `<slot />`
- Passare dati dal frontmatter Markdown ai tuoi layout
- Nidificare layout multipli
