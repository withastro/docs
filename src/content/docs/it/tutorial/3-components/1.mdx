Ora che hai lo stesso HTML scritto su più pagine del tuo sito Astro, è il momento di sostituire quel contenuto duplicato con un componente riutilizzabile di Astro!

<PreCheck>

  - Crea un componente Astro per visualizzare i tuoi link di navigazione

</PreCheck>

## Crea una nuova cartella `src/components/`

Per contenere i file `.astro` che genereranno HTML ma che non diventeranno nuove pagine sul tuo sito, avrai bisogno di una nuova cartella nel tuo progetto: `src/components/`. 


## Crea un componente di Navigazione

<Steps>
1. Crea un nuovo file: `src/components/Navigation.astro`.

2. Copia i tuoi link per navigare tra le pagine dalla parte superiore di qualsiasi pagina e incollali nel tuo nuovo file, `Navigation.astro`:

    ```astro title="src/components/Navigation.astro" "---"
    ---
    ---
    <a href="/">Home</a>
    <a href="/about/">About</a>
    <a href="/blog/">Blog</a>
    ```
    :::tip
    Se non c'è nulla nel frontmatter del tuo file `.astro`, non devi scrivere i delimitatori del codice. Puoi sempre aggiungerli di nuovo quando ne hai bisogno.
    :::
</Steps>

### Importa e usa Navigation.astro

<Steps>
1. Torna a `index.astro` e importa il tuo nuovo componente all'interno del delimitatore del codice:

    ```astro title="src/pages/index.astro" ins={2}
    ---
    import Navigation from '../components/Navigation.astro';
    import "../styles/global.css";

    const pageTitle = "Home Page";
    ---
    ```

2. Sostituisci quindi l'HTML esistente dei link di navigazione con il nuovo componente di navigazione che hai appena importato:

    ```astro title="src/pages/index.astro" del={1-3} ins={4}
    <a href="/">Home</a>
    <a href="/about/">About</a>
    <a href="/blog/">Blog</a>
    <Navigation />
    ```

3. Controlla l'anteprima nel tuo browser e nota che dovrebbe sembrare esattamente la stessa… ed è quello che vuoi! 
</Steps>

Il tuo sito contiene lo stesso HTML di prima, ma ora, quelle tre righe di codice sono fornite dal tuo componente `<Navigation />`. 

<Box icon="puzzle-piece">

## Prova tu stesso - Aggiungi la navigazione al resto del sito

Importa e usa il componente `<Navigation />` nelle altre due pagine del tuo sito (`about.astro` e `blog.astro`) utilizzando lo stesso metodo.

Non dimenticare di
- Aggiungere una dichiarazione di importazione nella parte superiore dello script del componente, all'interno del delimitatore del codice.
- Sostituire il codice esistente con il componente di navigazione.

</Box>

:::note
Quando ristrutturi il tuo codice ma non cambi il modo in cui la pagina appare nel browser, stai **refattorizzando**. Refattorizzerai **più volte** in questa unità, mentre sostituisci le parti del tuo HTML della pagina con componenti. 

Questo ti consente di partire rapidamente con qualsiasi codice funzionante, spesso duplicato in tutto il tuo progetto. Poi, puoi migliorare il design del codice esistente in modo incrementale senza cambiare l'aspetto esteriore del sito.
:::
