Automatically configures a Chrome DevTools workspace for your project when running `astro dev`. 

This allows you to edit files directly in the browser and have those changes reflected in your local filesystem.

```js title="astro.config.mjs" ins={4-6}
import { defineConfig } from 'astro/config';

export default defineConfig({
  experimental: {
    chromeDevtoolsWorkspace: true,
  },
});
```

You can then open your Astro site in Chrome, open DevTools, and navigate to the "Sources" tab. You should see your project listed under the "Workspace" section. Click "Connect" to add your project directory as a workspace.

See the [Chrome DevTools workspace documentation](https://developer.chrome.com/docs/devtools/workspaces#connect) for more information.
