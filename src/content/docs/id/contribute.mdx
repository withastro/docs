Kami menyambut kontribusi dalam ukuran apa pun dan dari tingkat kemampuan apa pun. Sebagai proyek open-source, kami percaya pada memberi kembali kepada para kontributor. Kami senang membantu memberi arahan untuk PR, penulisan teknis, dan mewujudkan ide fitur menjadi kenyataan.

:::tip[Open Issues]
Jelajahi [semua issue GitHub Astro berlabel `help wanted` dan `good first issue`](https://github.com/search?q=org%3Awithastro+label%3A%22good+first+issue%22%2C%22help+wanted%22%2C+state%3Aopen&type=issues) untuk seluruh repositori kode: kode Astro, dokumentasi, Starlight, situs web Astro, alat bahasa, GitHub Actions, bot, kompiler... ada banyak tempat untuk berkontribusi!
:::

Ingin terlibat lebih jauh? Lihat [dokumen Tata Kelola (Governance)](https://github.com/withastro/.github/blob/main/GOVERNANCE.md) untuk deskripsi mendetail tentang berbagai peran, proses nominasi maintainer, proses code review, dan penegakan Kode Etik.

## Cara Berkontribusi

### Repositori proyek

Ada banyak cara berkontribusi ke Proyek Astro! Setiap repositori Astro memiliki README dengan tautan ke berkas `CONTRIBUTING.md` di root proyek.

Kunjungi [profil GitHub Astro](https://github.com/withastro) untuk menemukan repositori berikut:

- [Kode utama Astro](https://github.com/withastro/astro), termasuk integrasi resmi dan template starter.

- [Astro Docs](https://github.com/withastro/docs), sebuah situs Astro lengkap! Berkontribusilah bukan hanya konten tulisan, tetapi juga kode Astro yang menyentuh a11y, CSS, UI, dan UX. Dokumentasi kami tersedia dalam beberapa bahasa, jadi kami juga butuh bantuan menerjemahkan seluruh situs.

- [Kompiler Astro](https://github.com/withastro/compiler), ditulis dalam Go, dan didistribusikan sebagai WASM.

- [Alat bahasa Astro](https://github.com/withastro/language-tools), tooling editor yang dibutuhkan untuk bahasa Astro (berkas `.astro`).

- [Starlight](https://github.com/withastro/starlight), kerangka dokumentasi resmi milik Astro.

- [Astro Roadmap](https://github.com/withastro/roadmap) tempat masa depan Astro dibentuk! Ide, saran, dan proposal RFC resmi untuk proyek Astro.

### Jenis kontribusi

Selain menyumbang kode atau konten sendiri, Anda juga bisa berkontribusi besar dengan terlibat menuliskan komentar review di PR, menambahkan ide pada Issue dan Discussion yang ada, serta berpartisipasi dalam tugas pemeliharaan issue “Pinned”!

Setiap PR, terutama PR terjemahan, membutuhkan reviewer! Melakukan review PR dan meninggalkan komentar, saran, atau “LGTM!” (“Looks Good To Me!”) adalah cara yang bagus untuk mulai berkontribusi di repositori mana pun, sekaligus belajar lebih banyak tentang Astro.

Kami juga punya komunitas [Discord](https://astro.build/chat) yang sangat aktif! Kami menghargai kontribusi mereka yang menyambut anggota baru, menjawab pertanyaan dukungan, dan berbagi apa yang telah mereka bangun dengan dan untuk Astro! Di luar kontribusi GitHub tradisional, Astro juga mengakui serta mendukung anggota komunitas yang aktif berbagi Astro lewat blog, video, dan talk konferensi, serta membantu menjaga kesehatan komunitas.

## Berkontribusi ke Dokumentasi

Kami memiliki beberapa panduan untuk membantu Anda berkontribusi ke Astro Docs.

Baik itu kontribusi open-source pertama Anda, menambah dokumentasi untuk fitur Astro baru yang baru saja Anda bangun, Anda seorang penerjemah berpengalaman yang mencari halaman berikutnya untuk diterjemahkan, atau Anda ingin belajar lebih jauh tentang membantu sebagai reviewer PR… semuanya ada!

Silakan kunjungi situs khusus [*Astro Docs* Docs](https://contribute.docs.astro.build), tempat Anda akan menemukan dokumentasi untuk membantu berkontribusi ke Astro Docs sebagai pengoreksi typo, penulis, penerjemah, pembuat fitur, hingga reviewer PR.
