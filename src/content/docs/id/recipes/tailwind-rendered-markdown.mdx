Anda dapat menggunakan plugin Typography milik [Tailwind](https://tailwindcss.com) untuk menata Markdown yang dirender dari sumber seperti [**content collections**](/en/guides/content-collections/) milik Astro.

Resep ini akan mengajarkan Anda cara membuat komponen Astro yang dapat digunakan kembali untuk menata konten Markdown menggunakan utility class Tailwind.

## Prasyarat

Sebuah proyek Astro yang:

	- telah memasang [plugin Vite Tailwind](/en/guides/styling/#tailwind).
	- menggunakan [content collections](/en/guides/content-collections/) milik Astro.

## Menyiapkan `@tailwindcss/typography`

Pertama, instal `@tailwindcss/typography` menggunakan manajer paket pilihan Anda.

<PackageManagerTabs>
 	<Fragment slot="npm">
	```shell 
	npm install -D @tailwindcss/typography
	```
	</Fragment>
  	<Fragment slot="pnpm">
	```shell 
	pnpm add -D @tailwindcss/typography
	```
	</Fragment>
  	<Fragment slot="yarn">
	```shell
	yarn add --dev @tailwindcss/typography
	```
	</Fragment>
</PackageManagerTabs>

Kemudian, tambahkan paket tersebut sebagai plugin di berkas konfigurasi Tailwind Anda.

```css title="src/styles/global.css" ins={2}
@import 'tailwindcss';
@plugin '@tailwindcss/typography';
```

## Resep

<Steps>
1. Buat komponen `<Prose />` untuk menyediakan `<div>` pembungkus dengan `<slot />` bagi Markdown yang dirender. Tambahkan kelas gaya `prose` bersama [Tailwind element modifiers](https://tailwindcss.com/docs/typography-plugin#element-modifiers) yang diinginkan pada elemen induk.

    ```astro title="src/components/Prose.astro"
    ---
    ---
    <div 
      class="prose dark:prose-invert 
      prose-h1:font-bold prose-h1:text-xl 
      prose-a:text-blue-600 prose-p:text-justify prose-img:rounded-xl 
      prose-headings:underline">
      <slot />
    </div>
    ```
    :::tip
    Plugin `@tailwindcss/typography` menggunakan [**element modifiers**](https://tailwindcss.com/docs/typography-plugin#element-modifiers) untuk menata komponen anak dari sebuah kontainer dengan kelas `prose`. 

    Modifier ini mengikuti sintaks umum berikut: 

      ```
      prose-[element]:class-to-apply
      ``` 

    Misalnya, `prose-h1:font-bold` memberikan semua tag `<h1>` kelas Tailwind `font-bold`.
    :::

2. Query entri koleksi Anda pada halaman tempat Anda ingin merender Markdown. Teruskan komponen `<Content />` dari `await render(entry)` ke `<Prose />` sebagai child untuk membungkus konten Markdown Anda dengan gaya Tailwind.

    ```astro title="src/pages/index.astro"
    ---
    import Prose from '../components/Prose.astro';
    import Layout from '../layouts/Layout.astro';
    import { getEntry, render } from 'astro:content';

    const entry = await getEntry('collection', 'entry');
    const { Content } = await render(entry);
    ---
    <Layout>
      <Prose>
        <Content />
      </Prose>
    </Layout>
    ```
</Steps>
