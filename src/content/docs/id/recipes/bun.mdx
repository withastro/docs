[Bun](https://bun.sh/) adalah runtime & toolkit JavaScript serba ada. Lihat [dokumentasi Bun](https://bun.sh/docs) untuk informasi lebih lanjut.

:::caution
Menggunakan Bun dengan Astro mungkin masih memiliki kekasaran. Beberapa integrasi mungkin tidak berfungsi seperti yang diharapkan. Lihat [dokumentasi resmi Bun untuk bekerja dengan Astro](https://bun.sh/guides/ecosystem/astro) untuk detailnya.

Jika Anda mengalami masalah saat menggunakan Bun, mohon [buka Issue di GitHub langsung ke Bun](https://github.com/oven-sh/bun/issues/new/choose).
:::

## Prasyarat

- Bun terpasang secara lokal di mesin Anda. Lihat [instruksi instalasi](https://bun.sh/docs/installation) di dokumentasi resmi Bun.

## Buat proyek Astro baru dengan Bun

Buat proyek Astro baru dengan Bun menggunakan perintah `create-astro` berikut:

```bash
bun create astro my-astro-project-using-bun
```

## Instal dependensi

Jika Anda melewati langkah “Install dependencies?” saat wizard CLI, pastikan Anda menginstal dependensi sebelum melanjutkan.

```bash
bun install
```

## Tambahkan Types

Bun menerbitkan paket [`@types/bun`](https://www.npmjs.com/package/@types/bun) yang berisi tipe runtime untuk Bun.

Instal `@types/bun` dengan perintah berikut:

```sh
bun add -d @types/bun
```

## Flag instalasi CLI

### Menggunakan integrasi

Anda juga dapat menggunakan integrasi resmi Astro mana pun dengan perintah `astro add`:

```bash
bun astro add react
```

### Gunakan tema atau template starter

Anda dapat memulai proyek Astro baru berdasarkan [contoh resmi](https://github.com/withastro/astro/tree/main/examples) atau branch utama repositori GitHub mana pun dengan menambahkan argumen `--template` pada perintah `create astro`.

Jalankan perintah berikut di terminal Anda, ganti dengan nama starter resmi Astro, atau username dan repository GitHub tema yang ingin Anda gunakan:

```bash
# buat proyek baru dengan contoh resmi
bun create astro@latest --template <example-name>
# buat proyek baru berbasis branch utama sebuah repositori GitHub
bun create astro@latest --template <github-username>/<github-repo>
```

## Develop dan build

Untuk menjalankan server pengembangan, gunakan perintah berikut:

```bash
bun run dev
```

### Build dan preview situs Anda

Untuk melakukan build situs Anda, gunakan perintah berikut:

```bash
bun run build
```

Setelah build selesai, jalankan perintah preview yang sesuai (mis. `bun run preview`) di terminal dan Anda dapat melihat versi hasil build situs Anda secara lokal di jendela pratinjau browser yang sama.

## Pengujian

Bun menyertakan test runner bawaan yang cepat dan kompatibel dengan Jest melalui perintah [`bun test`](https://bun.sh/docs/cli/test). Anda juga dapat menggunakan [alat pengujian lain untuk Astro](/en/guides/testing/).
