Referensi berikut adalah daftar lengkap error yang mungkin Anda temui saat menggunakan Astro. Untuk bantuan tambahan, termasuk jebakan umum, silakan lihat juga [Panduan Pemecahan Masalah](/id/guides/troubleshooting/).

## Astro Errors

- [**UnknownCompilerError**](/id/reference/errors/unknown-compiler-error/)<br/>Kesalahan kompilator yang tidak diketahui.
- [**ClientAddressNotAvailable**](/id/reference/errors/client-address-not-available/)<br/>`Astro.clientAddress` tidak tersedia di adapter saat ini.
- [**PrerenderClientAddressNotAvailable**](/id/reference/errors/prerender-client-address-not-available/)<br/>`Astro.clientAddress` tidak dapat digunakan di dalam rute yang diprerender.
- [**StaticClientAddressNotAvailable**](/id/reference/errors/static-client-address-not-available/)<br/>`Astro.clientAddress` tidak tersedia di halaman yang diprerender.
- [**NoMatchingStaticPathFound**](/id/reference/errors/no-matching-static-path-found/)<br/>Tidak ada path statis yang ditemukan untuk path yang diminta.
- [**OnlyResponseCanBeReturned**](/id/reference/errors/only-response-can-be-returned/)<br/>Tipe yang dikembalikan oleh halaman Astro tidak valid.
- [**MissingMediaQueryDirective**](/id/reference/errors/missing-media-query-directive/)<br/>Nilai untuk direktif `client:media` hilang.
- [**NoMatchingRenderer**](/id/reference/errors/no-matching-renderer/)<br/>Tidak ada renderer yang cocok ditemukan.
- [**NoClientEntrypoint**](/id/reference/errors/no-client-entrypoint/)<br/>Tidak ada entrypoint klien yang ditentukan di renderer.
- [**NoClientOnlyHint**](/id/reference/errors/no-client-only-hint/)<br/>Petunjuk untuk direktif `client:only` tidak ada.
- [**InvalidGetStaticPathParam**](/id/reference/errors/invalid-get-static-path-param/)<br/>Nilai tidak valid yang dikembalikan oleh entri/path `getStaticPaths`.
- [**InvalidGetStaticPathsEntry**](/id/reference/errors/invalid-get-static-paths-entry/)<br/>Entri tidak valid di dalam nilai balik `getStaticPaths`.
- [**InvalidGetStaticPathsReturn**](/id/reference/errors/invalid-get-static-paths-return/)<br/>Nilai yang dikembalikan oleh `getStaticPaths` tidak valid.
- [**GetStaticPathsExpectedParams**](/id/reference/errors/get-static-paths-expected-params/)<br/>Properti `params` pada rute `getStaticPaths` hilang.
- [**GetStaticPathsInvalidRouteParam**](/id/reference/errors/get-static-paths-invalid-route-param/)<br/>Nilai parameter rute `getStaticPaths` tidak valid.
- [**GetStaticPathsRequired**](/id/reference/errors/get-static-paths-required/)<br/>Fungsi `getStaticPaths()` diperlukan untuk rute dinamis.
- [**ReservedSlotName**](/id/reference/errors/reserved-slot-name/)<br/>Nama slot tidak valid.
- [**NoAdapterInstalled**](/id/reference/errors/no-adapter-installed/)<br/>Tidak dapat menggunakan Server-side Rendering tanpa adapter.
- [**AdapterSupportOutputMismatch**](/id/reference/errors/adapter-support-output-mismatch/)<br/>Adapter tidak mendukung output server.
- [**NoAdapterInstalledServerIslands**](/id/reference/errors/no-adapter-installed-server-islands/)<br/>Tidak dapat menggunakan Server Islands tanpa adapter.
- [**NoMatchingImport**](/id/reference/errors/no-matching-import/)<br/>Tidak ditemukan import untuk komponen.
- [**InvalidPrerenderExport**](/id/reference/errors/invalid-prerender-export/)<br/>Ekspor prerender tidak valid.
- [**InvalidComponentArgs**](/id/reference/errors/invalid-component-args/)<br/>Argumen komponen tidak valid.
- [**PageNumberParamNotFound**](/id/reference/errors/page-number-param-not-found/)<br/>Parameter nomor halaman tidak ditemukan.
- [**ImageMissingAlt**](/id/reference/errors/image-missing-alt/)<br/>Gambar tidak memiliki properti "alt" yang wajib.
- [**InvalidImageService**](/id/reference/errors/invalid-image-service/)<br/>Galat saat memuat layanan gambar.
- [**MissingImageDimension**](/id/reference/errors/missing-image-dimension/)<br/>Dimensi gambar hilang.
- [**FailedToFetchRemoteImageDimensions**](/id/reference/errors/failed-to-fetch-remote-image-dimensions/)<br/>Gagal mengambil dimensi gambar jarak jauh.
- [**UnsupportedImageFormat**](/id/reference/errors/unsupported-image-format/)<br/>Format gambar tidak didukung.
- [**UnsupportedImageConversion**](/id/reference/errors/unsupported-image-conversion/)<br/>Konversi gambar tidak didukung.
- [**PrerenderDynamicEndpointPathCollide**](/id/reference/errors/prerender-dynamic-endpoint-path-collide/)<br/>Endpoint dinamis yang diprerender mengalami tabrakan path.
- [**PrerenderRouteConflict**](/id/reference/errors/prerender-route-conflict/)<br/>Rute yang diprerender menghasilkan path yang sama dengan rute lain.
- [**ExpectedImage**](/id/reference/errors/expected-image/)<br/>Diharapkan `src` berupa gambar.
- [**ExpectedImageOptions**](/id/reference/errors/expected-image-options/)<br/>Opsi gambar diharapkan.
- [**ExpectedNotESMImage**](/id/reference/errors/expected-not-esmimage/)<br/>Diharapkan opsi gambar, bukan gambar yang diimpor sebagai ESM.
- [**IncompatibleDescriptorOptions**](/id/reference/errors/incompatible-descriptor-options/)<br/>Tidak dapat mengatur keduanya `densities` dan `widths`.
- [**ImageNotFound**](/id/reference/errors/image-not-found/)<br/>Gambar tidak ditemukan.
- [**NoImageMetadata**](/id/reference/errors/no-image-metadata/)<br/>Tidak dapat memproses metadata gambar.
- [**CouldNotTransformImage**](/id/reference/errors/could-not-transform-image/)<br/>Tidak dapat mentransformasi gambar.
- [**ResponseSentError**](/id/reference/errors/response-sent-error/)<br/>Tidak dapat menetapkan respons.
- [**MiddlewareNoDataOrNextCalled**](/id/reference/errors/middleware-no-data-or-next-called/)<br/>Middleware tidak mengembalikan objek `Response`.
- [**MiddlewareNotAResponse**](/id/reference/errors/middleware-not-aresponse/)<br/>Middleware mengembalikan sesuatu yang bukan objek `Response`.
- [**EndpointDidNotReturnAResponse**](/id/reference/errors/endpoint-did-not-return-aresponse/)<br/>Endpoint tidak mengembalikan `Response`.
- [**LocalsNotAnObject**](/id/reference/errors/locals-not-an-object/)<br/>Nilai yang diberikan ke `locals` tidak diterima.
- [**LocalsReassigned**](/id/reference/errors/locals-reassigned/)<br/>`locals` tidak boleh diubah (reassigned).
- [**AstroResponseHeadersReassigned**](/id/reference/errors/astro-response-headers-reassigned/)<br/>`Astro.response.headers` tidak boleh diubah.
- [**MiddlewareCantBeLoaded**](/id/reference/errors/middleware-cant-be-loaded/)<br/>Tidak dapat memuat middleware.
- [**LocalImageUsedWrongly**](/id/reference/errors/local-image-used-wrongly/)<br/>Gambar lokal harus diimpor.
- [**AstroGlobUsedOutside**](/id/reference/errors/astro-glob-used-outside/)<br/>`Astro.glob()` digunakan di luar file Astro.
- [**AstroGlobNoMatch**](/id/reference/errors/astro-glob-no-match/)<br/>`Astro.glob()` tidak cocok dengan file mana pun.
- [**RedirectWithNoLocation**](/id/reference/errors/redirect-with-no-location/)<br/>Redirect harus diberikan lokasi melalui header `Location`.
- [**UnsupportedExternalRedirect**](/id/reference/errors/unsupported-external-redirect/)<br/>URL tidak didukung atau tidak valid.
- [**InvalidDynamicRoute**](/id/reference/errors/invalid-dynamic-route/)<br/>Rute dinamis tidak valid.
- [**MissingSharp**](/id/reference/errors/missing-sharp/)<br/>Tidak dapat menemukan Sharp.
- [**UnknownViteError**](/id/reference/errors/unknown-vite-error/)<br/>Kesalahan Vite yang tidak diketahui.
- [**FailedToLoadModuleSSR**](/id/reference/errors/failed-to-load-module-ssr/)<br/>Tidak dapat mengimpor file.
- [**InvalidGlob**](/id/reference/errors/invalid-glob/)<br/>Pola glob tidak valid.
- [**FailedToFindPageMapSSR**](/id/reference/errors/failed-to-find-page-map-ssr/)<br/>Astro tidak dapat menemukan halaman yang benar untuk dirender.
- [**MissingLocale**](/id/reference/errors/missing-locale/)<br/>Locale yang diberikan tidak ada.
- [**MissingIndexForInternationalization**](/id/reference/errors/missing-index-for-internationalization/)<br/>Halaman indeks tidak ditemukan.
- [**IncorrectStrategyForI18n**](/id/reference/errors/incorrect-strategy-for-i18n/)<br/>Anda tidak dapat menggunakan fungsi ini dengan strategi saat ini.
- [**NoPrerenderedRoutesWithDomains**](/id/reference/errors/no-prerendered-routes-with-domains/)<br/>Rute yang diprerender tidak didukung saat domain internasionalisasi diaktifkan.
- [**MissingMiddlewareForInternationalization**](/id/reference/errors/missing-middleware-for-internationalization/)<br/>Mengaktifkan routing internasionalisasi manual tanpa middleware.
- [**CantRenderPage**](/id/reference/errors/cant-render-page/)<br/>Astro tidak dapat merender rute.
- [**UnhandledRejection**](/id/reference/errors/unhandled-rejection/)<br/>Penolakan (promise) yang tidak tertangani.
- [**i18nNotEnabled**](/id/reference/errors/i18n-not-enabled/)<br/>i18n tidak diaktifkan.
- [**i18nNoLocaleFoundInPath**](/id/reference/errors/i18n-no-locale-found-in-path/)<br/>Path tidak mengandung locale apa pun.
- [**RouteNotFound**](/id/reference/errors/route-not-found/)<br/>Rute tidak ditemukan.
- [**EnvInvalidVariables**](/id/reference/errors/env-invalid-variables/)<br/>Variabel lingkungan tidak valid.
- [**ServerOnlyModule**](/id/reference/errors/server-only-module/)<br/>Modul hanya tersedia di sisi server.
- [**RewriteWithBodyUsed**](/id/reference/errors/rewrite-with-body-used/)<br/>Tidak dapat menggunakan `Astro.rewrite` setelah body request dibaca.
- [**ForbiddenRewrite**](/id/reference/errors/forbidden-rewrite/)<br/>Rewrite ke rute statis dilarang.
- [**UnknownFilesystemError**](/id/reference/errors/unknown-filesystem-error/)<br/>Terjadi kesalahan yang tidak diketahui saat membaca/menulis file ke disk.
- [**CannotExtractFontType**](/id/reference/errors/cannot-extract-font-type/)<br/>Tidak dapat mengekstrak tipe font dari URL yang diberikan.
- [**CannotDetermineWeightAndStyleFromFontFile**](/id/reference/errors/cannot-determine-weight-and-style-from-font-file/)<br/>Tidak dapat menentukan `weight` dan `style` dari file font.
- [**CannotFetchFontFile**](/id/reference/errors/cannot-fetch-font-file/)<br/>Tidak dapat mengambil file font yang diberikan.
- [**CannotLoadFontProvider**](/id/reference/errors/cannot-load-font-provider/)<br/>Tidak dapat memuat penyedia font.
- [**ExperimentalFontsNotEnabled**](/id/reference/errors/experimental-fonts-not-enabled/)<br/>Fitur font eksperimental tidak diaktifkan.
- [**FontFamilyNotFound**](/id/reference/errors/font-family-not-found/)<br/>Keluarga font tidak ditemukan.
- [**CspNotEnabled**](/id/reference/errors/csp-not-enabled/)<br/>Fitur CSP tidak diaktifkan.

- [**UnsupportedConfigTransformError**](/id/reference/errors/unsupported-config-transform-error/)<br/>Transformasi tidak didukung dalam konfigurasi konten.
- [**FileParserNotFound**](/id/reference/errors/file-parser-not-found/)<br/>Parser file tidak ditemukan.
- [**FileGlobNotSupported**](/id/reference/errors/file-glob-not-supported/)<br/>Pola glob tidak didukung di file loader.
