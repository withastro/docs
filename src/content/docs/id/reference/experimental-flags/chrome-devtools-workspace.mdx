Mengaktifkan [integrasi ruang kerja Chrome DevTools](https://developer.chrome.com/docs/devtools/workspaces) yang bersifat eksperimental untuk server pengembangan Astro.

Fitur ini memungkinkan Anda mengedit file langsung di Chrome DevTools dan melihat perubahan tersebut tercermin pada sistem berkas lokal melalui folder ruang kerja yang terhubung. Ini berguna untuk menerapkan suntingan seperti menyesuaikan nilai CSS tanpa meninggalkan tab peramban Anda.

Dengan fitur ini diaktifkan, menjalankan `astro dev` akan secara otomatis mengonfigurasi ruang kerja Chrome DevTools untuk proyek Anda. Proyek Anda kemudian akan muncul sebagai [sumber ruang kerja yang dapat Anda hubungkan](#connecting-your-project). Setelah itu, perubahan yang Anda lakukan di panel "Sources" akan otomatis disimpan ke kode sumber proyek Anda.

Untuk mengaktifkan fitur ini, tambahkan flag eksperimental `chromeDevtoolsWorkspace` ke konfigurasi Astro Anda:

```js title="astro.config.mjs" ins={4-6}
import { defineConfig } from 'astro/config';

export default defineConfig({
  experimental: {
    chromeDevtoolsWorkspace: true,
  },
});
```

## Menghubungkan proyek Anda

Astro akan membuat berkas konfigurasi yang diperlukan untuk mendukung ruang kerja Chrome DevTools. Namun, proyek Anda juga harus [diÂ­hubungkan sebagai sumber](https://developer.chrome.com/docs/devtools/workspaces#manual-connection) agar penyimpanan file dapat diaktifkan.

<Steps>

1. [Mulai server pengembangan Astro](/id/develop-and-build/#start-the-astro-dev-server) dengan perintah CLI yang sesuai untuk manajer paket Anda.

2. Buka pratinjau situs Anda (mis. `http://localhost:4321/`) di Chrome dan buka DevTools.

3. Di tab **Sources** > **Workspaces**, Anda akan menemukan folder proyek Astro Anda. Klik **Connect** untuk menambahkan direktori Anda sebagai ruang kerja.

</Steps>

Lihat dokumentasi [ruang kerja Chrome DevTools](https://developer.chrome.com/docs/devtools/workspaces#connect) untuk informasi lebih lanjut.
