---
title: Mengonfigurasi flag eksperimental
i18nReady: true
---

Fitur eksperimental hanya tersedia setelah mengaktifkan sebuah flag di berkas konfigurasi Astro.

```js title="astro.config.mjs"
import { defineConfig } from 'astro/config';

export default defineConfig({
    experimental: {
        // enable experimental flags
        // to try out new features
    },
});
```

Astro menawarkan flag eksperimental untuk memberi akses awal kepada pengguna terhadap fitur baru demi pengujian dan umpan balik.

Flag ini memungkinkan Anda ikut serta dalam pengembangan fitur dengan melaporkan isu dan berbagi pendapat. Fitur-fitur ini **tidak** dijamin stabil dan dapat menyertakan perubahan pemutus (breaking changes) bahkan pada rilis `patch` kecil saat fitur masih aktif dikembangkan.

Kami merekomendasikan untuk [memperbarui Astro](/id/upgrade-astro/#upgrade-to-the-latest-version) secara berkala, dan mengikuti catatan rilis di [changelog Astro](https://github.com/withastro/astro/blob/main/packages/astro/CHANGELOG.md) yang akan memberi tahu Anda perubahan apa pun yang diperlukan pada kode proyek Anda. Dokumentasi fitur eksperimental selalu diperbarui untuk **versi rilis saat ini** saja.
