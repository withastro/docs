Mengaktifkan fitur Intellisense (mis. pelengkapan kode, hint cepat) untuk entri koleksi konten Anda di editor yang kompatibel.

Saat diaktifkan, fitur ini akan membuat dan menambahkan skema JSON ke direktori `.astro` di proyek Anda. Berkas-berkas ini dapat digunakan oleh Astro language server untuk menyediakan Intellisense di dalam berkas konten (`.md`, `.mdx`, `.mdoc`).

```js
{
  experimental: {
    contentIntellisense: true,
  },
}
```

Untuk menggunakan fitur ini dengan ekstensi Astro VS Code, Anda juga harus mengaktifkan opsi `astro.content-intellisense` di pengaturan VS Code Anda. Untuk editor yang menggunakan Astro language server secara langsung, berikan parameter inisialisasi `contentIntellisense: true` untuk mengaktifkan fitur ini.
