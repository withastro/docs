Modul virtual `astro:config` mengekspos versi konfigurasi Astro yang dapat diserialisasi, aman‑tipe (type‑safe), dan tidak mencakup semuanya. Ada dua submodul untuk mengakses subset nilai konfigurasi Anda: [`/client`](#imports-from-astroconfigclient) dan [`/server`](#imports-from-astroconfigserver).

Semua nilai konfigurasi yang tersedia dapat diakses dari `astro:config/server`. Namun, untuk kode yang dieksekusi di sisi klien, hanya nilai yang diekspos oleh `astro:config/client` yang akan tersedia. Ini melindungi informasi Anda dengan hanya membuat sebagian data tersedia bagi klien.

## Imports from `astro:config/client`

```js
import {
  i18n, 
  trailingSlash,
  base,
  build,
  site,
} from "astro:config/client";
```

Gunakan submodul ini untuk kode sisi klien:

```js title="src/utils.js" ins={1} ins="trailingSlash"
import { trailingSlash } from "astro:config/client";

function addForwardSlash(path) {
  if (trailingSlash === "always") {
    return path.endsWith("/") ? path : path + "/"
  } else {
    return path
  }
}
```

Lihat selengkapnya tentang impor konfigurasi yang tersedia dari `astro:config/client`:

- [`i18n`](/id/reference/configuration-reference/#i18n)
- [`trailingSlash`](/id/reference/configuration-reference/#trailingslash)
- [`base`](/id/reference/configuration-reference/#base)
- [`build.format`](/id/reference/configuration-reference/#buildformat)
- [`site`](/id/reference/configuration-reference/#site)



## Imports from `astro:config/server`

```js
import {
  i18n, 
  trailingSlash,
  base,
  build,
  site,
  srcDir,
  cacheDir,
  outDir,
  publicDir,
  root,
} from "astro:config/server";
```

Impor ini mencakup semua yang tersedia dari `astro:config/client` sekaligus informasi tambahan yang sensitif tentang konfigurasi sistem berkas Anda yang tidak aman untuk diungkap ke klien.

Gunakan submodul ini untuk kode sisi server:

```js title="astro.config.mjs"
import { integration } from "./integration.mjs";

export default defineConfig({
    integrations: [
      integration(),
    ]
});
```

```js title="integration.mjs" ins={1} ins="outDir"
import { outDir } from "astro:config/server";
import { writeFileSync } from "node:fs";
import { fileURLToPath } from "node:url";

export default function() {
  return {
    name: "internal-integration",
    hooks: {
      "astro:build:done": () => {
        let file = new URL("result.json", outDir);
        // generate data from some operation
        let data = JSON.stringify([]);
        writeFileSync(fileURLToPath(file), data, "utf-8");
      }
    }
  }
}
```
