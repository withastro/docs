## Dapatkan alat dev yang Anda butuhkan

### Terminal

Anda akan menggunakan **command line (terminal)** untuk membuat proyek Astro dan menjalankan perintah penting guna membangun, mengembangkan, serta menguji situs Anda.

Anda dapat mengakses command line lewat program terminal lokal untuk sistem operasi Anda. Aplikasi umum antara lain **Terminal** (macOS/Linux), **Command Prompt** (Windows), dan **Termux** (Android). Salah satunya kemungkinan besar sudah ada di komputer Anda. 

### Node.js

Agar Astro dapat berjalan di sistem Anda, Anda juga perlu memasang versi [**Node.js**](https://nodejs.org/en/) yang kompatibel. Astro mendukung versi Node.js **bernomor genap**. Versi minimum yang didukung saat ini adalah: `v18.20.8`, `v20.3.0`, dan `v22.0.0`. (`v19` dan `v21` tidak didukung.)

Untuk memeriksa apakah Anda sudah memasang versi yang kompatibel, jalankan perintah berikut di terminal:

```sh
node -v

// Contoh keluaran
v18.20.8
```

Jika perintah menampilkan nomor versi yang didukung Astro, Anda siap lanjut!

Jika perintah menampilkan pesan kesalahan seperti `Command 'node' not found`, atau nomor versi lebih rendah dari yang dibutuhkan, maka Anda perlu [memasang versi Node.js yang kompatibel](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm).

### Code Editor

Selain itu, Anda perlu mengunduh dan memasang **code editor** untuk menulis kode.

:::tip[Kita akan menggunakan…]
Tutorial ini akan menggunakan **VS Code**, tetapi Anda dapat memakai editor apa pun untuk sistem operasi Anda.
:::

<Steps>
1. [Unduh dan pasang VS Code](https://code.visualstudio.com/#alt-downloads) atau editor lain pilihan Anda. 
</Steps>
