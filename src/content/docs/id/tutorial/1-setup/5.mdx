Di sini, Anda akan menghubungkan repositori GitHub ke Netlify. Netlify akan menggunakan proyek tersebut untuk membangun dan menerbitkan situs Anda secara langsung di web setiap kali Anda *commit* perubahan kode.

:::tip[Kita akan menggunakan…]
Tutorial ini akan menggunakan **Netlify**, tetapi Anda bisa memakai layanan hosting favorit Anda untuk *deploy* situs ke internet.
:::

## Buat situs Netlify baru

<Steps>
1. Buat akun gratis di [Netlify](https://netlify.com) jika Anda belum punya.

    Catat *username* Anda. Anda akan melihat dasbor dan situs yang Anda buat di `https://app.netlify.com/teams/username`

2. Klik <kbd>Add new site</kbd> > <kbd>Import an existing project</kbd>.

    Anda akan diminta terhubung ke penyedia Git. Pilih GitHub dan ikuti langkah di layar untuk mengautentikasi akun GitHub Anda. Lalu, pilih repositori GitHub proyek Astro Anda dari daftar yang disediakan.

3. Pada langkah terakhir, Netlify akan menampilkan pengaturan situs aplikasi Anda. Nilai bawaan sudah benar untuk proyek Astro, jadi Anda bisa gulir dan klik <kbd>Deploy site</kbd>.
</Steps>

Selamat, Anda punya situs Astro!

## Ubah nama proyek Anda

Di halaman ringkasan situs Netlify, Anda akan melihat nama proyek acak dan URL situs Anda dalam bentuk `https://project-name-123456.netlify.app`. Anda bisa mengubah nama proyek menjadi sesuatu yang lebih mudah diingat, dan URL Anda akan otomatis ikut diperbarui.

## Kunjungi situs baru Anda

Klik URL di pengaturan situs, atau ketik di jendela peramban untuk melihat situs baru Anda.
