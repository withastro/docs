## Tambahkan stylesheet global

Kamu telah melihat bahwa tag `<style>` Astro **terscope secara default**, artinya hanya memengaruhi elemen di file itu sendiri.

Ada beberapa cara mendefinisikan style **global** di Astro, namun pada tutorial ini kamu akan membuat dan mengimpor file `global.css` ke setiap halamanmu. Kombinasi stylesheet dan tag `<style>` ini memberi kendali untuk menerapkan sebagian style di seluruh situs, dan sebagian lagi spesifik hanya di tempat yang kamu inginkan.

<Steps>
1. Buat file baru di lokasi `src/styles/global.css` (Kamu perlu membuat folder `styles` terlebih dahulu.)

2. Salin kode berikut ke file barumu, `global.css`

    ```css title="src/styles/global.css"
    html {
      background-color: #f1f5f9;
      font-family: sans-serif;
    }

    body {
      margin: 0 auto;
      width: 100%;
      max-width: 80ch;
      padding: 1rem;
      line-height: 1.5;
    }

    * {
      box-sizing: border-box;
    }

    h1 {
      margin: 1rem 0;
      font-size: 2.5rem;
    }
    ```

3. Di `about.astro`, tambahkan pernyataan import berikut ke frontmatter: 

    ```astro title="src/pages/about.astro" ins={2}
    ---
    import '../styles/global.css';

    const pageTitle = "About Me";

    const identity = {
      firstName: "Sarah",
      country: "Canada",
      occupation: "Technical Writer",
      hobbies: ["photography", "birdwatching", "baseball"],
    };

    const skills = ["HTML", "CSS", "JavaScript", "React", "Astro", "Writing Docs"];

    const happy = true;
    const finished = false;
    const goal = 3;

    const skillColor = "crimson";
    const fontWeight = "bold";
    const textCase = "uppercase";
    ---
    ```

4. Cek pratinjau browser untuk halaman About, dan kamu akan melihat style baru diterapkan!
</Steps>

## Coba sendiri - Import stylesheet globalmu

Tambahkan baris kode yang diperlukan ke setiap file `.astro` di proyekmu untuk menerapkan style global ke setiap halaman situs.

<details>
<summary>✅ Tunjukkan kodenya! ✅</summary>

Tambahkan pernyataan import berikut ke dua file halaman lainnya: `src/pages/index.astro` dan `src/pages/blog.astro`

```astro title="src/pages/index.astro" ins={2}
---
import '../styles/global.css';
---
```
</details>
</Box>

Buat perubahan/konten tambahan apa pun yang kamu mau pada halaman About dengan menambahkan elemen HTML ke template halaman, baik secara statis maupun dinamis. Tulis JavaScript tambahan di script frontmatter untuk menyediakan nilai yang akan kamu gunakan di HTML. Jika sudah puas, commit perubahanmu ke GitHub sebelum lanjut ke pelajaran berikutnya.

### Analisis Pola

Halaman About-mu kini distyling menggunakan *kedua* file `global.css` yang diimpor *dan* tag `<style>`.

- Apakah style dari dua metode tersebut diterapkan? 

    <p>
      <Spoiler>Ya</Spoiler>
    </p>

- Apakah ada style yang konflik? Jika iya, yang mana yang diterapkan?

    <p>
      <Spoiler>Ada. `<h1>` berukuran `2.5rem` secara global, tetapi `4rem` secara lokal di tag `<style>`. Aturan lokal `4rem` yang diterapkan pada halaman About.</Spoiler>
    </p>

- Jelaskan bagaimana `global.css` dan `<style>` bekerja bersama.

    <p>
      <Spoiler>Ketika style yang sama didefinisikan secara global dan di tag `<style>` lokal halaman, style lokal akan menimpa style global. (Namun, bisa ada faktor lain. Selalu inspeksi visual situsmu untuk memastikan style diterapkan dengan benar!)</Spoiler>
    </p>

- Bagaimana kamu memilih untuk menulis style di `global.css` atau di tag `<style>`?

    <p>
      <Spoiler>Jika ingin style berlaku di seluruh situs, gunakan `global.css`. Jika hanya ingin berlaku pada konten HTML di satu file `.astro` saja dan tidak memengaruhi halaman lain, gunakan tag `<style>`.</Spoiler>
    </p>
