## Gaya untuk halaman individual

Dengan tag `<style></style>` milik Astro, kamu bisa memberi style pada item di halamanmu. Menambahkan **atribut** dan **directive** pada tag ini memberikan lebih banyak cara untuk styling.

<Steps>
1. Salin kode berikut dan tempelkan ke `src/pages/about.astro`:

    ```astro title="src/pages/about.astro" ins={6-11}
    <html lang="en">
      <head>
        <meta charset="utf-8" />
        <meta name="viewport" content="width=device-width" />
        <title>{pageTitle}</title>
        <style>
          h1 {
            color: purple;
            font-size: 4rem;
          }
        </style>
      </head> 

    ```

    Periksa ketiga halamanmu di pratinjau browser.
    
    - Warna judul halaman (h1) di:

        - Home?  <Spoiler>hitam</Spoiler>
        - About? <Spoiler>ungu</Spoiler>
        - Blog? <Spoiler>hitam</Spoiler>

    - Halaman dengan ukuran judul paling besar adalah? <Spoiler>Halaman About</Spoiler>

    :::tip
    Jika kamu sulit membedakan warna secara visual, gunakan dev tools di browser untuk menginspeksi elemen judul `<h1>` dan pastikan warna teks yang diterapkan.
    :::

2. Tambahkan nama kelas `skill` pada elemen `<li>` yang dihasilkan di halaman About agar bisa kamu-styling. Kodenya akan terlihat seperti ini:

    ```astro title="src/pages/about.astro" 'class="skill"'
    <p>My skills are:</p>
    <ul>
      {skills.map((skill) => <li class="skill">{skill}</li>)}
    </ul>
    ```

3. Tambahkan kode berikut ke tag style yang sudah ada:

    ```astro title="src/pages/about.astro" ins={6-9}
    <style>
      h1 {
        color: purple;
        font-size: 4rem;
      }
      .skill {
        color: green;
        font-weight: bold;
      }
    </style>
    ```



  4. Kunjungi kembali halaman About di browser, dan pastikan—secara visual atau via dev tools—bahwa setiap item pada daftar skill kini berwarna hijau dan tebal.
</Steps>

## Gunakan variabel CSS pertamamu
Tag `<style>` Astro juga bisa mereferensikan variabel dari script frontmatter menggunakan directive `define:vars={ {...} }`. Kamu bisa **mendefinisikan variabel di dalam pagar kode (code fence)**, lalu **menggunakannya sebagai variabel CSS di tag style**.

<Steps>
1. Definisikan variabel `skillColor` dengan menambahkannya ke script frontmatter `src/pages/about.astro` seperti ini:

    ```astro title="src/pages/about.astro" ins={17}
    ---
    const pageTitle = "About Me";

    const identity = {
      firstName: "Sarah",
      country: "Canada",
      occupation: "Technical Writer",
      hobbies: ["photography", "birdwatching", "baseball"],
    };

    const skills = ["HTML", "CSS", "JavaScript", "React", "Astro", "Writing Docs"];

    const happy = true;
    const finished = false;
    const goal = 3;
  
    const skillColor = "crimson";
    ---
    ```

2. Perbarui tag `<style>` yang ada untuk terlebih dulu mendefinisikan lalu memakai variabel `skillColor` ini di dalam kurung kurawal ganda.

    ```astro title="src/pages/about.astro" "define:vars={{skillColor}}" "var(--skillColor)" del={7} ins={8}
    <style define:vars={{skillColor}}> 
      h1 {
        color: purple;
        font-size: 4rem;
      }
      .skill {
        color: green;
        color: var(--skillColor);
        font-weight: bold;
      }
    </style>
    ```

3. Cek halaman About di pratinjau browser. Kamu harus melihat daftar skill kini berwarna merah tua (crimson), sesuai nilai variabel `skillColor` yang dipassing ke directive `define:vars`.
</Steps>

## Coba sendiri - Definisikan variabel CSS
 
<Steps>
 1. Perbarui tag `<style>` di halaman About hingga sama seperti di bawah ini. 

    ```astro title="src/pages/about.astro"
    <style define:vars={{skillColor, fontWeight, textCase}}>
      h1 {
        color: purple;
        font-size: 4rem;
      }
      .skill {
        color: var(--skillColor);
        font-weight: var(--fontWeight);
        text-transform: var(--textCase);
      }
    </style>
    ```
 
 2. Tambahkan definisi variabel yang belum ada di script frontmatter agar tag `<style>` baru ini berhasil menerapkan gaya berikut pada daftar skill-mu:
    - Warna teks merah crimson
    - Teks tebal
    - Item daftar menggunakan huruf besar semua (uppercase)
</Steps>
