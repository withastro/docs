Sekarang setelah kamu membuat halaman menggunakan berkas `.astro`, saatnya membuat postingan blog menggunakan berkas `.md`!

<PreCheck>
  - Buat folder baru dan buat postingan baru
  - Tulis konten Markdown
  - Tautkan postingan blogmu dari halaman Blog
</PreCheck>

## Buat berkas `.md` pertamamu

<Steps>
1. Buat direktori baru di `src/pages/posts/`. 

2. Tambahkan berkas (kosong) baru `post-1.md` di dalam folder `/posts/` barumu.

3. Lihat halaman ini di pratinjau peramban dengan menambahkan `/posts/post-1` pada akhir URL pratinjau yang ada. (mis. `http://localhost:4321/posts/post-1`)

4. Ubah URL pratinjau peramban untuk melihat `/posts/post-2` sebagai gantinya. (Ini halaman yang belum kamu buat.) 

    Perhatikan keluaran yang berbeda saat mempratinjau halaman "kosong" dan halaman yang tidak ada. Ini akan membantu *troubleshooting* di masa depan.
</Steps>

## Tulis konten Markdown

<Steps>
1. Salin atau ketik kode berikut ke `post-1.md`

    ```markdown title="src/pages/posts/post-1.md"
    ---
    title: 'My First Blog Post'
    pubDate: 2022-07-01
    description: 'This is the first post of my new Astro blog.'
    author: 'Astro Learner'
    image:
        url: 'https://docs.astro.build/assets/rose.webp' 
        alt: 'The Astro logo on a dark background with a pink glow.'
    tags: ["astro", "blogging", "learning in public"]
    ---
    # My First Blog Post

    Published on: 2022-07-01

    Welcome to my _new blog_ about learning Astro! Here, I will share my learning journey as I build a new website.

    ## What I've accomplished

    1. **Installing Astro**: First, I created a new Astro project and set up my online accounts.

    2. **Making Pages**: I then learned how to make pages by creating new `.astro` files and placing them in the `src/pages/` folder.

    3. **Making Blog Posts**: This is my first blog post! I now have Astro pages and Markdown posts!

    ## What's next

    I will finish the Astro tutorial, and then keep adding more posts. Watch this space for more to come.
    ```

2. Cek lagi pratinjau peramban di `http://localhost:4321/posts/post-1`. Kamu sekarang akan melihat konten pada halaman ini. Mungkin belum terformat dengan baik, tapi jangan khawatir, kamu akan memperbaruinya nanti di tutorial!

3. Gunakan DevTools peramban untuk menginspeksi halaman ini. Perhatikan bahwa walaupun kamu tidak menulis elemen HTML, Markdown-mu dikonversi menjadi HTML. Kamu bisa melihat elemen seperti heading, paragraf, dan item daftar.
</Steps>

:::note
Informasi di bagian atas berkas, di dalam pagar kode, disebut *frontmatter*. Data ini—termasuk *tags* dan gambar postingan—adalah informasi *tentang* postinganmu yang bisa digunakan Astro. Data ini tidak otomatis muncul di halaman, tetapi nanti di tutorial kamu akan mengaksesnya untuk meningkatkan situs.
:::

## Tautkan ke postinganmu

<Steps>
1. Tautkan ke postingan pertamamu dengan *anchor tag* di `src/pages/blog.astro`:
    ```astro title="src/pages/blog.astro" ins={16-18}
    ---
    ---
    <html lang="en">
      <head>
        <meta charset="utf-8"/>
        <meta name="viewport" content="width=device-width" />
        <title>Astro</title>
      </head>
      <body>
        <a href="/">Home</a>
        <a href="/about/">About</a>
        <a href="/blog/">Blog</a>

        <h1>My Astro Learning Blog</h1>
        <p>This is where I will post about my journey learning Astro.</p>
        <ul>
          <li><a href="/posts/post-1/">Post 1</a></li>
        </ul>
      </body>
    </html>
    ```

2. Sekarang, tambahkan dua berkas lagi di `src/pages/posts/`: `post-2.md` dan `post-3.md`. Berikut contoh kode yang bisa kamu salin-tempel, atau kamu bisa membuat sendiri!

    ```md title="src/pages/posts/post-2.md"
    ---
    title: My Second Blog Post
    author: Astro Learner
    description: "After learning some Astro, I couldn't stop!"
    image: 
        url: "https://docs.astro.build/assets/arc.webp"
        alt: "The Astro logo on a dark background with a purple gradient arc."
    pubDate: 2022-07-08
    tags: ["astro", "blogging", "learning in public", "successes"]
    ---
    After a successful first week learning Astro, I decided to try some more. I wrote and imported a small component from memory!
    ```

    ```md title="src/pages/posts/post-3.md"
    ---
    title: My Third Blog Post
    author: Astro Learner
    description: "I had some challenges, but asking in the community really helped!"
    image: 
        url: "https://docs.astro.build/assets/rays.webp"
        alt: "The Astro logo on a dark background with rainbow rays."
    pubDate: 2022-07-15
    tags: ["astro", "learning in public", "setbacks", "community"]
    ---
    It wasn't always smooth sailing, but I'm enjoying building with Astro. And, the [Discord community](https://astro.build/chat) is really friendly and helpful!
    ```

3. Tambahkan tautan ke dua postingan baru ini:

    ```astro title="src/pages/blog.astro" ins={18-19}
    ---
    ---
    <html lang="en">
      <head>
        <meta charset="utf-8"/>
        <meta name="viewport" content="width=device-width" />
        <title>Astro</title>
      </head>
      <body>
        <a href="/">Home</a>
        <a href="/about/">About</a>
        <a href="/blog/">Blog</a>

        <h1>My Astro Learning Blog</h1>
        <p>This is where I will post about my journey learning Astro.</p>
        <ul>
          <li><a href="/posts/post-1/">Post 1</a></li>
          <li><a href="/posts/post-2/">Post 2</a></li>
          <li><a href="/posts/post-3/">Post 3</a></li>
        </ul>
      </body>
    </html>
    ```

4. Cek pratinjau peramban dan pastikan bahwa:

    Semua tautanmu untuk Post 1, Post 2, dan Post 3 mengarah ke halaman yang berfungsi pada situsmu. (Jika ada kesalahan, cek tautan di `blog.astro` atau nama berkas Markdown-mu.)
</Steps>
