Sekarang setelah situsmu bekerja di web, saatnya menambah halaman dan post!

## Melihat ke depan

Di unit ini, kamu akan:
- Membuat halaman Astro pertamamu dengan sintaks `.astro`
- Menambahkan postingan blog dengan file Markdown (`.md`)
- Memberi style pada halaman individual dengan `<style>` 
- Menerapkan style global ke seluruh halaman

 Sepanjang jalan, kamu akan mempelajari bagaimana **dua bagian dari file `.astro`** bekerja sama membuat halaman, serta cara menggunakan variabel dan conditional rendering pada halamanmu.
