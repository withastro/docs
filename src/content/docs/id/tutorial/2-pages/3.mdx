Sekarang kamu sudah punya situs multi-halaman dengan konten HTML, saatnya menambahkan HTML dinamis!

<PreCheck>
  - Definisikan judul halaman di *frontmatter*, lalu gunakan di HTML
  - Tampilkan elemen HTML secara kondisional
  - Tambahkan beberapa konten tentang dirimu
</PreCheck>

Berkas HTML apa pun adalah bahasa Astro yang valid. Namun, kamu bisa melakukan lebih banyak hal dengan Astro daripada HTML biasa!

## Definisikan dan gunakan sebuah variabel

Buka `about.astro`, yang seharusnya terlihat seperti ini:

```astro title="src/pages/about.astro"
---
---
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width" />
    <title>Astro</title>
  </head>
  <body>
    <a href="/">Home</a>
    <a href="/about/">About</a>
    <a href="/blog/">Blog</a>
    <h1>About Me</h1>
    <h2>... and my new Astro site!</h2>

    <p>I am working through Astro's introductory tutorial. This is the second page on my website, and it's the first one I built myself!</p>

    <p>This site will update as I complete more of the tutorial, so keep checking back and see how my journey is going!</p>
  </body>
</html>
```

<Steps>
1. Tambahkan baris JavaScript berikut di skrip *frontmatter*, di antara pagar kode (**code fences**):

    ```astro title="src/pages/about.astro" ins={2}
    ---
    const pageTitle = "About Me";
    ---
    ```

2. Gantikan judul "Astro" statis dan heading "About Me" di HTML dengan variabel dinamis `{pageTitle}`.

    ```astro title="src/pages/about.astro" del={5,12} ins={6,13}
    <html lang="en">
      <head>
        <meta charset="utf-8" />
        <meta name="viewport" content="width=device-width" />
        <title>Astro</title>
        <title>{pageTitle}</title>
      </head>
      <body>
        <a href="/">Home</a>
        <a href="/about/">About</a>
        <a href="/blog/">Blog</a>
        <h1>About Me</h1>
        <h1>{pageTitle}</h1>
        <h2>... and my new Astro site!</h2>

        <p>I am working through Astro's introductory tutorial. This is the second page on my website, and it's the first one I built myself!</p>

        <p>This site will update as I complete more of the tutorial, so keep checking back and see how my journey is going!</p>
      </body>
    </html>
    ```

3. Muat ulang pratinjau langsung halaman `/about`-mu.
  
    Teks halaman akan terlihat sama, dan judul halaman di tab peramban sekarang bertuliskan "About Me" alih-alih "Astro." 

    Alih-alih mengetik teks langsung ke *tag* HTML, kamu baru saja **mendefinisikan dan menggunakan variabel** di dua bagian berkas `.astro`-mu.

4. Gunakan pola yang sama untuk membuat nilai `pageTitle` pada `index.astro` ("Home Page") dan `blog.astro` ("My Astro Learning Blog"). Perbarui HTML kedua halaman itu di dua tempat agar judul halaman sesuai dengan heading yang ditampilkan di setiap halaman.
</Steps>

:::note[Rangkuman]
1. **Definisikan** variabel di skrip Astro menggunakan ekspresi JavaScript atau TypeScript.
2. **Gunakan** variabel tersebut di templat Astro di dalam kurung kurawal `{ }` untuk memberi tahu Astro bahwa kamu memakai JavaScript.
:::

## Tulis ekspresi JavaScript di Astro

<Steps>
1. Tambahkan JavaScript berikut ke *frontmatter*, di antara pagar kode (**code fences**):

      (Kamu boleh menyesuaikan sendiri, tetapi tutorial ini menggunakan contoh berikut.)

    ```astro title="src/pages/about.astro" ins={4-9, 11}
    ---
    const pageTitle = "About Me";

    const identity = {
      firstName: "Sarah",
      country: "Canada",
      occupation: "Technical Writer",
      hobbies: ["photography", "birdwatching", "baseball"],
    };

    const skills = ["HTML", "CSS", "JavaScript", "React", "Astro", "Writing Docs"];
    ---
    ```

2. Lalu, tambahkan kode berikut ke templat HTML-mu, di bawah konten yang sudah ada:

    ```astro title="src/pages/about.astro"
    <p>Here are a few facts about me:</p>
    <ul>
      <li>My name is {identity.firstName}.</li>
      <li>I live in {identity.country} and I work as a {identity.occupation}.</li>
      {identity.hobbies.length >= 2 && 
        <li>Two of my hobbies are: {identity.hobbies[0]} and {identity.hobbies[1]}</li>
      } 
    </ul>
    <p>My skills are:</p>
    <ul>
      {skills.map((skill) => <li>{skill}</li>)}
    </ul>
    ```
</Steps>

:::note[Rangkuman]
1. Menulis templat Astro sangat mirip dengan **menulis HTML**, namun kamu dapat menyertakan ekspresi JavaScript di dalamnya.
2. Skrip *frontmatter* Astro hanya berisi JavaScript. 
3. Kamu dapat menggunakan semua **operator logika**, **ekspresi**, dan **fungsi** JavaScript modern di salah satu bagian berkas `.astro`. Namun, kurung kurawal diperlukan (hanya) di tubuh templat HTML.
:::

## Render elemen secara kondisional

Kamu juga bisa menggunakan variabel skrip untuk menentukan **apakah** elemen tertentu pada konten `<body>` HTML ditampilkan atau tidak.

<Steps>
1. Tambahkan baris berikut ke skrip *frontmatter* untuk **mendefinisikan variabel**:

    ```astro title="src/pages/about.astro" ins={13-15}
    ---
    const pageTitle = "About Me";

    const identity = {
      firstName: "Sarah",
      country: "Canada",
      occupation: "Technical Writer",
      hobbies: ["photography", "birdwatching", "baseball"],
    };

    const skills = ["HTML", "CSS", "JavaScript", "React", "Astro", "Writing Docs"];

    const happy = true;
    const finished = false;
    const goal = 3;
    ---
    ```

2. Tambahkan baris berikut di bawah paragraf yang ada.

    Lalu, cek pratinjau langsung di peramban untuk melihat apa yang ditampilkan:

    ```astro title="src/pages/about.astro" /:|&&/ "?"
    {happy && <p>I am happy to be learning Astro!</p>}

    {finished && <p>I finished this tutorial!</p>}

    {goal === 3 ? <p>My goal is to finish in 3 days.</p> : <p>My goal is not 3 days.</p>}
    ```

3. *Commit* perubahanmu ke GitHub sebelum lanjut. Lakukan ini kapan saja kamu ingin menyimpan pekerjaan dan memperbarui situs langsungmu.
</Steps>

:::tip
Sintaks templat Astro mirip dengan JSX. Jika kamu bingung bagaimana menggunakan skrip di HTML, cari cara melakukannya di JSX sebagai titik awal yang baik!
:::

### Analisis Pola

Diberikan skrip `.astro` berikut:

```astro title="src/pages/about.astro"
---
const operatingSystem = "Linux";
const quantity = 3;
const footwear = "boots";
const student = false;
---
```

Untuk setiap ekspresi templat Astro berikut, bisakah kamu menebak HTML (jika ada!) yang akan dikirim ke peramban? Klik untuk melihat apakah tebakanmu benar!

1.  `<p>{operatingSystem}</p>`

    <p>
      <Spoiler>`<p>Linux</p>`</Spoiler>
    </p>

2.  `{student && <p>I am still in school.</p>}`

    <p>
      <Spoiler>Tidak ada yang tampil karena `student` bernilai *false*.</Spoiler>
    </p>

3.  `<p>I have {quantity + 8} pairs of {footwear}</p>`

    <p>
      <Spoiler>`<p>I have 11 pairs of boots</p>`</Spoiler>
    </p>

4.  `{operatingSystem === "MacOS" ? <p>I am using a Mac.</p> : <p>I am not using a Mac.</p>}`

    <p>
      <Spoiler>`<p>I am not using a Mac.</p>`</Spoiler>
    </p>
</Box>
