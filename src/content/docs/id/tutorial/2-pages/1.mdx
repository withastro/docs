Sekarang kamu tahu bahwa berkas `.astro` bertanggung jawab atas halaman di situs webmu, saatnya membuat satu!

<PreCheck>
  - Buat dua halaman baru di situsmu: About dan Blog
  - Tambahkan tautan navigasi ke halaman-halamanmu
  - *Deploy* versi terbaru situsmu ke web
</PreCheck>

## Buat berkas `.astro` baru

<Steps>
1. Di panel berkas editor kode, buka folder `src/pages/` tempat kamu akan melihat berkas `index.astro` yang sudah ada.

2. Di folder yang sama, buat berkas baru bernama `about.astro`.

3. Salin, atau ketik ulang isi `index.astro` ke berkas barumu `about.astro`.

    :::tip
    Editor kamu mungkin menampilkan lingkaran putih solid pada label tab berkas ini. Itu berarti berkas belum disimpan. Di menu File pada VS Code, aktifkan "Auto Save" sehingga kamu tidak perlu menyimpan berkas secara manual lagi.
    :::

4. Tambahkan `/about` pada akhir URL pratinjau situsmu di bilah alamat dan cek bahwa halaman dapat dimuat di sana. (mis. `http://localhost:4321/about`)
</Steps>

Saat ini, halaman "About" kamu seharusnya terlihat persis sama dengan halaman pertama, tapi kita akan mengubahnya!

## Edit halamanmu

Edit konten HTML agar halaman ini tentang dirimu.

Untuk mengubah atau menambah konten pada halaman About, tambahkan lebih banyak *tag* elemen HTML berisi konten. Kamu bisa menyalin dan menempelkan kode HTML di bawah ini di antara *tag* `<body></body>` yang ada, atau buat sendiri.

```astro title="src/pages/about.astro" ins={3-8} del={2}
<body>
  <h1>My Astro Site</h1>
  <h1>About Me</h1>
  <h2>... and my new Astro site!</h2>

  <p>I am working through Astro's introductory tutorial. This is the second page on my website, and it's the first one I built myself!</p>

  <p>This site will update as I complete more of the tutorial, so keep checking back and see how my journey is going!</p>
</body>
```

Sekarang, kunjungi lagi halaman `/about` di tab perambanmu, dan kamu akan melihat konten yang diperbarui.

## Tambahkan tautan navigasi

Agar lebih mudah melihat semua halaman, tambahkan tautan navigasi HTML sebelum `<h1>` di bagian atas dua halamanmu (`index.astro` dan `about.astro`):

```astro title="src/pages/about.astro" ins={1-2}
<a href="/">Home</a>
<a href="/about/">About</a>

<h1>About Me</h1>
<h2>... and my new Astro site!</h2>
```

Cek bahwa kamu bisa mengklik tautan untuk berpindah antarhalaman di situsmu.

:::note
Tidak seperti banyak kerangka kerja, Astro menggunakan elemen HTML standar `<a>` untuk bernavigasi antarhalaman (disebut juga _routes_), dengan penyegaran halaman tradisional.
:::

## Coba sendiri - Tambahkan halaman Blog

Tambahkan halaman ketiga `blog.astro` ke situsmu, mengikuti [langkah yang sama seperti di atas](#buat-berkas-astro-baru).

(Jangan lupa menambahkan tautan navigasi ketiga di setiap halaman.)

<details>
<summary>Tunjukkan langkah-langkahnya.</summary>
<Steps>
1. Buat berkas baru di `src/pages/blog.astro`.
2. Salin seluruh isi `index.astro` dan tempelkan ke `blog.astro`.
3. [Tambahkan tautan navigasi ketiga](#tambahkan-tautan-navigasi) di bagian atas setiap halaman:
</Steps>

```astro title="src/pages/index.astro" ins={4}
<body>
  <a href="/">Home</a>
  <a href="/about/">About</a>
  <a href="/blog/">Blog</a>

  <h1>My Astro Site</h1>
</body>
```
</details>
</Box>

Sekarang kamu mestinya punya situs dengan tiga halaman yang saling menaut. Saatnya menambahkan konten ke halaman Blog.

Perbarui konten halaman di `blog.astro` dengan:
```astro astro title="src/pages/blog.astro" ins={7-8} del={6}
<body>
  <a href="/">Home</a>
  <a href="/about/">About</a>
  <a href="/blog/">Blog</a>

  <h1>My Astro Site</h1>
  <h1>My Astro Learning Blog</h1>
  <p>This is where I will post about my journey learning Astro.</p>
</body>
```

Pratinjau seluruh situsmu dengan mengunjungi ketiga halaman di peramban dan cek bahwa:
- Setiap halaman menaut ke ketiga halaman dengan benar
- Dua halaman barumu masing-masing punya heading yang deskriptif
- Dua halaman barumu masing-masing punya teks paragraf sendiri

## Publikasikan perubahanmu ke web
Jika kamu mengikuti *setup* di Unit 1, kamu bisa memublikasikan perubahan ke situs langsungmu melalui Netlify.

Saat sudah puas dengan tampilan pratinjau, **commit** perubahanmu ke repositori daring di GitHub. 

<Steps>
1. Di VS Code, pratinjau berkas yang berubah sejak *commit* terakhir ke GitHub. 

    - Buka **Source Control tab** di menu kiri. Seharusnya ada angka kecil "3".

    - Kamu akan melihat `index.astro`, `about.astro`, dan `blog.astro` tercantum sebagai berkas yang berubah.

2. Masukkan pesan commit (mis. "Tambah dua halaman baru - about dan blog") pada kotak teks, lalu tekan <kbd>Ctrl + Enter</kbd> (macOS: <kbd>Cmd ⌘ + Enter</kbd>) untuk melakukan commit ke *workspace* saat ini.

3. Klik tombol <kbd>Sync Changes</kbd> ke GitHub.

4. Setelah menunggu beberapa menit, kunjungi URL Netlify-mu untuk memastikan perubahan sudah tayang.
</Steps>

:::tip[Commit dan deploy secara berkala]
Ikuti langkah-langkah ini setiap kali kamu jeda bekerja! Perubahanmu akan diperbarui di repositori GitHub. Jika kamu *deploy* ke Netlify, situsmu akan dibangun ulang dan dipublikasikan ulang.
:::
