Sekarang kamu sudah memiliki beberapa posting blog, saatnya menggunakan API Astro untuk bekerja dengan file-filemu!

## Melihat ke depan

Di unit ini, kamu akan menyupercharge blog-mu dengan sebuah halaman indeks, halaman tag, dan RSS feed. 

Sepanjang jalan, kamu akan belajar cara menggunakan:
- `import.meta.glob()` untuk mengakses data dari file di proyekmu
- `getStaticPaths()` untuk membuat banyak halaman (rute) sekaligus
- Paket Astro RSS untuk membuat RSS feed
