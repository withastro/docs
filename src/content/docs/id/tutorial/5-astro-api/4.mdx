## Instal paket RSS milik Astro

Astro menyediakan paket khusus untuk menambahkan RSS feed ke websitemu dengan cepat. 

Paket resmi ini menghasilkan dokumen non-HTML berisi informasi mengenai semua posting blog-mu yang bisa dibaca oleh **feed reader** seperti Feedly, The Old Reader, dan lainnya. Dokumen ini akan diperbarui setiap kali situsmu dibangun ulang.

Orang dapat berlangganan feed-mu di feed reader, dan menerima notifikasi ketika kamu menerbitkan posting blog baru di situsmu, menjadikannya fitur blog yang populer.

<Steps>
1. Di terminal, hentikan server pengembangan Astro (<kbd>Ctrl + C</kbd>) dan jalankan perintah berikut untuk memasang paket RSS milik Astro.

    <PackageManagerTabs>
      <Fragment slot="npm">
      ```shell
      npm install @astrojs/rss
      ```
      </Fragment>
      <Fragment slot="pnpm">
      ```shell
      pnpm add @astrojs/rss
      ```
      </Fragment>
      <Fragment slot="yarn">
      ```shell
      yarn add @astrojs/rss
      ```
      </Fragment>
    </PackageManagerTabs>

2. Mulai ulang dev server untuk kembali mengerjakan proyek Astromu.

    <PackageManagerTabs>
      <Fragment slot="npm">
      ```shell
      npm run dev
      ```
      </Fragment>
      <Fragment slot="pnpm">
      ```shell
      pnpm run dev
      ```
      </Fragment>
      <Fragment slot="yarn">
      ```shell
      yarn run dev
      ```
      </Fragment>
    </PackageManagerTabs>
</Steps>

## Buat dokumen feed `.xml`

<Steps>
1. Buat file baru di `src/pages/` bernama `rss.xml.js`

2. Salin kode berikut ke dokumen baru ini. Sesuaikan properti `title` dan `description`, dan jika perlu, tentukan bahasa berbeda di `customData`:

    ```js title="src/pages/rss.xml.js"

    import rss, { pagesGlobToRssItems } from '@astrojs/rss';

    export async function GET(context) {
      return rss({
        title: 'Astro Learner | Blog',
        description: 'My journey learning Astro',
        site: context.site,
        items: await pagesGlobToRssItems(import.meta.glob('./**/*.md')),
        customData: `<language>en-us</language>`,
      });
    }
    ```

3. Tambahkan properti `site` ke konfigurasi Astro dengan URL Netlify unik situsmu.

    ```js title="astro.config.mjs" ins={4}
    import { defineConfig } from "astro/config";

    export default defineConfig({
      site: "https://example.com"
    });
    ```
    
4. Buka `http://localhost:4321/rss.xml` dan pastikan kamu dapat melihat teks (tidak terformat) di halaman dengan `item` untuk setiap file `.md` milikmu. Setiap item harus berisi informasi posting blog seperti `title`, `url`, dan `description`.

    :::tip[Lihat RSS feed-mu di reader]
    Unduh feed reader, atau daftar layanan feed reader online dan berlangganan situsmu dengan menambahkan URL Netlify-mu sendiri. Kamu juga bisa membagikan tautan ini agar orang lain dapat berlangganan postinganmu, dan diberi tahu ketika ada yang baru.
    :::

</Steps>
