Sekarang setelah Anda bisa membangun dengan komponen, saatnya membuat beberapa layout kustom!

## Melihat ke depan

Di unit ini, Anda akan membangun layout untuk berbagi elemen dan gaya umum di seluruh halaman serta postingan blog Anda.

Untuk melakukannya, Anda akan:

- Membuat komponen layout yang dapat digunakan ulang
- Mengirim konten ke layout dengan `<slot />`
- Meneruskan data dari frontmatter Markdown ke layout
- Menyusun (nesting) beberapa layout
