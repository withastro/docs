Sekarang setelah Anda menambahkan layout ke setiap postingan, saatnya membuat postingan Anda tampil seperti halaman lain di situs!

<PreCheck>
  - Susun (nest) layout postingan blog di dalam layout halaman utama
</PreCheck>

## Susun dua layout Anda

Anda sudah memiliki `BaseLayout.astro` untuk mendefinisikan layout umum halaman.

`MarkdownPostLayout.astro` memberi templating tambahan untuk properti posting umum seperti `title` dan `date`, tetapi postingan Anda belum tampak seperti halaman lain di situs. Anda dapat menyamakan tampilannya dengan **menyatukan (nesting) layout**.

<Steps>
1. Di `src/layouts/MarkdownPostLayout.astro`, impor `BaseLayout.astro` dan gunakan untuk membungkus seluruh konten template. Jangan lupa teruskan prop `pageTitle`:

    ```astro title="src/layouts/MarkdownPostLayout.astro" ins={2,5,13}
    ---
    import BaseLayout from './BaseLayout.astro';
    const { frontmatter } = Astro.props;
    ---
    <BaseLayout pageTitle={frontmatter.title}>
      <meta charset="utf-8" />
      <h1>{frontmatter.title}</h1>
      <p>{frontmatter.pubDate.toString().slice(0,10)}</p>
      <p><em>{frontmatter.description}</em></p>
      <p>Written by: {frontmatter.author}</p>
      <img src={frontmatter.image.url} width="300" alt={frontmatter.image.alt} />
      <slot />
    </BaseLayout>
    ```

2. Di `src/layouts/MarkdownPostLayout.astro`, Anda kini dapat menghapus tag `meta` karena sudah disertakan di `BaseLayout`:

    ```astro title="src/layouts/MarkdownPostLayout.astro" del={6}
    ---
    import BaseLayout from './BaseLayout.astro';
    const { frontmatter } = Astro.props;
    ---
    <BaseLayout pageTitle={frontmatter.title}>
      <meta charset="utf-8" />
      <h1>{frontmatter.title}</h1>
      <p>{frontmatter.pubDate.toString().slice(0,10)}</p>
      <p><em>{frontmatter.description}</em></p>
      <p>Written by: {frontmatter.author}</p>
      <img src={frontmatter.image.url} width="300" alt={frontmatter.image.alt} />
      <slot />
    </BaseLayout>
    ```

3. Cek pratinjau di `http://localhost:4321/posts/post-1`. Kini Anda harus melihat konten yang dirender oleh:

    - **Layout halaman utama**, termasuk gaya, tautan navigasi, dan footer sosial.
    - **Layout postingan blog**, termasuk properti frontmatter seperti deskripsi, tanggal, judul, dan gambar.
    - **Konten Markdown postingan individual**, yaitu teks yang Anda tulis pada postingan tersebut.

4. Perhatikan judul halaman kini ditampilkan dua kali, masing‑masing oleh setiap layout.

    Hapus baris yang menampilkan judul dari `MarkdownPostLayout.astro`:

    ```astro title="src/layouts/MarkdownPostLayout.astro" del={2}
    <BaseLayout pageTitle={frontmatter.title}>
      <h1>{frontmatter.title}</h1>
      <p>{frontmatter.pubDate.toString().slice(0,10)}</p>
      <p><em>{frontmatter.description}</em></p>
      <p>Written by: {frontmatter.author}</p>
      <img src={frontmatter.image.url} width="300" alt={frontmatter.image.alt} />
      <slot />
    </BaseLayout>
    ```

5. Periksa lagi pratinjau di `http://localhost:4321/posts/post-1` dan pastikan baris tersebut sudah tidak tampil dan judul hanya muncul sekali. Lakukan penyesuaian lain jika perlu untuk memastikan tidak ada konten yang terduplikasi.
</Steps>


Pastikan bahwa:

  - Setiap postingan menampilkan template halaman yang sama, dan tidak ada konten yang hilang. (Jika ada konten yang hilang, periksa properti frontmatter postingan tersebut.)

  - Tidak ada konten yang duplikat pada halaman. (Jika sesuatu dirender dua kali, hapus dari `MarkdownPostLayout.astro`.)

  Jika ingin, Anda bebas menyesuaikan template halaman Anda.
