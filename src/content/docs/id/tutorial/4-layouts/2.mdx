## Tambahkan layout ke postingan blog Anda

Saat Anda menyertakan properti `layout` di frontmatter berkas `.md`, semua nilai frontmatter YAML Anda tersedia untuk berkas layout.

<Steps>
1. Buat file baru di `src/layouts/MarkdownPostLayout.astro`

2. Salin kode berikut ke `MarkdownPostLayout.astro`

    ```astro title="src/layouts/MarkdownPostLayout.astro"
    ---
    const { frontmatter } = Astro.props;
    ---
    <meta charset="utf-8" />
    <h1>{frontmatter.title}</h1>
    <p>Ditulis oleh {frontmatter.author}</p>
    <slot />
    ```

3. Tambahkan properti frontmatter berikut di `post-1.md`

    ```markdown title="src/pages/posts/post-1.md" ins={2}
    ---
    layout: ../../layouts/MarkdownPostLayout.astro
    title: 'My First Blog Post'
    pubDate: 2022-07-01
    description: 'This is the first post of my new Astro blog.'
    author: 'Astro Learner'
    image:
        url: 'https://docs.astro.build/assets/rose.webp' 
        alt: 'The Astro logo on a dark background with a pink glow.'
    tags: ["astro", "blogging", "learning in public"]
    ---
    ```

4. Periksa lagi pratinjau browser di `http://localhost:4321/posts/post-1` dan perhatikan apa saja yang ditambahkan layout ke halaman Anda. 

5. Tambahkan properti layout yang sama ke dua postingan lain `post-2.md` dan `post-3.md`. Verifikasi di browser bahwa layout juga diterapkan ke postingan‑postingan tersebut.
</Steps>

:::tip
Saat menggunakan layout, Anda kini dapat memilih untuk menyertakan elemen (seperti judul halaman) di konten Markdown atau di layout. Ingatlah untuk memeriksa pratinjau secara visual dan lakukan penyesuaian jika perlu agar tidak terjadi duplikasi elemen. 
:::

## Coba sendiri — Kustomisasi layout postingan blog Anda

**Tantangan**: Identifikasi item yang umum untuk setiap postingan blog, dan gunakan `MarkdownPostLayout.astro` untuk merendernya, alih‑alih menuliskannya di Markdown pada `post-1.md` dan setiap postingan berikutnya.

Berikut contoh refactor untuk menyertakan `pubDate` di komponen layout alih‑alih menulisnya di badan Markdown:

```markdown title="src/pages/posts/post-1.md" del={1}
Published on: 2022-07-01

Welcome to my _new blog_ about learning Astro! Here, I will share my learning journey as I build a new website.
```

```astro title="src/layouts/MarkdownPostLayout.astro" ins={6}
---
const { frontmatter } = Astro.props;
---
<meta charset="utf-8" />
<h1>{frontmatter.title}</h1>
<p>Published on: {frontmatter.pubDate.toString().slice(0,10)}</p>
<p>Written by {frontmatter.author}</p>
<slot />
```

Refactor sebanyak yang menurut Anda berguna, dan tambahkan sebanyak yang Anda inginkan ke layout—ingat bahwa setiap hal yang Anda tambahkan ke layout adalah satu hal yang tidak perlu Anda tulis di setiap postingan blog!

Berikut contoh layout yang telah di‑refactor sehingga hanya konten postingan individual yang dirender oleh slot. Silakan gunakan ini atau buat versi Anda sendiri! 

```astro title="src/layouts/MarkdownPostLayout.astro"
---
const { frontmatter } = Astro.props;
---
<meta charset="utf-8" />
<h1>{frontmatter.title}</h1>
<p>{frontmatter.pubDate.toString().slice(0,10)}</p>
<p><em>{frontmatter.description}</em></p>
<p>Written by: {frontmatter.author}</p>
<img src={frontmatter.image.url} width="300" alt={frontmatter.image.alt} />
<slot />
```
</Box>

:::note[Hindari duplikasi]
Apa pun yang dirender oleh layout **tidak** perlu Anda ketik di postingan! Jika Anda melihat duplikasi saat memeriksa pratinjau, pastikan untuk menghapus konten dari berkas Markdown.
 :::

### Uji pengetahuan Anda
Bisakah Anda mengisi bagian kosong agar kedua komponen ini menghasilkan kode Astro yang berfungsi?

1.  ```markdown title="src/pages/posts/learning-astro.md"
    ---
    layout: ../../__________/MyMarkdownLayout.astro
    title: "Learning About Markdown in Astro"
    author: Astro Learner
    ____: 2022-08-08
    ---
    I learned so much today! Astro allows me to write in Markdown, but also use variables from the frontmatter. I can even access those values in an Astro layout component.
    ```

2.  ```astro title="src/layouts/MyMarkdownLayout.astro"
    ---
    import ____________ from '../components/Footer.astro'
    const { ___________ } = Astro.props
    ---
    <h1>{frontmatter.title}</h1>
    <p>Written by: {frontmatter.______} on {frontmatter.pubDate}</p>
    < _______ />
    <Footer />
    ```
    <details>
