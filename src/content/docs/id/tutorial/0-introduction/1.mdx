<details>
<summary>Bagaimana cara menggunakan checklist di bagian bawah setiap halaman?</summary>

Anda tinggal mencentangnya!

Di akhir tiap halaman, Anda akan menemukan checklist tugas yang bisa Anda kerjakan sekarang. Centang item-item ini untuk melihat progres Anda di Tutorial Tracker.

Menggunakan tracker bersifat opsional, tetapi dapat membantu Anda mengingat progres jika menyelesaikan tutorial dalam beberapa kali kunjungan. Anda juga bisa membiarkan sebagian checklist kosong sebagai pengingat unit yang ingin Anda kunjungi kembali nanti.

(Data ini hanya disimpan di local storage browser Anda dan tidak tersedia di tempat lain. Tidak ada data yang dikirimkan ke, atau disimpan oleh Astro.)

<details>
<summary>Unit 1 berisi hal-hal yang sudah saya kuasai. Boleh saya lewati?</summary>

Anda dapat menggunakan pelajaran di dalam [Unit 1](/id/tutorial/1-setup/) untuk memastikan Anda memiliki alat pengembangan dan akun online yang diperlukan untuk menyelesaikan tutorial. Unit ini akan memandu Anda membuat proyek Astro baru, menyimpannya di GitHub, dan deploy ke Netlify.

Jika Anda membuat [proyek Astro baru yang kosong](/id/install-and-setup/) dan sudah nyaman dengan setup Anda, Anda bisa langsung lompat ke [Unit 2](/id/tutorial/2-pages/) di mana Anda akan mulai membuat halaman baru di proyek Anda.
</details>
