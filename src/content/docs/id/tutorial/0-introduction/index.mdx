Sepanjang perjalanan, Anda akan: 
- Menyiapkan lingkungan pengembangan 
- Membuat halaman dan posting blog untuk situs Anda
- Membangun dengan komponen Astro
- Query dan bekerja dengan berkas lokal
- Menambahkan interaktivitas ke situs Anda 
- Deploy situs Anda ke web

Ingin melihat pratinjau dari apa yang akan Anda bangun? Anda bisa melihat proyek akhirnya di GitHub: https://github.com/withastro/blog-tutorial-demo atau membuka versi yang berjalan di lingkungan coding daring seperti IDX (https://idx.google.com/import?url=https:%2F%2Fgithub.com%2Fwithastro%2Fblog-tutorial-demo%2F) atau StackBlitz (https://stackblitz.com/github/withastro/blog-tutorial-demo/tree/complete?file=src%2Fpages%2Findex.astro).


:::note
Jika Anda lebih suka mulai menjelajah Astro dengan situs yang sudah jadi, kunjungi https://astro.new lalu pilih starter template untuk dibuka dan diedit di editor online.
:::
