Sekarang setelah kamu menggunakan komponen Astro di halaman, saatnya menggunakan **komponen di dalam komponen** lainnya!

## Buat Komponen Footer
<Steps>
1. Buat file baru di `src/components/Footer.astro`.

2. Salin kode berikut ke file barumu, `Footer.astro`.

    ```astro title="src/components/Footer.astro"
    ---
    const platform = "github";
    const username = "withastro";
    ---

    <footer>
      <p>Pelajari lebih lanjut proyekku di <a href={`https://www.${platform}.com/${username}`}>{platform}</a>!</p>
    </footer>
    ```
</Steps>

### Import dan gunakan `Footer.astro`

<Steps>
1. Tambahkan pernyataan import berikut ke frontmatter di **setiap** dari tiga halaman Astro-mu (`index.astro`, `about.astro`, dan `blog.astro`):

    ```js
    import Footer from '../components/Footer.astro';
    ```

2. Tambahkan komponen `<Footer />` baru pada template Astro di setiap halaman, tepat sebelum tag penutup `</body>` agar footer tampil di bagian bawah halaman.

    ```astro ins={1}
        <Footer />
      </body>
    </html>
    ```

3. Di pratinjau browser, pastikan kamu bisa melihat teks footer baru di setiap halaman.
</Steps>

## Coba sendiri — Personalisasikan footer-mu

Kustomisasi footer untuk menampilkan beberapa jejaring sosial (mis. Instagram, Twitter, LinkedIn) dan sertakan nama pengguna milikmu agar langsung terhubung ke profilmu.
</Box>

### Cek Kode
Jika kamu mengikuti setiap langkah tutorial, file `index.astro`-mu seharusnya seperti ini:

```astro title="src/pages/index.astro"
---
import Navigation from '../components/Navigation.astro';
import Footer from '../components/Footer.astro';
import '../styles/global.css';

const pageTitle = 'Home Page';
---

<html lang="en">
	<head>
		<meta charset="utf-8" />
		<link rel="icon" type="image/svg+xml" href="/favicon.svg" />
		<meta name="viewport" content="width=device-width" />
		<meta name="generator" content={Astro.generator} />
		<title>{pageTitle}</title>
	</head>
	<body>
    <Navigation />
		<h1>{pageTitle}</h1>
		<Footer />
	</body>
</html>
```

## Buat komponen Media Sosial

Karena kamu mungkin punya beberapa akun online untuk ditautkan, kamu dapat membuat satu komponen yang dapat digunakan ulang dan menampilkannya berkali-kali. Setiap kali, kamu akan meneruskan properti (`props`) yang berbeda: platform dan nama pengguna.

<Steps>
1. Buat file baru di `src/components/Social.astro`.

2. Salin kode berikut ke `Social.astro`.

    ```astro title="src/components/Social.astro"
    ---
    const { platform, username } = Astro.props;
    ---
    <a href={`https://www.${platform}.com/${username}`}>{platform}</a>
    ```
</Steps>

### Import dan gunakan `Social.astro` di Footer

<Steps>
1. Ubah kode di `src/components/Footer.astro` untuk mengimpor lalu menggunakan komponen baru ini tiga kali, dengan **atribut komponen** berbeda sebagai props tiap kali:

    ```astro title="src/components/Footer.astro" del={2,3,8} ins={4,9-11}
    ---
    const platform = "github";
    const username = "withastro";
    import Social from './Social.astro';
    ---

    <footer>
      <p>Pelajari lebih lanjut proyekku di <a href={`https://www.${platform}.com/${username}`}>{platform}</a>!</p>
      <Social platform="twitter" username="astrodotbuild" />
      <Social platform="github" username="withastro" />
      <Social platform="youtube" username="astrodotbuild" />
    </footer>
    ```

2. Cek pratinjau browser: kamu akan melihat footer menampilkan tautan ke tiga platform tersebut di setiap halaman.
</Steps>

## Gaya untuk Komponen Media Sosial

<Steps>
1. Kustomisasi tampilan tautan dengan menambahkan tag `<style>` ke `src/components/Social.astro`.

    ```astro title="src/components/Social.astro" ins={6-17} 'class="social-platform'
    ---
    const { platform, username } = Astro.props;
    ---
    <a href={`https://www.${platform}.com/${username}`}>{platform}</a>

    <style>
      a {
        padding: 0.5rem 1rem;
        color: white;
        background-color: #4c1d95;
        text-decoration: none;
      }
    </style>
    ```

2. Tambahkan tag `<style>` ke `src/components/Footer.astro` untuk memperbaiki tata letaknya.

    ```astro title="src/components/Footer.astro" ins={4-10}
    ---
    import Social from './Social.astro';
    ---
    <style>
      footer {
        display: flex;
        gap: 1rem;
        margin-top: 2rem;
      }
    </style>

    <footer>
      <Social platform="twitter" username="astrodotbuild" />
      <Social platform="github" username="withastro" />
      <Social platform="youtube" username="astrodotbuild" />
    </footer>
    ```

3. Periksa lagi pratinjau browser dan pastikan setiap halaman menampilkan footer yang sudah diperbarui.
</Steps>
