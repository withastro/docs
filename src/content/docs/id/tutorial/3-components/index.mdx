Sekarang setelah file `.astro` dan `.md`-mu menghasilkan **halaman penuh** di website, saatnya membuat dan menggunakan kembali potongan HTML yang lebih kecil dengan **komponen Astro**!

## Melihat ke depan

Di unit ini, kamu akan belajar membuat **komponen Astro** untuk menggunakan kembali kode elemen umum di seluruh website-mu.

Kamu akan membangun:
- Komponen Navigation yang menampilkan menu tautan ke halamanmu
- Komponen Footer untuk disertakan di bagian bawah setiap halaman
- Komponen Media Sosial, digunakan di Footer, yang menaut ke halaman profil
- Komponen Menu interaktif untuk toggle Navigation di mobile

Sepanjang jalan, kamu akan menggunakan CSS dan JavaScript untuk membangun desain yang responsif terhadap ukuran layar dan input pengguna.
