Sekarang kamu punya HTML yang sama ditulis di beberapa halaman situs Astro-mu, saatnya mengganti konten duplikat itu dengan **komponen Astro** yang dapat digunakan kembali!

<PreCheck>
  - Buat folder baru untuk komponen
  - Bangun komponen Astro untuk menampilkan tautan navigasi
  - Ganti HTML yang ada dengan komponen navigasi yang dapat digunakan ulang
</PreCheck>

## Buat folder `src/components/` baru

Untuk menampung file `.astro` yang akan menghasilkan HTML tetapi **tidak** menjadi halaman baru di websitemu, kamu memerlukan folder baru di proyekmu: `src/components/`.

## Buat komponen Navigation

<Steps>
1. Buat file baru: `src/components/Navigation.astro`.

2. Salin tautan navigasi antarkomponen dari bagian atas salah satu halaman dan tempel ke file barumu, `Navigation.astro`:

    ```astro title="src/components/Navigation.astro" "---"
    ---
    ---
    <a href="/">Home</a>
    <a href="/about/">About</a>
    <a href="/blog/">Blog</a>
    ```
    :::tip
    Jika tidak ada apa pun di frontmatter file `.astro`-mu, kamu tidak harus menulis pagar kode (code fences) tersebut. Kamu selalu bisa menambahkannya kembali saat diperlukan.
    :::
</Steps>

### Import dan gunakan `Navigation.astro`
<Steps>
1. Kembali ke `index.astro` dan import komponen barumu di dalam code fence:

    ```astro title="src/pages/index.astro" ins={2}
    ---
    import Navigation from '../components/Navigation.astro';
    import "../styles/global.css";

    const pageTitle = "Home Page";
    ---
    ```

2. Lalu, di bawahnya ganti elemen tautan HTML navigasi yang ada dengan komponen navigasi yang baru kamu impor:

    ```astro title="src/pages/index.astro" del={1-3} ins={4}
    <a href="/">Home</a>
    <a href="/about/">About</a>
    <a href="/blog/">Blog</a>
    <Navigation />
    ```

3. Periksa pratinjau di browser: tampilannya seharusnya **tetap sama**… dan itulah yang kita inginkan!
</Steps>

Situsmu berisi HTML yang sama seperti sebelumnya. Namun kini, tiga baris kode itu disediakan oleh komponen `<Navigation />`.

<Box icon="puzzle-piece">

## Coba sendiri — Tambahkan navigasi ke seluruh situs

Import dan gunakan komponen `<Navigation />` di dua halaman lainnya (`about.astro` dan `blog.astro`) dengan cara yang sama.

Jangan lupa untuk:
- Menambahkan pernyataan `import` di bagian atas script komponen, di dalam code fence.
- Mengganti kode tautan yang ada dengan komponen navigasi.
</Box>

:::note
Saat kamu menata ulang kode tetapi **tidak** mengubah tampilan halaman di browser, itu disebut **refactoring**. Kamu akan **melakukan refactor** beberapa kali di unit ini ketika mengganti bagian HTML halaman dengan komponen.

Ini memungkinkanmu memulai dengan cepat menggunakan kode apa pun yang bekerja (meski duplikatif). Lalu, kamu dapat meningkatkan desain kode secara bertahap tanpa mengubah tampilan luar situsmu.
:::
