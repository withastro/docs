Karena situsmu akan dilihat di berbagai perangkat, saatnya membuat navigasi halaman yang dapat merespons berbagai ukuran layar!

<PreCheck>
  - Buat Header untuk situsmu yang berisi komponen Navigation
  - Jadikan komponen Navigation responsif
</PreCheck>

<Box icon="puzzle-piece">

## Coba sendiri — Bangun komponen Header baru

<Steps>
1. Buat komponen Header. Import dan gunakan komponen `Navigation.astro`-mu di dalam elemen `<nav>` yang berada di dalam elemen `<header>`.

    <details>
    <summary>Tunjukkan kodenya!</summary>

    Buat file bernama `Header.astro` di `src/components/`
    ```astro title="src/components/Header.astro"
    ---
    import Navigation from './Navigation.astro';
    ---
    <header>
      <nav>
        <Navigation />
      </nav>
    </header>
    ```

    </details>
</Steps>

</Box>

<Box icon="puzzle-piece">

## Coba sendiri — Perbarui halaman-halamanmu

<Steps>
1. Di setiap halaman, ganti komponen `<Navigation/>` yang ada dengan header barumu.

    <details>
    <summary>Tunjukkan kodenya!</summary>

    ```astro title="src/pages/index.astro" ins={3,18} del={2,17}
    ---
    import Navigation from '../components/Navigation.astro';
    import Header from '../components/Header.astro';
    import Footer from '../components/Footer.astro';
    import '../styles/global.css';
    const pageTitle = "Home Page";
    ---
    <html lang="en">
      <head>
        <meta charset="utf-8" />
        <link rel="icon" type="image/svg+xml" href="/favicon.svg" />
        <meta name="viewport" content="width=device-width" />
        <meta name="generator" content={Astro.generator} />
        <title>{pageTitle}</title>
      </head>
      <body>
        <Navigation />
        <Header />
        <h1>{pageTitle}</h1>
        <Footer />
      </body>
    </html>
    ```
    </details>

2. Periksa pratinjau browser dan pastikan header ditampilkan di setiap halaman. Tampilannya belum berubah, tetapi jika kamu menginspeksi pratinjau menggunakan dev tools, kamu akan melihat bahwa kini kamu punya elemen seperti `<header>` dan `<nav>` mengelilingi tautan navigasi.
</Steps>
</Box>

## Tambahkan gaya responsif

<Steps>
1. Perbarui `Navigation.astro` dengan kelas CSS untuk mengontrol tautan navigasi. Bungkus tautan navigasi yang ada di dalam `<div>` dengan kelas `nav-links` dan atribut id `main-menu`.

    ```astro title="src/components/Navigation.astro" ins={3,7}
    ---
    ---
    <div id="main-menu" class="nav-links">
      <a href="/">Home</a>
      <a href="/about/">About</a>
      <a href="/blog/">Blog</a>
    </div>
    ```

2. Salin gaya CSS di bawah ini ke `global.css`. Gaya ini:

    - Menata dan memosisikan tautan navigasi untuk perangkat mobile
    - Menyertakan kelas `expanded` yang dapat di-toggle untuk menampilkan/menyembunyikan tautan pada mobile
    - Menggunakan `@media` query untuk mendefinisikan gaya berbeda di ukuran layar yang lebih besar

    :::tip[Desain mobile-first]
    Mulailah dengan mendefinisikan perilaku untuk layar kecil terlebih dahulu! Layar kecil membutuhkan tata letak yang lebih sederhana. Lalu, sesuaikan gaya untuk perangkat yang lebih besar. Jika kamu mendesain kasus rumit terlebih dahulu, kamu harus bekerja ekstra untuk menyederhanakannya lagi.
    :::

    ```css title="src/styles/global.css" ins={23-60}
    html {
      background-color: #f1f5f9;
      font-family: sans-serif;
    }

    body {
      margin: 0 auto;
      width: 100%;
      max-width: 80ch;
      padding: 1rem;
      line-height: 1.5;
    }

    * {
      box-sizing: border-box;
    }

    h1 {
      margin: 1rem 0;
      font-size: 2.5rem;
    }

    /* nav styles */

    .nav-links {
      width: 100%;
      display: none;
      margin: 0;
    }

    .nav-links a {
      display: block;
      text-align: center;
      padding: 10px 0;
      text-decoration: none;
      font-size: 1.2rem;
      font-weight: bold;
      text-transform: uppercase;
      color: #0d0950;
    }

    .nav-links a:hover,
    .nav-links a:focus {
      background-color: #ff9776;
    }

    @media screen and (min-width: 636px) {
      .nav-links {
        margin-left: 5em;
        display: block;
        position: static;
        width: auto;
        background: none;
      }

      .nav-links a {
        display: inline-block;
        padding: 15px 20px;
      }
    }
    ```
</Steps>

Ubah ukuran jendela dan perhatikan gaya yang berbeda diterapkan pada lebar layar yang berbeda. Header-mu kini **responsif** terhadap ukuran layar melalui penggunaan `@media` query.
