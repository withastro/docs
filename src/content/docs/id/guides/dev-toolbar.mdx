Saat dev server berjalan, Astro menyertakan dev toolbar di bagian bawah setiap halaman pada pratinjau browser lokal Anda.

Toolbar ini mencakup sejumlah alat berguna untuk debugging dan inspeksi situs selama pengembangan dan dapat [diperluas dengan lebih banyak aplikasi dev toolbar](#extending-the-dev-toolbar) yang tersedia di direktori integrasi. Anda bahkan bisa [membangun aplikasi toolbar Anda sendiri](/id/recipes/making-toolbar-apps/) menggunakan [Dev Toolbar API](/id/reference/dev-toolbar-app-reference/)!

Toolbar ini diaktifkan secara default dan akan muncul ketika Anda mengarahkan kursor ke bagian bawah halaman. Ini hanya alat pengembangan dan tidak akan muncul di situs yang dipublikasikan.

## Aplikasi bawaan

### Astro Menu

Aplikasi Astro Menu menyediakan akses mudah ke berbagai informasi tentang proyek saat ini dan tautan ke sumber tambahan. Secara khusus, aplikasi ini menyediakan akses satu klik ke dokumentasi Astro, repositori GitHub, dan server Discord.

Aplikasi ini juga menyertakan tombol "Copy debug info" yang akan menjalankan perintah [`astro info`](/id/reference/cli-reference/#astro-info) dan menyalin keluarannya ke papan klip Anda. Ini berguna saat meminta bantuan atau melaporkan masalah.

### Inspect

Aplikasi Inspect menyediakan informasi tentang [islands](/id/concepts/islands/) apa pun di halaman saat ini. Ini akan menampilkan properti yang diteruskan ke setiap island, dan direktif klien yang digunakan untuk merendernya.

### Audit

Aplikasi Audit secara otomatis menjalankan serangkaian audit pada halaman saat ini, memeriksa masalah kinerja dan aksesibilitas yang paling umum. Ketika suatu masalah ditemukan, titik merah akan muncul di toolbar. Mengklik aplikasi akan menampilkan daftar hasil audit dan menyorot elemen terkait langsung di halaman.

:::note
Audit kinerja dan aksesibilitas dasar yang dilakukan oleh dev toolbar bukan pengganti alat khusus seperti [Pa11y](https://pa11y.org/) atau [Lighthouse](https://developers.google.com/web/tools/lighthouse), atau bahkan lebih baik, manusia!

Dev toolbar bertujuan memberikan cara cepat dan mudah untuk menangkap masalah umum selama pengembangan, tanpa perlu berganti konteks ke alat lain.
:::

### Settings

Aplikasi Settings memungkinkan Anda mengonfigurasi opsi untuk dev toolbar, seperti logging yang lebih detail, menonaktifkan notifikasi, dan menyesuaikan penempatannya di layar Anda.

## Memperluas dev toolbar

Integrasi Astro dapat menambahkan aplikasi baru ke dev toolbar, memungkinkan Anda memperluasnya dengan alat khusus yang spesifik untuk proyek Anda. Anda dapat menemukan [lebih banyak aplikasi dev tool untuk dipasang di direktori integrasi](https://astro.build/integrations/?search=&categories%5B%5D=toolbar) atau menggunakan [Astro Menu](#astro-menu).

Instal integrasi aplikasi dev toolbar tambahan di proyek Anda seperti halnya [integrasi Astro](/id/guides/integrations-guide/) lainnya sesuai petunjuk pemasangannya.

## Menonaktifkan dev toolbar

Dev toolbar diaktifkan secara default untuk setiap situs. Anda dapat memilih untuk menonaktifkannya untuk proyek dan/atau pengguna tertentu sesuai kebutuhan.

### Per proyek

Untuk menonaktifkan dev toolbar bagi semua orang yang mengerjakan suatu proyek, setel `devToolbar: false` di [berkas konfigurasi Astro](/id/reference/configuration-reference/#devtoolbarenabled).

```js title="astro.config.mjs" ins={4-6}
import { defineConfig } from "astro/config";

export default defineConfig({
  devToolbar: {
    enabled: false
  }
});
```

Untuk mengaktifkan kembali dev toolbar, hapus barisâ€‘baris ini dari konfigurasi Anda, atau setel `enabled: true`.

### Per pengguna

Untuk menonaktifkan dev toolbar hanya untuk diri Anda pada proyek tertentu, jalankan perintah [`astro preferences`](/id/reference/cli-reference/#astro-preferences).

```shell
astro preferences disable devToolbar
```

Untuk menonaktifkan dev toolbar di semua proyek Astro bagi seorang pengguna pada mesin saat ini, tambahkan flag `--global` saat menjalankan `astro-preferences`:

```shell
astro preferences disable --global devToolbar
```

Dev toolbar dapat diaktifkan kembali dengan:

```shell
astro preferences enable devToolbar
```
