[Eleventy](https://11ty.dev) adalah static site generator open-source yang bekerja dengan banyak bahasa template.

## Persamaan Utama antara Eleventy (11ty) dan Astro

Eleventy (11ty) dan Astro memiliki beberapa kesamaan yang akan membantu Anda memigrasikan proyek Anda:

- Baik Astro maupun Eleventy adalah pembangun situs modern berbasis JavaScript (Jamstack).

- Astro dan Eleventy sama-sama memungkinkan Anda menggunakan [headless CMS, API, atau berkas Markdown untuk data](/id/guides/data-fetching/). Anda dapat terus menggunakan sistem penulisan konten pilihan Anda, dan tetap mempertahankan konten yang sudah ada.

## Perbedaan Utama antara Eleventy (11ty) dan Astro

Saat Anda membangun ulang situs Eleventy (11ty) di Astro, Anda akan melihat beberapa perbedaan penting:

- Eleventy mendukung berbagai bahasa templating. Astro mendukung [menyertakan komponen dari beberapa kerangka kerja JS populer (mis. React, Svelte, Vue, Solid)](/id/guides/framework-components/), tetapi menggunakan [layout, halaman, dan komponen Astro](/id/basics/astro-components/) untuk sebagian besar templating halaman.

- Astro menggunakan [direktori `src/`](/id/basics/project-structure/#src) untuk semua berkas—termasuk metadata situs—yang tersedia untuk di-query dan diproses saat build situs. Di dalamnya ada [folder khusus `src/pages/` untuk routing berbasis file](/id/basics/astro-pages/).

- Astro menggunakan [folder `public/` untuk aset statis](/id/basics/project-structure/#public) yang tidak perlu diproses maupun di-transform saat build.

- Di Eleventy, bundling CSS, JavaScript, dan aset lainnya perlu dikonfigurasi manual. [Astro menangani ini untuk Anda secara bawaan](/id/concepts/why-astro/#easy-to-use).

## Beralih dari Eleventy ke Astro

Untuk mengonversi blog Eleventy ke Astro, mulailah dengan starter template tema blog kami, atau jelajahi lebih banyak tema blog komunitas di [theme showcase](https://astro.build/themes/).

Anda dapat menambahkan argumen `--template` ke perintah `create astro` untuk memulai proyek Astro baru dengan salah satu starter resmi kami. Atau, Anda dapat [memulai proyek baru dari repositori Astro mana pun di GitHub](/id/install-and-setup/#install-from-the-cli-wizard).

  <PackageManagerTabs>
    <Fragment slot="npm">
    ```shell
    npm create astro@latest -- --template blog
    ```
    </Fragment>
    <Fragment slot="pnpm">
    ```shell
    pnpm create astro@latest --template blog
    ```
    </Fragment>
    <Fragment slot="yarn">
    ```shell
    yarn create astro --template blog
    ```
    </Fragment>
  </PackageManagerTabs>

 Bawa berkas Markdown (atau MDX, dengan integrasi opsional kami) Anda sebagai konten untuk [membuat halaman Markdown atau MDX](/id/guides/markdown-content/).

Proyek Eleventy Anda memungkinkan penggunaan berbagai bahasa templating untuk membangun situs. Di proyek Astro, templating halaman Anda sebagian besar akan dilakukan dengan komponen Astro, yang dapat digunakan sebagai elemen UI, layout, dan bahkan halaman penuh. Anda mungkin ingin mempelajari [sintaks komponen Astro](/id/basics/astro-components/) untuk melihat bagaimana melakukan templating di Astro menggunakan komponen.

Untuk mengonversi jenis situs lain, seperti portofolio atau situs dokumentasi, lihat lebih banyak starter template resmi di [astro.new](https://astro.new). Anda akan menemukan tautan ke setiap repositori GitHub proyek, serta tautan sekali klik untuk membuka proyek yang berfungsi di lingkungan pengembangan online seperti IDX, StackBlitz, CodeSandbox, dan Gitpod.
