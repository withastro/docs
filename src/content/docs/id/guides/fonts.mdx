Panduan ini akan menunjukkan cara menambahkan web font ke proyek Anda dan menggunakannya di komponen.

:::tip[API Font Eksperimental]
Pelajari tentang [API Font eksperimental](/id/reference/experimental-flags/fonts/) Astro yang memungkinkan Anda menggunakan font dari sistem berkas dan berbagai penyedia font melalui API terpadu yang sepenuhnya dapat disesuaikan dan bertipe aman.
:::

## Menggunakan berkas font lokal

Contoh ini akan mendemonstrasikan penambahan font kustom menggunakan berkas font `DistantGalaxy.woff`.

<Steps>
1. Tambahkan berkas font Anda ke `public/fonts/`.
2. Tambahkan pernyataan `@font-face` berikut ke CSS Anda. Ini bisa berada di berkas `.css` global yang Anda impor, blok `<style is:global>`, atau blok `<style>` di layout atau komponen tertentu tempat Anda ingin menggunakan font ini.

    ```css
    /* Daftarkan keluarga font kustom Anda dan beri tahu browser lokasi berkasnya. */
    @font-face {
      font-family: 'DistantGalaxy';
      src: url('/fonts/DistantGalaxy.woff') format('woff');
      font-weight: normal;
      font-style: normal;
      font-display: swap;
    }
    ```

3. Gunakan nilai `font-family` dari pernyataan `@font-face` untuk menata elemen di komponen atau layout Anda. Pada contoh ini, heading `<h1>` akan menggunakan font kustom, sedangkan paragraf `<p>` tidak.

    ```astro title="src/pages/example.astro" {9-11}
    ---
    ---

    <h1>In a galaxy far, far away...</h1>

    <p>Custom fonts make my headings much cooler!</p>

    <style>
    h1 {
      font-family: 'DistantGalaxy', sans-serif;
    }
    </style>
    ```
</Steps>

## Menggunakan Fontsource

Proyek [Fontsource](https://fontsource.org/) memudahkan penggunaan Google Fonts dan font open‑source lainnya. Fontsource menyediakan modul npm yang dapat Anda instal untuk font yang ingin digunakan.

<Steps>
1. Temukan font yang ingin Anda gunakan di [katalog Fontsource](https://fontsource.org/). Contoh ini akan menggunakan [Twinkle Star](https://fontsource.org/fonts/twinkle-star).
2. Instal paket untuk font pilihan Anda.

    <PackageManagerTabs>
      <Fragment slot="npm">
      ```shell
      npm install @fontsource/twinkle-star
      ```
      </Fragment>
      <Fragment slot="pnpm">
      ```shell
      pnpm add @fontsource/twinkle-star
      ```
      </Fragment>
      <Fragment slot="yarn">
      ```shell
      yarn add @fontsource/twinkle-star
      ```
      </Fragment>
    </PackageManagerTabs>

    :::tip
    Anda akan menemukan nama paket yang benar di bagian “Quick Installation” pada setiap halaman font di situs Fontsource. Nama paket akan diawali dengan `@fontsource/` atau `@fontsource-variable/` diikuti dengan nama font.
    :::

3. Impor paket font di komponen tempat Anda ingin menggunakan font tersebut. Biasanya, Anda ingin melakukannya di komponen layout umum agar font tersedia di seluruh situs.

    Impor ini akan secara otomatis menambahkan aturan `@font-face` yang diperlukan untuk menyiapkan font.

    ```astro title="src/layouts/BaseLayout.astro"
    ---
    import '@fontsource/twinkle-star';
    ---
    ```

4. Gunakan nama font seperti yang ditampilkan pada contoh `body` di halaman Fontsource‑nya sebagai nilai `font-family`. Ini akan berfungsi di mana pun Anda dapat menulis CSS di proyek Astro Anda.

    ```css
    h1 {
      font-family: "Twinkle Star", cursive;
    }
    ```
</Steps>

Untuk mengoptimalkan waktu rendering situs Anda, Anda mungkin ingin melakukan preload pada font yang esensial untuk tampilan awal halaman.
Lihat [panduan Fontsource tentang preload font](https://fontsource.org/docs/getting-started/preload) untuk informasi dan penggunaannya.

## Mendaftarkan font di Tailwind

Jika Anda menggunakan [Tailwind](/id/guides/styling/#tailwind), Anda dapat menggunakan salah satu dari metode sebelumnya pada halaman ini untuk memasang font, dengan beberapa penyesuaian. Anda dapat menambahkan [pernyataan `@font-face` untuk font lokal](#menggunakan-berkas-font-lokal) atau menggunakan [strategi `import` Fontsource](#menggunakan-fontsource) untuk memasang font.

Untuk mendaftarkan font Anda di Tailwind:

<Steps>
1. Ikuti salah satu panduan di atas, tetapi lewati langkah terakhir untuk menambahkan `font-family` ke CSS Anda.
2. Tambahkan nama tipe huruf ke `src/styles/global.css`.

    Contoh ini menambahkan `Inter` ke stack font sans-serif.

    ```css title="src/styles/global.css" ins={3-5}
    @import 'tailwindcss';

    @theme {
      --font-sans: 'Inter', 'sans-serif'; 
    }
    ```

    Sekarang, semua teks sans‑serif (default di Tailwind) dalam proyek Anda akan menggunakan font pilihan Anda dan kelas `font-sans` juga akan menerapkan font Inter.
</Steps>

Lihat [dokumentasi Tailwind tentang menambahkan keluarga font kustom](https://tailwindcss.com/docs/font-family#using-custom-values) untuk informasi lebih lanjut.
