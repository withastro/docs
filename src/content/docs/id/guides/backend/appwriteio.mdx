[Appwrite](https://appwrite.io/) adalah platform backend-as-a-service yang dapat dihosting sendiri (self-hosted) yang menyediakan autentikasi dan manajemen akun, preferensi pengguna, persistensi basis data dan penyimpanan, fungsi cloud, pelokalan, manipulasi gambar, serta berbagai utilitas sisi server lainnya.
