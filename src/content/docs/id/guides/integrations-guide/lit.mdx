:::caution[Deprecated]
Integrasi Astro untuk mengaktifkan rendering on-demand dan hidrasi sisi klien untuk elemen kustom [Lit](https://lit.dev/) ini telah **dideprekasi** di Astro 5.0.
:::

Anda tetap dapat menggunakan Lit untuk komponen klien dengan menambahkan tag skrip sisi klien. Contoh:

```astro
<script>
  import "../components/MyTabs";
</script>

<my-tabs title="These are my tabs">...</my-tabs>
```

Jika Anda tertarik untuk memelihara integrasi Lit sendiri, Anda dapat menggunakan [rilis terakhir `@astrojs/lit` yang dipublikasikan](https://github.com/withastro/astro/tree/astro%404.13.0/packages/integrations/lit) sebagai titik awal dan memperbarui paket-paket yang relevan.
