Adapter Deno memungkinkan Astro untuk mendeploy situs SSR Anda ke target Deno termasuk Deno Deploy. 

Adapter Deno sebelumnya dipelihara oleh tim Astro, tetapi sekarang dipelihara langsung oleh Deno. Penggunaan kini didokumentasikan [di repositori adapter Deno](https://github.com/denoland/deno-astro-adapter).

Jika Anda saat ini menggunakan adapter Astro ini, Anda perlu bermigrasi ke versi Deno yang baru atau [menambahkan adapter lain](/id/guides/on-demand-rendering/) untuk terus menggunakan SSR di proyek Anda.
