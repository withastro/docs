Integrasi **[Astro][astro-integration]** ini mengaktifkan perenderan dan hidrasi sisi klien untuk komponen [Svelte](https://svelte.dev/) 5 Anda. Untuk dukungan Svelte 3 dan 4, instal `@astrojs/svelte@5` sebagai gantinya.

## Instalasi

Astro menyertakan perintah `astro add` untuk mengotomatisasi penyiapan integrasi resmi. Jika Anda mau, Anda dapat [menginstal integrasi secara manual](#manual-install) sebagai gantinya.

Untuk memasang `@astrojs/svelte`, jalankan salah satu perintah berikut dari direktori proyek Anda dan ikuti petunjuknya:

<PackageManagerTabs>
  <Fragment slot="npm">
  ```sh
  npx astro add svelte
  ```
  </Fragment>
  <Fragment slot="pnpm">
  ```sh
  pnpm astro add svelte
  ```
  </Fragment>
  <Fragment slot="yarn">
  ```sh
  yarn astro add svelte
  ```
  </Fragment>
</PackageManagerTabs>

Jika Anda mengalami masalah, [silakan laporkan kepada kami di GitHub](https://github.com/withastro/astro/issues) dan coba langkah instalasi manual di bawah ini.

### Manual Install

Pertama, pasang paket `@astrojs/svelte`:

<PackageManagerTabs>
  <Fragment slot="npm">
  ```sh
  npm install @astrojs/svelte
  ```
  </Fragment>
  <Fragment slot="pnpm">
  ```sh
  pnpm add @astrojs/svelte
  ```
  </Fragment>
  <Fragment slot="yarn">
  ```sh
  yarn add @astrojs/svelte
  ```
  </Fragment>
</PackageManagerTabs>

Sebagian besar manajer paket juga akan memasang dependensi sejawat yang terkait. Jika Anda melihat peringatan `Cannot find package 'svelte'` (atau serupa) saat menjalankan Astro, Anda perlu memasang Svelte dan TypeScript:

<PackageManagerTabs>
  <Fragment slot="npm">
  ```sh
  npm install svelte typescript
  ```
  </Fragment>
  <Fragment slot="pnpm">
  ```sh
  pnpm add svelte typescript
  ```
  </Fragment>
  <Fragment slot="yarn">
  ```sh
  yarn add svelte typescript
  ```
  </Fragment>
</PackageManagerTabs>

Lalu, terapkan integrasi ke berkas `astro.config.*` Anda menggunakan properti `integrations`:

```js title="astro.config.mjs" ins={2} ins="svelte()"
import { defineConfig } from 'astro/config';
import svelte from '@astrojs/svelte';

export default defineConfig({
  // ...
  integrations: [svelte()],
});
```

Dan buat berkas baru bernama `svelte.config.js` di direktori root proyek Anda lalu tambahkan kode berikut:

```js title="svelte.config.js"
import { vitePreprocess } from '@astrojs/svelte';

export default {
  preprocess: vitePreprocess(),
}
```

## Opsi

Integrasi ini ditenagai oleh `@sveltejs/vite-plugin-svelte`. Untuk menyesuaikan compiler Svelte, opsi dapat diberikan ke integrasi. Lihat dokumentasi [`@sveltejs/vite-plugin-svelte`](https://github.com/sveltejs/vite-plugin-svelte/blob/HEAD/docs/config.md) untuk detail lebih lanjut.

Anda dapat menetapkan opsi dengan meneruskannya ke integrasi `svelte` di `astro.config.mjs` atau di `svelte.config.js`. Opsi di `astro.config.mjs` akan memiliki prioritas lebih tinggi dibanding opsi di `svelte.config.js` jika keduanya ada:

```js title="astro.config.mjs" "extensions: ['.svelte']"
import { defineConfig } from 'astro/config';
import svelte from '@astrojs/svelte';

export default defineConfig({
  integrations: [svelte({ extensions: ['.svelte'] })],
});
```

```js title="svelte.config.js"
export default {
  extensions: ['.svelte'],
};
```

## Praprosesor

<Since v="2.0.0" pkg="@astrojs/svelte" />

Jika Anda menggunakan SCSS atau Stylus di berkas Svelte, Anda dapat membuat `svelte.config.js` agar berkas tersebut dipraproses oleh Svelte, dan ekstensi IDE Svelte dapat mengurai berkas Svelte dengan benar.

```js title="svelte.config.js"
import { vitePreprocess } from '@astrojs/svelte';

export default {
  preprocess: vitePreprocess(),
};
```

Berkas konfigurasi ini akan ditambahkan secara otomatis saat Anda menjalankan `astro add svelte`. Lihat dokumentasi [`@sveltejs/vite-plugin-svelte`](https://github.com/sveltejs/vite-plugin-svelte/blob/HEAD/docs/preprocess.md) untuk detail lebih lanjut tentang `vitePreprocess`.
