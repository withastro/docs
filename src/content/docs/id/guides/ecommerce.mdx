Dengan Astro, Anda dapat membangun berbagai opsi e‑commerce, mulai dari tautan checkout hingga halaman pembayaran ter‑hosting, hingga membangun seluruh etalase menggunakan API layanan pembayaran.

## Overlay pemrosesan pembayaran

Beberapa layanan pemrosesan pembayaran (mis. [Lemon Squeezy](#lemon-squeezy), [Paddle](#paddle)) menambahkan formulir pembayaran agar pelanggan dapat melakukan pembelian dari situs Anda. Ini bisa berupa overlay yang di‑hosting atau disematkan di sebuah halaman di situs Anda. Layanan ini biasanya menawarkan kustomisasi dasar atau branding situs, dan dapat ditambahkan ke proyek Astro Anda sebagai skrip, tombol, atau tautan eksternal.

### Lemon Squeezy

[Lemon Squeezy](https://www.lemonsqueezy.com/) adalah platform serba ada untuk pembayaran dan langganan dengan dukungan multi‑mata uang, kepatuhan pajak global, integrasi PayPal, dan lainnya. Layanan ini memungkinkan Anda membuat serta mengelola produk dan layanan digital melalui dasbor akun dan menyediakan URL produk untuk proses checkout.

Pustaka JavaScript [Lemon.js](https://docs.lemonsqueezy.com/help/lemonjs/what-is-lemonjs) yang mendasar memungkinkan Anda menjual produk Lemon Squeezy menggunakan tautan checkout.

#### Penggunaan dasar

Berikut contoh menambahkan elemen “Buy now” Lemon Squeezy ke sebuah halaman Astro. Mengklik tautan ini akan membuka checkout dan memungkinkan pengunjung menyelesaikan satu pembelian.

<Steps>
1. Tambahkan tag `<script>` berikut ke `head` atau `body` halaman Anda:

    ```html title="src/pages/my-product-page.astro"
    <script src="https://app.lemonsqueezy.com/js/lemon.js" defer></script>
    ```

2. Buat tag anchor di halaman yang menaut ke URL produk Anda. Sertakan kelas `lemonsqueezy-button` untuk membuka overlay checkout saat diklik.

    ```html title="src/pages/my-product-page.astro"
    <a class="lemonsqueezy-button" href="https://demo.lemonsqueezy.com/checkout/...">
      Buy Now
    </a>
    ```
</Steps>

#### Lemon.js

Lemon.js juga menyediakan perilaku tambahan seperti [membuka overlay secara terprogram](https://docs.lemonsqueezy.com/help/lemonjs/opening-overlays) dan [menangani event overlay](https://docs.lemonsqueezy.com/help/lemonjs/handling-events).

<ReadMore> Baca [panduan memulai pengembang Lemon Squeezy](https://docs.lemonsqueezy.com/guides/developer-guide) untuk informasi selengkapnya.</ReadMore>

### Paddle

[Paddle](https://www.paddle.com/) adalah solusi penagihan untuk produk dan layanan digital. Paddle menangani pembayaran, pajak, dan manajemen langganan melalui checkout overlay atau inline. 

[Paddle.js](https://developer.paddle.com/paddlejs/overview) adalah pustaka JavaScript ringan yang memungkinkan Anda membangun pengalaman penagihan langganan yang kaya dan terintegrasi menggunakan Paddle.

#### Penggunaan dasar

Berikut contoh menambahkan elemen “Buy Now” Paddle ke halaman Astro. Mengklik tautan ini akan membuka checkout dan memungkinkan pengunjung menyelesaikan satu pembelian.

Setelah domain tautan pembayaran default Anda (situs Anda sendiri) disetujui oleh Paddle, Anda dapat mengubah elemen apa pun di halaman menjadi pemicu overlay checkout menggunakan atribut data HTML.

<Steps>
1. Tambahkan dua tag `<script>` berikut ke `head` atau `body` halaman Anda:

    ```html title="src/pages/my-product-page.astro"
    <script src="https://cdn.paddle.com/paddle/v2/paddle.js"></script>
    <script type="text/javascript">
      Paddle.Setup({ 
        token: '7d279f61a3499fed520f7cd8c08' // ganti dengan token sisi-klien Anda
      });
    </script>
    ```

2. Ubah elemen apa pun di halaman menjadi tombol Paddle Checkout dengan menambahkan kelas `paddle_button`:

    ```html title="src/pages/my-product-page.astro"
    <a href="#" class="paddle_button">Buy Now</a>
    ```

3. Tambahkan atribut `data-items` untuk menentukan `priceId` dan `quantity` produk Paddle Anda. Anda juga dapat secara opsional meneruskan [atribut data HTML yang didukung](https://developer.paddle.com/paddlejs/html-data-attributes) untuk isian awal data, menangani keberhasilan checkout, atau menata tombol dan overlay checkout Anda:

    ```html title="src/pages/my-product-page.astro"
    <a 
      href="#"
      class="paddle_button"
      data-display-mode="overlay"
      data-theme="light"
      data-locale="en"
      data-success-url="https://example.com/thankyou"
      data-items='[
        {
          "priceId": "pri_01gs59hve0hrz6nyybj56z04eq", 
          "quantity": 1
        }
      ]'
    >
      Buy now
    </a>
    ```
</Steps>

#### Paddle.js

Alih‑alih meneruskan atribut data HTML, Anda dapat mengirim data ke overlay checkout menggunakan JavaScript untuk meneruskan banyak atribut dan kustomisasi yang lebih luas. Anda juga dapat membuat alur peningkatan (upgrade) menggunakan checkout inline.

<ReadMore>Baca selengkapnya tentang [menggunakan Paddle.js untuk membangun checkout inline bermerek](https://developer.paddle.com/build/checkout/build-branded-inline-checkout).</ReadMore>


## Solusi e‑commerce lengkap

Untuk kustomisasi yang lebih luas atas keranjang belanja dan proses checkout situs Anda, Anda dapat menghubungkan penyedia layanan finansial yang lebih lengkap (mis. [Snipcart](#snipcart)) ke proyek Astro Anda. Platform e‑commerce ini juga dapat terintegrasi dengan layanan pihak ketiga lain untuk manajemen akun pengguna, personalisasi, inventaris, dan analitik.

### Snipcart

[Snipcart](https://snipcart.com/) adalah platform keranjang belanja HTML/JavaScript yang kuat dan berfokus pada developer.

Snipcart juga memungkinkan Anda berintegrasi dengan layanan pihak ketiga seperti penyedia pengiriman, mengaktifkan webhook untuk integrasi e‑commerce tingkat lanjut antara keranjang belanja dan sistem lain, memilih beberapa gerbang pembayaran (mis. Stripe, Paypal, dan Square), menyesuaikan template email, dan bahkan menyediakan lingkungan pengujian langsung.

:::tip
Ingin solusi Snipcart siap pakai? Coba [`astro-snipcart`](https://astro-snipcart.vercel.app/), templat komunitas Astro yang sepenuhnya fungsional termasuk sistem desain opsional, siap Anda integrasikan dengan akun Snipcart yang sudah ada.
:::

#### Penggunaan dasar

Berikut contoh mengonfigurasi checkout Snipcart dan menambahkan elemen tombol “Add to cart” serta “Check out now” ke halaman Astro. Ini memungkinkan pengunjung menambahkan produk ke keranjang tanpa langsung diarahkan ke halaman checkout.

<ReadMore>Untuk instruksi lengkap, termasuk menyiapkan toko Snipcart Anda, lihat [dokumentasi instalasi Snipcart](https://docs.snipcart.com/v3/setup/installation).</ReadMore>

<Steps>
1. Tambahkan skrip [seperti pada instruksi instalasi Snipcart](https://docs.snipcart.com/v3/setup/installation) di halaman Anda setelah elemen `<body>`.
    ```html title="src/pages/my-product-page.astro"
    <body></body>
    <script>
      window.SnipcartSettings = {
        publicApiKey: "YOUR_API_KEY",
        loadStrategy: "on-user-interaction",
      };

      (function()...); // tersedia dari dokumentasi Snipcart
    </script>
    ```

2. Sesuaikan `window.SnipcartSettings` dengan salah satu [pengaturan Snipcart yang tersedia](https://docs.snipcart.com/v3/setup/installation#settings) untuk mengontrol perilaku dan tampilan keranjang Anda.

    ```html title="src/pages/my-product-page.astro"
    <script>
      window.SnipcartSettings = {
        publicApiKey: "YOUR_API_KEY",
        loadStrategy: "manual",
        version: "3.7.1",
        addProductBehavior: "none",
        modalStyle: "side",
      };

      (function()...); // tersedia dari dokumentasi Snipcart
    </script>
    ```

3. Tambahkan `class="snipcart-add-item"` ke elemen HTML apa pun, seperti `<button>`, untuk menambahkan item ke keranjang saat diklik. Sertakan juga elemen data lain untuk [atribut produk Snipcart umum](https://docs.snipcart.com/v3/setup/products) seperti harga dan deskripsi, dan field opsional apa pun.

    ```html title="src/pages/my-product-page.astro"
    <button
      class="snipcart-add-item"
      data-item-id="astro-print"
      data-item-price="39.99"
      data-item-description="A framed print of the Astro logo."
      data-item-image="/assets/images/astro-print.jpg"
      data-item-name="Astro Print"
      data-item-custom1-name="Frame color"
      data-item-custom1-options="Brown|Silver[+10.00]|Gold[+20.00]"
      data-item-custom2-name="Delivery instructions"
      data-item-custom2-type="textarea"
    >
      Add to cart
    </button>
    ```

4. Tambahkan tombol checkout Snipcart dengan kelas `snipcart-checkout` untuk membuka keranjang dan memungkinkan tamu menyelesaikan pembelian dengan modal checkout.

    ```html title="src/pages/my-product-page.astro"
    <button class="snipcart-checkout">Click here to checkout</button>
    ```
</Steps>

#### SDK JavaScript Snipcart

[Snipcart JavaScript SDK](https://docs.snipcart.com/v3/sdk/basics) memungkinkan Anda mengonfigurasi, menyesuaikan, dan mengelola keranjang Snipcart secara terprogram.

Ini memungkinkan Anda melakukan tindakan seperti:

- Mengambil informasi yang relevan tentang sesi Snipcart saat ini dan menerapkan operasi tertentu ke keranjang.
- Mendengarkan event yang masuk dan memicu callback secara dinamis.
- Mendengarkan perubahan state dan menerima snapshot lengkap dari state keranjang.

<ReadMore>Lihat [dokumentasi Snipcart](https://docs.snipcart.com/v3/) untuk informasi lebih lanjut tentang semua opsi integrasi Snipcart dengan Proyek Astro Anda.</ReadMore>

#### `astro-snipcart`

Ada dua paket komunitas `astro-snipcart` yang dapat menyederhanakan penggunaan Snipcart.

- [`@lloydjatkinson/astro-snipcart` Astro template](https://astro-snipcart.vercel.app/): Templat Astro ini mencakup sistem desain opsional untuk solusi e‑commerce lengkap siap pakai. Pelajari lebih lanjut di situs dokumentasinya sendiri, termasuk [motivasi di balik pembangunan `astro-snipcart`](https://astro-snipcart.vercel.app/motivation) sebagai cara praktis, native Astro untuk berinteraksi dengan API Snipcart.

- [`@Adammatthiesen/astro-snipcart` integration](https://github.com/Adammatthiesen/astro-snipcart): Integrasi ini terinspirasi dari tema `astro-snipcart` dan menyediakan komponen Astro (atau Vue) yang dapat Anda tambahkan ke proyek Astro yang ada untuk membuat produk, mengontrol keranjang, dan lainnya. Lihat [tutorial lengkapnya](https://matthiesen.xyz/blog/getting-started-with-my-astro-snipcart-addon) untuk informasi lebih lanjut.
