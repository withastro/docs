Anda dapat mendeploy proyek Astro menggunakan [CloudRay](https://cloudray.io), platform terpusat yang membantu Anda mengelola server, mengorganisir skrip Bash, dan mengotomatiskan tugas deployment di mesin virtual serta server cloud.

:::note
CloudRay sendiri tidak meng‑host situs Anda. Sebaliknya, CloudRay menyediakan alat otomasi untuk menjalankan skrip deployment pada infrastruktur Anda sendiri (mis. server Ubuntu) melalui agen yang terhubung.
:::

## Prasyarat 

Untuk memulai, Anda memerlukan:

- [Akun CloudRay](https://app.cloudray.io)
- Kode aplikasi Anda tersimpan di repository [GitHub](https://github.com/)

## Cara Deploy melalui Dashboard CloudRay

Deployment dengan CloudRay biasanya mencakup tiga langkah utama:

<Steps>
1. Pasang [CloudRay Agent](https://cloudray.io/docs/agent) di server Anda untuk mendaftarkan mesin secara aman dan mengaktifkan otomasi jarak jauh.

2. Di Dashboard CloudRay, tulis skrip Bash yang dapat digunakan ulang untuk mengklona repo Astro Anda, memasang dependensi, membangun situs, dan mengonfigurasi web server. Definisikan nilai spesifik repository menggunakan [variable groups CloudRay](https://cloudray.io/docs/variable-groups).

3. Gunakan antarmuka Runlog CloudRay untuk mengeksekusi skrip Anda pada server yang terhubung dan memantau proses deployment secara real‑time.
</Steps>
