Anda dapat mendeply situs Astro ke AWS menggunakan [SST](https://sst.dev), framework open‑source untuk mendeply aplikasi full‑stack modern dengan dukungan SSG dan SSR.

Anda juga dapat menggunakan komponen SST tambahan seperti cron job, Bucket, Queue, dan lain‑lain dengan tetap menjaga type‑safety.

## Quickstart

<Steps>
1. Buat proyek astro.

2. Jalankan `npx sst@latest init`.

3. SST akan mendeteksi bahwa Anda menggunakan Astro dan meminta konfirmasi.

4. Saat siap untuk deployment jalankan `npx sst deploy --stage production`.
</Steps>

Anda juga bisa membaca [tutorial lengkap Astro di AWS dengan SST](https://sst.dev/docs/start/aws/astro) yang memandu Anda langkah demi langkah.

### Komponen SST

Untuk menggunakan [komponen SST tambahan](https://sst.dev/docs/), tambahkan ke `sst.config.ts`. 

```ts {1} {5} title="sst.config.ts"
const bucket = new sst.aws.Bucket("MyBucket", {
  access: "public",
});
new sst.aws.Astro("MyWeb", {
  link: [bucket],
});
```

Lalu akses di file `.astro` Anda.

```astro
---
import { Resource } from "sst"
console.log(Resource.MyBucket.name)
---
```

Pelajari lebih lanjut di [dokumentasi SST tentang linking resource](https://sst.dev/docs/linking).

Jika Anda punya pertanyaan, Anda bisa [bertanya di Discord SST](https://discord.gg/sst).
