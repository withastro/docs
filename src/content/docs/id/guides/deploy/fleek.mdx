Anda dapat menggunakan [Fleek](http://fleek.xyz/) untuk men-deploy situs Astro statis ke jaringan terdesentralisasi yang dioptimalkan di edge milik mereka.

Panduan ini memberikan langkah lengkap untuk men-deploy situs Astro Anda ke Fleek menggunakan UI dan CLI Fleek.

## Konfigurasi Proyek

Proyek Astro Anda dapat dideploy ke Fleek sebagai situs statis.

## Cara men-deploy

Anda dapat men-deploy ke Fleek melalui antarmuka situs (UI) atau menggunakan CLI (command line interface) Fleek.

### Deployment via Platform UI

<Steps>
1. Buat akun [Fleek](https://app.fleek.xyz).

2. Push kode Anda ke repositori Git daring (GitHub).

3. Impor proyek Anda ke Fleek.

4. Fleek akan secara otomatis mendeteksi Astro dan kemudian Anda dapat mengonfigurasi pengaturan yang benar.

5. Aplikasi Anda ter-deploy!
</Steps>


### Fleek CLI

<Steps>
1. Instal Fleek CLI.

    ```bash
    # Anda perlu Nodejs >= 18.18.2
    npm install -g @fleek-platform/cli
    ```
2. Masuk ke akun Fleek Anda dari terminal.

    ```bash
    fleek login
    ```

3. Jalankan perintah build untuk menghasilkan berkas statis. Secara bawaan, berkas-berkas ini berada di direktori `dist/`.

    ```bash
    npm run build
    ```

4. Inisialisasi proyek Anda. Ini akan menghasilkan berkas konfigurasi.

    ```bash
    fleek sites init
    ```

5. Anda akan diminta untuk membuat Fleek Site baru atau menggunakan yang sudah ada. Beri nama situs dan pilih direktori tempat proyek Anda berada.

6. Deploy situs Anda.

    ```bash
    fleek sites deploy
    ```

</Steps>
