[Firebase Hosting](https://firebase.google.com/products/hosting) adalah layanan dari platform pengembangan aplikasi [Firebase](https://firebase.google.com/) milik Google yang dapat digunakan untuk melakukan deploy situs Astro. 

Lihat panduan terpisah kami untuk [menambahkan layanan backend Firebase](/id/guides/backend/google-firebase/) seperti database, autentikasi, dan penyimpanan.

## Konfigurasi Proyek

Proyek Astro Anda dapat dideploy ke Firebase sebagai situs statis, atau sebagai situs yang dirender di sisi server (SSR).

### Situs Statis

Proyek Astro Anda secara default adalah situs statis. Anda tidak memerlukan konfigurasi tambahan untuk melakukan deploy situs Astro statis ke Firebase.

### Adapter untuk SSR

Untuk mengaktifkan SSR di proyek Astro Anda dan melakukan deploy di Firebase, tambahkan [adapter Node.js](/id/guides/integrations-guide/node/). 

:::note
Melakukan deploy situs Astro SSR ke Firebase memerlukan [paket Blaze](https://firebase.google.com/pricing) atau yang lebih tinggi.
:::

## Cara melakukan deploy

<Steps>
1. Instal [Firebase CLI](https://github.com/firebase/firebase-tools). Ini adalah alat baris perintah yang memungkinkan Anda berinteraksi dengan Firebase dari terminal.

    <PackageManagerTabs>
      <Fragment slot="npm">
      ```shell
      npm install firebase-tools
      ```
      </Fragment>
      <Fragment slot="pnpm">
      ```shell
      pnpm add firebase-tools
      ```
      </Fragment>
      <Fragment slot="yarn">
      ```shell
      yarn add firebase-tools
      ```
      </Fragment>
    </PackageManagerTabs>

2. Autentikasi Firebase CLI dengan akun Google Anda. Ini akan membuka jendela browser untuk masuk ke akun Google Anda.

    <PackageManagerTabs>
      <Fragment slot="npm">
      ```shell
      npx firebase login
      ```
      </Fragment>
      <Fragment slot="pnpm">
      ```shell
      pnpm exec firebase login
      ```
      </Fragment>
      <Fragment slot="yarn">
      ```shell
      yarn firebase login
      ```
      </Fragment>
    </PackageManagerTabs>

3. Aktifkan dukungan web frameworks eksperimental. Ini adalah fitur eksperimental yang memungkinkan Firebase CLI mendeteksi dan mengonfigurasi pengaturan deployment untuk Astro.

    <PackageManagerTabs>
      <Fragment slot="npm">
      ```shell
      npx firebase experiments:enable webframeworks
      ```
      </Fragment>
      <Fragment slot="pnpm">
      ```shell
      pnpm exec firebase experiments:enable webframeworks
      ```
      </Fragment>
      <Fragment slot="yarn">
      ```shell
      yarn firebase experiments:enable webframeworks
      ```
      </Fragment>
    </PackageManagerTabs>

4. Inisialisasi Firebase Hosting di proyek Anda. Ini akan membuat file `firebase.json` dan `.firebaserc` di root proyek.

    <PackageManagerTabs>
      <Fragment slot="npm">
      ```shell
      npx firebase init hosting
      ```
      </Fragment>
      <Fragment slot="pnpm">
      ```shell
      pnpm exec firebase init hosting
      ```
      </Fragment>
      <Fragment slot="yarn">
      ```shell
      yarn firebase init hosting
      ```
      </Fragment>
    </PackageManagerTabs>

5. Deploy situs Anda ke Firebase Hosting. Ini akan membangun situs Astro Anda dan mendeploy-nya ke Firebase.

    <PackageManagerTabs>
      <Fragment slot="npm">
      ```shell
      npx firebase deploy --only hosting
      ```
      </Fragment>
      <Fragment slot="pnpm">
      ```shell
      pnpm exec firebase deploy --only hosting
      ```
      </Fragment>
      <Fragment slot="yarn">
      ```shell
      yarn firebase deploy --only hosting
      ```
      </Fragment>
    </PackageManagerTabs>
</Steps>
