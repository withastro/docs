Anda dapat mendeply proyek Astro ke [Render](https://render.com/), layanan untuk membangun website dengan sertifikat TLS gratis, CDN global, perlindungan DDoS, private network, dan auto deploy dari Git.

## Cara deploy

<Steps>
1. Buat akun [render.com](https://dashboard.render.com/) dan masuk

2. Klik tombol **New +** dari dashboard Anda dan pilih **Static Site**

3. Hubungkan repositori [GitHub](https://github.com/) atau [GitLab](https://about.gitlab.com/) Anda, atau masukkan URL publik dari repositori publik

4. Beri nama situs Anda, pilih branch, dan tentukan build command serta publish directory

   - **Build Command:** `npm run build`
   - **Publish Directory:** `dist` untuk situs statis; `dist/client` jika Anda memiliki halaman yang dirender onâ€‘demand.

5. Klik tombol **Create Static Site**
</Steps>
