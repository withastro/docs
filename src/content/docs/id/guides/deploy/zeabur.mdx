Panduan ini mencakup instruksi mendeploy ke Zeabur melalui antarmuka situs web.

## Konfigurasi Proyek

### Situs statis

Astro secara default menghasilkan situs statis. Tidak diperlukan konfigurasi tambahan untuk mendeploy situs Astro statis ke Zeabur. 

### Adapter untuk SSR

Untuk mengaktifkan SSR di proyek Astro Anda dan mendeploy di Zeabur:

<Steps>
1. Pasang [adapter `@zeabur/astro-adapter`](https://www.npmjs.com/package/@zeabur/astro-adapter) ke dependensi proyek Anda menggunakan manajer paket pilihan Anda. Jika Anda menggunakan npm atau tidak yakin, jalankan perintah berikut di terminal:

    ```bash
      npm install @zeabur/astro-adapter
    ```

2. Tambahkan dua baris baru ke file konfigurasi proyek `astro.config.mjs` Anda.

    ```js title="astro.config.mjs" ins={2, 5-6}
    import { defineConfig } from 'astro/config';
    import zeabur from '@zeabur/astro-adapter/serverless';

    export default defineConfig({
      output: 'server',
      adapter: zeabur(),
    });
    ```
</Steps>

## Cara mendeploy

Anda dapat mendeploy situs Astro ke Zeabur jika proyek disimpan di GitHub.

<Steps>
1. Klik <kbd>Create new project</kbd> di [dashboard Zeabur](https://dash.zeabur.com).

2. Konfigurasikan instalasi GitHub dan impor repositori.

3. Zeabur akan secara otomatis mendeteksi bahwa proyek Anda adalah proyek Astro dan akan membangunnya menggunakan perintah `astro build`.

4. Setelah build selesai, Anda dapat mengaitkan domain ke situs Anda dan mengunjunginya.
</Steps>

Setelah proyek Anda diimpor dan dideploy, setiap push berikutnya ke cabang mana pun akan menghasilkan build baru.

Pelajari lebih lanjut di [Panduan Deployment Zeabur](https://zeabur.com/docs/get-started/).
