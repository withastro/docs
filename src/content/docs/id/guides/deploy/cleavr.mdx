Anda dapat mendeploy proyek Astro Anda ke Virtual Private Server (VPS) milik Anda menggunakan [Cleavr](https://cleavr.io/), alat manajemen server dan deployment aplikasi.

:::tip
Lihat juga [panduan Astro pada dokumentasi Cleavr](https://docs.cleavr.io/guides/astro)!
:::

## Prasyarat

Untuk memulai, Anda memerlukan:

- Akun Cleavr
- Sebuah server pada penyedia VPS Anda yang dikelola melalui Cleavr

## Tambahkan situs Anda

<Steps>
1. Di Cleavr, buka server tempat Anda ingin menambahkan proyek Astro.

2. Pilih **Add Site** dan isi detail aplikasi Anda, seperti nama domain.

3. Untuk **App Type**, pilih 'NodeJS Static' atau 'NodeJS SSR' sesuai dengan cara Anda menyiapkan aplikasi Astro.
   
4. Untuk aplikasi Static, atur **Artifact Folder** ke `dist`.
   
5. Untuk aplikasi SSR:
   - Atur **Entry Point** ke `entry.mjs`.
   - Atur **Artifact Folder** ke `dist/server`.
   
6. Pilih **Add** untuk menambahkan situs ke server Anda.
</Steps>

## Penyiapan dan deployment

<Steps>
1. Setelah situs baru ditambahkan, klik **Setup and deploy**.
   
2. Pilih **VC Profile**, **Repo**, dan **Branch** untuk proyek Astro Anda.
   
3. Lakukan konfigurasi tambahan yang diperlukan untuk proyek Anda.
   
4. Klik tab **Deployments** lalu klik **Deploy**.
</Steps>

Selamat, Anda telah berhasil mendeploy aplikasi Astro!
