Anda bisa menggunakan [Kinsta Application Hosting](https://kinsta.com/application-hosting/) untuk meng-host situs Astro di cloud mereka.

## Mengonfigurasi proyek Astro Anda

### Hosting statis

:::tip[Mencari contoh?]
Lihat [Starter resmi Kinsta Application Hosting untuk Astro](https://github.com/kinsta/hello-world-astro)!
:::

Untuk meng-host proyek Anda di **Kinsta Application Hosting**, Anda perlu:
- Menyertakan field `name` di `package.json`. (Nilainya bebas, tidak memengaruhi deployment.)
- Menyertakan script `build` di `package.json`. (Proyek Astro Anda seharusnya sudah memilikinya.)
- Menginstal paket [`serve`](https://www.npmjs.com/package/serve) dan mengatur script `start` menjadi `serve dist/`.

Berikut baris yang diperlukan di file `package.json` Anda:
```json title="package.json" {2,5,6} ins={12} "serv dist/"
{
  "name": "anything", // Ini wajib, namun nilainya bebas.
  "scripts": {
    "dev": "astro dev",
    "start": "serve dist/",
    "build": "astro build",
    "preview": "astro preview",
    "astro": "astro"
  },
  "dependencies": {
    "astro": "^2.2.0",
    "serve": "^14.0.1"
  },
}
```

### SSR

:::tip[Mencari contoh?]
Lihat [Starter resmi Kinsta Application Hosting untuk Astro SSR](https://github.com/kinsta/hello-world-astro-ssr)!
:::

Untuk meng-host proyek Anda di **Kinsta Application Hosting**, Anda perlu:
- Menyertakan field `name` di `package.json`. (Nilainya bebas, tidak memengaruhi deployment.)
- Menyertakan script `build` di `package.json`. (Proyek Astro Anda seharusnya sudah memilikinya.)
- Menginstal paket [`@astrojs/node`](https://www.npmjs.com/package/@astrojs/node) dan mengatur script `start` menjadi `node ./dist/server/entry.mjs`.
- Mengatur `astro.config.mjs` untuk menggunakan `@astrojs/node` dan `host: true`.

Berikut baris yang diperlukan di file `package.json` Anda:
```json title="package.json" {2,5,6} ins={12} "node ./dist/server/entry.mjs"
{
  "name": "anything", // Ini wajib, namun nilainya bebas.
  "scripts": {
    "dev": "astro dev",
    "start": "node ./dist/server/entry.mjs",
    "build": "astro build",
    "preview": "astro preview",
    "astro": "astro"
  },
  "dependencies": {
    "astro": "^2.2.0",
    "@astrojs/node": "^5.1.1"
  },
}
```

Berikut baris yang diperlukan di file `astro.config.mjs` Anda:
```js title="astro.config.mjs"
  import { defineConfig } from 'astro/config';
  import node from "@astrojs/node";

  export default defineConfig({
    output: 'server',
    adapter: node({
      mode: "standalone"
    }),
    server: {
      host: true
    }
  });
```

## Cara men-deploy

Setelah repositori GitHub proyek Anda terhubung, Anda dapat memicu deploy manual ke Kinsta Application Hosting melalui **MyKinsta Admin Panel**. Anda juga dapat menyiapkan deployment otomatis di panel admin Anda.

### Mengonfigurasi aplikasi Kinsta baru

<Steps>
1. Buka panel admin [My Kinsta](https://my.kinsta.com/).

2. Masuk ke tab **Applications**.

3. Hubungkan repositori GitHub Anda.

4. Tekan tombol **Add service** > **Application**.

5. Ikuti langkah-langkah wizard.

6. Aplikasi Anda ter-deploy.
</Steps>
