Anda dapat mendeploy situs Astro menggunakan [Flightcontrol](https://www.flightcontrol.dev?ref=astro), yang menyediakan proses deployment otomatis penuh ke akun AWS Anda.

Mendukung situs Astro statis maupun SSR.

## Cara Mendeploy

<Steps>
1. Buat akun Flightcontrol di [app.flightcontrol.dev/signup](https://app.flightcontrol.dev/signup?ref=astro)

2. Kunjungi [app.flightcontrol.dev/projects/new/1](https://app.flightcontrol.dev/projects/new/1)

3. Hubungkan akun GitHub Anda dan pilih repositori Anda

4. Pilih **Config Type** yang diinginkan:
    - `GUI` (seluruh konfigurasi dikelola melalui dashboard Flightcontrol) di mana Anda akan memilih preset `Astro Static` atau `Astro SSR`
    - `flightcontrol.json` (opsi *infrastructure as code* di mana seluruh konfigurasi ada di repo Anda) di mana Anda akan memilih contoh konfigurasi Astro, lalu menambahkannya ke codebase Anda sebagai `flightcontrol.json`

5. Sesuaikan konfigurasi bila diperlukan

6. Klik **Create Project** dan selesaikan langkah-langkah yang dibutuhkan (seperti menautkan akun AWS Anda).
</Steps>

### Pengaturan SSR

Untuk mendeploy dengan dukungan SSR, pastikan Anda terlebih dahulu menyiapkan adapter [`@astrojs/node`](/id/guides/integrations-guide/node/). Lalu, ikuti langkah-langkah di atas dengan memilih konfigurasi yang sesuai untuk Astro SSR.
