[Azure](https://azure.microsoft.com/) adalah platform cloud dari Microsoft. Anda dapat men-deploy situs Astro dengan layanan [Static Web Apps](https://aka.ms/staticwebapps) milik Microsoft Azure.

Panduan ini membawa Anda men-deploy situs Astro yang tersimpan di GitHub menggunakan Visual Studio Code. Silakan lihat panduan Microsoft untuk menggunakan [Azure Pipelines Task](https://learn.microsoft.com/en-us/azure/devops/pipelines/tasks/reference/azure-static-web-app-v0?view=azure-pipelines) untuk skenario lainnya.

## Prasyarat

Untuk mengikuti panduan ini, Anda memerlukan:

- Akun Azure dan subscription key. Anda dapat membuat [akun Azure gratis di sini](https://azure.microsoft.com/free).
- Kode aplikasi Anda telah di-push ke [GitHub](https://github.com/).
- [Ekstensi SWA](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-azurestaticwebapps) di [Visual Studio Code](https://code.visualstudio.com/).

## Cara men-deploy

<Steps>
1. Buka proyek Anda di VS Code.

2. Buka ekstensi Static Web Apps, masuk ke Azure, lalu klik tombol **+** untuk membuat Static Web App baru. Anda akan diminta memilih subscription key yang akan digunakan.

3. Ikuti wizard yang dimulai oleh ekstensi untuk memberi nama aplikasi, memilih preset framework, serta menetapkan app root (biasanya `/`) dan lokasi berkas hasil build (gunakan `/dist`). Astro tidak tercantum di templat bawaan Azure sehingga Anda perlu memilih `custom`. Wizard akan berjalan dan membuat [GitHub Action](https://github.com/features/actions) di folder `.github` repo Anda. (Folder ini akan dibuat otomatis jika belum ada.)
</Steps>

GitHub Action akan men-deploy aplikasi Anda (Anda bisa melihat progresnya di tab Actions repo Anda di GitHub). Setelah selesai, Anda dapat melihat aplikasi pada alamat yang ditampilkan di jendela progres Ekstensi SWA dengan mengklik tombol **Browse Website** (ini akan muncul setelah GitHub Action selesai berjalan).

## Masalah yang Diketahui

File yaml GitHub Action yang dibuat untuk Anda mengasumsikan penggunaan Node 14. Ini menyebabkan build Astro gagal. Untuk mengatasinya, perbarui file package.json proyek Anda dengan cuplikan ini.

```
  "engines": {
    "node": ">=18.0.0"
  },
```
