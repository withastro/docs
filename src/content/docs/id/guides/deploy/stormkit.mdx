Anda dapat mendeply proyek Astro ke [Stormkit](https://stormkit.io/), platform deployment untuk website statis, singleâ€‘page application (SPA), dan serverless function.

## Cara deploy

<Steps>
1. [Masuk ke Stormkit](https://app.stormkit.io/auth).

2. Melalui antarmuka pengguna, impor proyek Astro Anda dari salah satu dari tiga penyedia Git yang didukung (GitHub, GitLab, atau Bitbucket).

3. Masuk ke environment produksi proyek di Stormkit atau buat environment baru jika diperlukan.

4. Verifikasi build command pada [konfigurasi Stormkit](https://stormkit.io/docs/deployments/configuration). Secara default, Stormkit CI akan menjalankan `npm run build`, namun Anda bisa menentukan perintah build kustom di halaman ini.

5. Klik tombol "Deploy Now" untuk mendeply situs Anda.
</Steps>
