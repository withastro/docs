[Heroku](https://www.heroku.com/) adalah platform-as-a-service untuk membangun, menjalankan, dan mengelola aplikasi modern di cloud. Anda dapat melakukan deploy situs Astro ke Heroku menggunakan panduan ini.

:::danger
Instruksi berikut menggunakan [**`heroku-static-buildpack` yang sudah deprecated**](https://github.com/heroku/heroku-buildpack-static#warning-heroku-buildpack-static-is-deprecated). Silakan lihat [dokumentasi Heroku untuk menggunakan `heroku-buildpack-nginx`](https://github.com/dokku/heroku-buildpack-nginx) sebagai gantinya.
:::

## Cara melakukan deploy

<Steps>
1. Instal [Heroku CLI](https://devcenter.heroku.com/articles/heroku-cli).

2. Buat akun Heroku dengan [mendaftar](https://signup.heroku.com/).

3. Jalankan `heroku login` dan isi kredensial Heroku Anda:

   ```bash
   $ heroku login
   ```

4. Buat file bernama `static.json` di root proyek Anda dengan konten berikut:

   ```json title="static.json"
   {
     "root": "./dist"
   }
   ```

   Ini adalah konfigurasi situs Anda; baca selengkapnya di [heroku-buildpack-static](https://github.com/heroku/heroku-buildpack-static).

5. Siapkan remote git Heroku Anda:

   ```bash
   # versi awal
   $ git init
   $ git add .
   $ git commit -m "My site ready for deployment."

   # membuat app baru dengan nama tertentu
   $ heroku apps:create example

   # set buildpack untuk situs statis
   $ heroku buildpacks:set https://github.com/heroku/heroku-buildpack-static.git
   ```

6. Deploy situs Anda:

   ```bash
   # publikasi situs
   $ git push heroku master

   # membuka browser untuk melihat Dashboard Heroku CI
   $ heroku open
   ```
</Steps>
