Anda dapat mendeply proyek Astro ke [Surge](https://surge.sh/), platform publikasi web satu perintah yang dirancang untuk pengembang frontâ€‘end.

## Cara deploy

<Steps>
1. Instal [Surge CLI](https://www.npmjs.com/package/surge) secara global dari terminal, jika belum.

    ```shell
    npm install -g surge
    ```

2. Build situs Astro Anda dari root direktori proyek.

    ```shell
    npm run build
    ```

3. Deploy ke Surge menggunakan CLI.

    ```shell
    surge dist
    ```

    Anda dapat [menggunakan domain kustom di Surge](http://surge.sh/help/adding-a-custom-domain) saat deployment dengan menjalankan `surge dist yourdomain.com`.
</Steps>
