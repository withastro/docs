Anda dapat mendeploy proyek Astro Anda menggunakan [Buddy](https://buddy.works/), sebuah solusi CI/CD yang dapat melakukan build situs Anda dan mengirimkannya ke berbagai target deployment, termasuk server FTP dan penyedia hosting cloud.

:::note
Buddy sendiri tidak akan meng-host situs Anda. Buddy membantu Anda mengelola proses build dan mengirimkan hasilnya ke platform deployment pilihan Anda.
:::

## Cara deploy

<Steps>
1. [Buat akun **Buddy**](https://buddy.works/sign-up).

2. Buat proyek baru dan hubungkan dengan repositori git (GitHub, GitLab, BitBucket, repositori Git privat, atau gunakan Buddy Git Hosting).

3. Tambahkan pipeline baru.

4. Pada pipeline yang baru dibuat, tambahkan aksi **[Node.js](https://buddy.works/actions/node-js)**.

5. Pada aksi tersebut, tambahkan:

   ```bash
   npm install
   npm run build
   ```

6. Tambahkan aksi deployment â€” ada banyak pilihan yang bisa Anda gunakan, telusuri pada [katalog aksi Buddy](https://buddy.works/actions). Meskipun pengaturannya bisa berbeda-beda, pastikan **Source path** diatur ke `dist`.

7. Tekan tombol **Run**.
</Steps>
