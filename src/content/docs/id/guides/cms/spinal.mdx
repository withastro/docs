[Spinal](https://spinalcms.com/cms-for-astro/) adalah CMS berbasis Git komersial yang berfokus pada SaaS.

## Memulai

<Steps>
1. [Buat akun Spinal](https://spinalcms.com/signup/).

2. Hubungkan akun GitHub Anda ke Spinal.

3. Pilih repositori Astro Anda saat diminta.
</Steps>

Semua konten Markdown dari folder yang dipilih akan diimpor ke akun Spinal Anda dan siap untuk diedit.
