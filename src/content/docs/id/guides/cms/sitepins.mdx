[Sitepins](https://sitepins.com) adalah CMS headless berbasis Git untuk situs web yang dibangun dengan framework modern seperti Astro. Ia menawarkan editor WYSIWYG yang bersih, alur kerja konten dengan kontrol versi, serta integrasi mulus dengan Astro dan SSG lainnya.

## Memulai


1. [Buat akun Sitepins](https://app.sitepins.com/register).

2. Hubungkan repositori GitHub yang berisi proyek Astro Anda.

3. Konfigurasikan folder konten, media, dan konfigurasi Anda lalu mulai mengedit di editor visual.


Setelah terhubung, Sitepins akan menyinkronkan konten Anda dari folder yang dipilih dan menyediakan antarmuka visual untuk mengelola serta menerbitkan konten dengan kontrol versi Git penuh.
