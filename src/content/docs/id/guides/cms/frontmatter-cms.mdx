[Front Matter CMS](https://frontmatter.codes/) membawa CMS ke editor Anda; ia berjalan di dalam Visual Studio Code, Gitpod, dan banyak lagi.

## Integrasi dengan Astro

Di bagian ini, kita akan membahas cara menambahkan Front Matter CMS ke proyek Astro Anda.

### Prasyarat 

- Visual Studio Code
- Gunakan [template Astro Blog](https://github.com/withastro/astro/tree/main/examples/blog) untuk menyediakan konfigurasi dasar dan konten contoh guna memulai dengan Front Matter CMS.

### Instal ekstensi Front Matter CMS

Anda bisa mendapatkan ekstensi dari [Visual Studio Code Marketplace - Front Matter](https://marketplace.visualstudio.com/items?itemName=eliostruyf.vscode-front-matter) atau dengan mengeklik tautan berikut: [buka ekstensi Front Matter CMS di VS Code](vscode:extension/eliostruyf.vscode-front-matter)

### Inisialisasi proyek

Setelah Front Matter CMS terpasang, Anda akan mendapatkan ikon baru di Activity Bar. Saat diklik, ikon ini akan membuka panel **Front Matter CMS** di sidebar utama. Ikuti langkah berikut untuk menginisialisasi proyek Anda:

- Klik tombol **Initialize project** di panel Front Matter
- Layar sambutan akan terbuka, dan Anda bisa mulai menginisialisikan proyek
- Klik langkah pertama **Initialize project**
- Karena Astro adalah salah satu framework yang didukung, Anda bisa memilihnya dari daftar
- Daftarkan folder konten Anda, dalam kasus ini folder `src/content/blog`.

  :::note
  Pendaftaran folder diperlukan agar Front Matter CMS mengetahui di mana ia dapat menemukan dan membuat konten Anda. Anda bisa memiliki beberapa jenis folder seperti pages, blog, docs, dan lainnya.
  :::

- Anda akan diminta memasukkan nama folder. Secara default, akan diambil dari nama folder.

  :::note
  Nama ini digunakan selama proses pembuatan konten baru. Misalnya, memiliki beberapa pendaftaran folder memungkinkan Anda memilih jenis konten yang ingin dibuat.
  :::

- Klik **Show the dashboard** untuk membuka dasbor konten

  :::tip
  Setelah Front Matter CMS diinisialisasi, Anda dapat membuka dasbor sebagai berikut:

  - Menggunakan pintasan keyboard: <kbd>alt</kbd> + <kbd>d</kbd> (Windows & Linux) atau <kbd>options</kbd> + <kbd>d</kbd> (macOS)
  - Buka command palette dan cari `Front Matter: Open dashboard`
  - Klik ikon **Front Matter** pada judul panel atau berkas.
  :::

### Konfigurasi proyek

Setelah proyek diinisialisasi, Anda akan mendapatkan berkas konfigurasi `frontmatter.json` dan folder `.frontmatter` di root proyek Anda.

<FileTree title="Struktur Proyek">

    - mediaDb.json

- astro.config.mjs

- package.json

#### Konfigurasi content-type

Content-type adalah cara Front Matter CMS mengelola konten Anda. Setiap content-type berisi serangkaian field yang dapat didefinisikan per jenis konten yang ingin Anda gunakan untuk situs Anda.

Field-field tersebut sesuai dengan front matter dari konten halaman Anda.

Anda dapat mengonfigurasi content-type di berkas `frontmatter.json`. 

- Buka berkas `frontmatter.json`
- Ganti array `frontMatter.taxonomy.contentTypes` dengan konfigurasi content-type berikut:

```json title="frontmatter.json"
"frontMatter.taxonomy.contentTypes": [
  {
    "name": "default",
    "pageBundle": false,
    "previewPath": "'blog'",
    "filePrefix": null,
    "fields": [
      {
        "title": "Title",
        "name": "title",
        "type": "string",
        "single": true
      },
      {
        "title": "Description",
        "name": "description",
        "type": "string"
      },
      {
        "title": "Publishing date",
        "name": "pubDate",
        "type": "datetime",
        "default": "{{now}}",
        "isPublishDate": true
      },
      {
        "title": "Content preview",
        "name": "heroImage",
        "type": "image",
        "isPreviewImage": true
      }
    ]
  }
]
```

:::note
Konfigurasi ini memastikan content-type Front Matter sesuai dengan skema content collection dari template blog Astro.
:::

:::tip
Anda dapat menemukan lebih banyak informasi tentang content-type dan field yang didukung di [bagian dokumentasi pembuatan konten](https://frontmatter.codes/docs/content-creation) dari Front Matter CMS.
:::

### Pratinjau artikel Anda di editor

Dari panel **Front Matter CMS**, klik tombol **Start server**. Tindakan ini akan menjalankan *local dev server* Astro. Setelah berjalan, Anda dapat membuka dasbor konten, memilih salah satu artikel, lalu klik tombol **Open preview** untuk membuka artikel di editor.

### Buat artikel baru

Buka **Front Matter CMS Dashboard**; Anda bisa melakukannya sebagai berikut: 

- Buka dasbor konten Front Matter CMS
- Klik tombol **Create content**
- Front Matter akan menanyakan judul artikel. Isikan lalu tekan enter
- Artikel baru Anda akan dibuat dan dibuka di editor. Anda dapat mulai menulis artikel Anda.

### Menggunakan Markdoc dengan Front Matter CMS

Untuk menggunakan Markdoc dengan Front Matter CMS, Anda harus mengonfigurasinya di `frontMatter.content.supportedFileTypes`. Pengaturan ini memberi tahu CMS tipe berkas apa yang dapat dikelola.

Anda dapat mengonfigurasikannya sebagai berikut:

```json title="frontmatter.json"
"frontMatter.content.supportedFileTypes": [ "md", "markdown", "mdx", "mdoc" ]
```

Untuk memungkinkan konten Anda dibuat sebagai Markdoc, tentukan properti `fileType` pada content-type.

```json title="frontmatter.json" ins={7}
"frontMatter.taxonomy.contentTypes": [
  {
    "name": "default",
    "pageBundle": false,
    "previewPath": "'blog'",
    "filePrefix": null,
    "fileType": "mdoc",
    "fields": [
      {
        "title": "Title",
        "name": "title",
        "type": "string",
        "single": true
      },
      {
        "title": "Description",
        "name": "description",
        "type": "string"
      },
      {
        "title": "Publishing date",
        "name": "pubDate",
        "type": "datetime",
        "default": "{{now}}",
        "isPublishDate": true
      },
      {
        "title": "Content preview",
        "name": "heroImage",
        "type": "image",
        "isPreviewImage": true
      }
    ]
  }
]
```
