[Sitecore Experience Manager (XM)](https://www.sitecore.com/products/experience-manager) adalah sistem manajemen konten tingkat enterprise yang dibangun di atas ASP.NET.

## Memulai

<Steps>
1. [Buat situs web Headless Sitecore](https://doc.sitecore.com/xp/id/developers/sxa/103/sitecore-experience-accelerator/create-a-headless-tenant-and-site.html) mengikuti dokumentasi resmi Sitecore.

2. Jalankan perintah inisialisasi proyek berikut di terminal Anda: 
    ```shell
    npx @astro-sitecore-jss/create-astro-sitecore-jss@latest
    ```

3. Ikuti instruksi di terminal untuk membuat proyek Anda.
</Steps>
