[Craft CMS](https://craftcms.com/) adalah CMS *open source* yang fleksibel dengan pengalaman penulisan yang mudah diakses. Craft menyertakan *front end*-nya sendiri, tetapi juga dapat digunakan sebagai headless CMS untuk menyediakan konten ke proyek Astro Anda.

## Sumber Resmi

- Lihat panduan resmi Craft CMS [GraphQL API](https://craftcms.com/docs/5.x/development/graphql.html)
- Dokumentasi resmi untuk *config setting* Craft [`headlessMode`](https://craftcms.com/docs/5.x/reference/config/general.html#headlessmode)
