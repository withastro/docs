Sesuaikan editor kode Anda untuk meningkatkan pengalaman pengembangan Astro dan membuka fitur-fitur baru.

## VS Code

[VS Code](https://code.visualstudio.com/) adalah editor kode populer untuk pengembang web, dibuat oleh Microsoft. Mesin VS Code juga menjalankan editor kode dalam-browser seperti [GitHub Codespaces](https://github.com/features/codespaces) dan [Gitpod](https://gitpod.io/).

Astro bekerja dengan editor apa pun. Namun, VS Code adalah editor yang kami rekomendasikan untuk proyek Astro. Kami memelihara [Ekstensi Astro untuk VS Code](https://marketplace.visualstudio.com/items?itemName=astro-build.astro-vscode) resmi yang membuka sejumlah fitur kunci dan peningkatan DX (developer experience) untuk proyek Astro.

- Penyorotan sintaks untuk berkas `.astro`.
- Informasi tipe TypeScript untuk berkas `.astro`.
- [VS Code IntelliSense](https://code.visualstudio.com/docs/editor/intellisense) untuk melengkapi kode, hint, dan lainnya.

Untuk memulai, pasang [Ekstensi Astro untuk VS Code](https://marketplace.visualstudio.com/items?itemName=astro-build.astro-vscode) sekarang.

## Zed

[Zed](https://zed.dev/) adalah editor kode multiplayer berkinerja tinggi yang dioptimalkan untuk kecepatan dan proyek besar. [Ekstensi Astro](https://zed.dev/extensions/astro) mereka menyertakan fitur seperti penyorotan sintaks untuk berkas `.astro`, pelengkapan kode, pemformatan, diagnostik, dan go-to-definition.

## JetBrains IDEs

[WebStorm](https://www.jetbrains.com/webstorm/) adalah IDE JavaScript dan TypeScript yang menambahkan dukungan untuk Astro Language Server pada versi 2024.2. Pembaruan ini menghadirkan fitur seperti penyorotan sintaks, pelengkapan kode, dan pemformatan.

Pasang plugin resmi melalui [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/20959-astro) atau dengan mencari "Astro" pada tab Plugins di IDE Anda. Anda dapat mengaktif/nonaktifkan language server di `Settings | Languages & Frameworks | TypeScript | Astro`.

Untuk informasi lebih lanjut tentang dukungan Astro di WebStorm, lihat [Dokumentasi WebStorm untuk Astro](https://www.jetbrains.com/help/webstorm/astro.html).

## Editor Kode Lain

Komunitas kami yang luar biasa memelihara sejumlah ekstensi untuk editor populer lainnya, termasuk:

- [Ekstensi VS Code di Open VSX](https://open-vsx.org/extension/astro-build/astro-vscode) <span style="margin: 0.25em;"><Badge text="Official" /></span> — Ekstensi Astro resmi untuk VS Code, tersedia di registry Open VSX untuk editor seperti [Cursor](https://cursor.com) atau [VSCodium](https://vscodium.com/).
- [Plugin Vim](https://github.com/wuelnerdotexe/vim-astro) <span style="margin: 0.25em;"><Badge class="neutral-badge" text="Community" /></span> — Menyediakan penyorotan sintaks, indentasi, dan dukungan code folding untuk Astro di Vim atau Neovim.
- Plugin Neovim [LSP](https://github.com/neovim/nvim-lspconfig/blob/master/doc/configs.md#astro) dan [TreeSitter](https://github.com/virchau13/tree-sitter-astro) <span style="margin: 0.25em;"><Badge class="neutral-badge" text="Community" /></span> — Menyediakan penyorotan sintaks, parsing treesitter, dan pelengkapan kode untuk Astro di Neovim.
- Emacs — Lihat instruksi [Konfigurasi Emacs dan Eglot](https://medium.com/@jrmjrm/configuring-emacs-and-eglot-to-work-with-astro-language-server-9408eb709ab0) <span style="margin: 0.25em;"><Badge class="neutral-badge" text="Community" /></span> agar bekerja dengan Astro.
- [Penyorotan sintaks Astro untuk Sublime Text](https://packagecontrol.io/packages/Astro) <span style="margin: 0.25em;"><Badge class="neutral-badge" text="Community" /></span> — Paket Astro untuk Sublime Text, tersedia di package manager Sublime Text.
- [Ekstensi Nova](https://extensions.panic.com/extensions/sciencefidelity/sciencefidelity.astro/) <span style="margin: 0.25em;"><Badge class="neutral-badge" text="Community" /></span> — Menyediakan penyorotan sintaks dan pelengkapan kode untuk Astro di Nova.

## Editor Dalam-Browser

Selain editor lokal, Astro juga berjalan baik di editor hosted dalam browser, termasuk:

- [StackBlitz](https://stackblitz.com/) dan [CodeSandbox](https://codesandbox.io/) — editor online yang berjalan di browser Anda, dengan dukungan penyorotan sintaks bawaan untuk berkas `.astro`. Tidak perlu pemasangan atau konfigurasi!
- [GitHub.dev](https://github.dev/) — memungkinkan Anda memasang ekstensi Astro VS Code sebagai [web extension](https://code.visualstudio.com/api/extension-guides/web-extensions), yang memberi akses hanya sebagian fitur ekstensi penuh. Saat ini, hanya penyorotan sintaks yang didukung.
- [IDX](https://idx.dev) dan [Gitpod](https://gitpod.io/) — lingkungan dev penuh di cloud yang dapat memasang Ekstensi Astro VS Code resmi dari Open VSX.

## Alat Lain

### ESLint

[ESLint](https://eslint.org/) adalah linter populer untuk JavaScript dan JSX. Untuk dukungan Astro, Anda dapat memasang [plugin komunitas](https://github.com/ota-meshi/eslint-plugin-astro).

Lihat [User Guide proyek](https://ota-meshi.github.io/eslint-plugin-astro/user-guide/) untuk informasi pemasangan dan pengaturan ESLint di proyek Anda.

### Stylelint

[Stylelint](https://stylelint.io/) adalah linter populer untuk CSS. [Konfigurasi Stylelint yang dipelihara komunitas](https://github.com/ota-meshi/stylelint-config-html) menyediakan dukungan Astro.

Instruksi pemasangan, integrasi editor, dan informasi tambahan tersedia di README proyek tersebut.

### Biome

[Biome](https://biomejs.dev/) adalah linter dan formatter serba-ada untuk web. [Biome saat ini mendukung sebagian berkas `.astro`](https://biomejs.dev/internals/language-support/#html-super-languages-support), dan dapat digunakan untuk lint/format frontmatter di berkas `.astro`.

### Prettier

[Prettier](https://prettier.io/) adalah formatter populer untuk JavaScript, HTML, CSS, dan lainnya. Jika Anda menggunakan [Ekstensi Astro VS Code](https://marketplace.visualstudio.com/items?itemName=astro-build.astro-vscode), pemformatan kode dengan Prettier sudah termasuk.

Untuk menambahkan dukungan pemformatan berkas `.astro` di luar editor (mis. CLI) atau di editor yang tidak mendukung tooling editor kami, pasang [plugin Prettier resmi untuk Astro](https://github.com/withastro/prettier-plugin-astro).

<Steps>
1. Pasang `prettier` dan `prettier-plugin-astro`.

    <PackageManagerTabs>
      <Fragment slot="npm">
      ```shell
      npm install --save-dev --save-exact prettier prettier-plugin-astro
      ```
      </Fragment>
      <Fragment slot="pnpm">
      ```shell
      pnpm add --save-dev --save-exact prettier prettier-plugin-astro
      ```
      </Fragment>
      <Fragment slot="yarn">
      ```shell
      yarn add --dev --exact prettier prettier-plugin-astro
      ```
      </Fragment>
    </PackageManagerTabs>

2. Buat berkas konfigurasi `.prettierrc` (atau `.prettierrc.json`, `.prettierrc.mjs`, atau [format lain yang didukung](https://prettier.io/docs/configuration)) di root proyek Anda dan tambahkan `prettier-plugin-astro` ke sana.

    Pada berkas ini, tentukan juga parser untuk berkas Astro secara manual.

    ```json title=".prettierrc"
    {
      "plugins": ["prettier-plugin-astro"],
      "overrides": [
        {
          "files": "*.astro",
          "options": {
            "parser": "astro"
          }
        }
      ]
    }
    ```

3. Opsional, pasang plugin Prettier lain untuk proyek Anda, dan tambahkan ke konfigurasi. Beberapa plugin tambahan mungkin perlu diurutkan secara spesifik. Misalnya, jika Anda memakai Tailwind, `prettier-plugin-tailwindcss` harus [menjadi plugin terakhir dalam array plugins](https://github.com/tailwindlabs/prettier-plugin-tailwindcss#compatibility-with-other-prettier-plugins).

    ```json title=".prettierrc"
    {
      "plugins": [
        "prettier-plugin-astro",
        "prettier-plugin-tailwindcss" // harus terakhir
      ],
      "overrides": [
        {
          "files": "*.astro",
          "options": {
            "parser": "astro"
          }
        }
      ]
    }
    ```

4. Jalankan perintah berikut di terminal untuk memformat berkas Anda.

    <PackageManagerTabs>
      <Fragment slot="npm">
      ```shell
      npx prettier . --write
      ```
      </Fragment>
      <Fragment slot="pnpm">
      ```shell
      pnpm exec prettier . --write
      ```
      </Fragment>
      <Fragment slot="yarn">
      ```shell
      yarn exec prettier . --write
      ```
      </Fragment>
    </PackageManagerTabs>
</Steps>

Lihat [README plugin Prettier](https://github.com/withastro/prettier-plugin-astro/blob/main/README.md) untuk informasi lebih lanjut tentang opsi yang didukung, cara menyiapkan Prettier di VS Code, dan lainnya.

### dprint

[dprint](https://dprint.dev/) adalah formatter kode yang sangat dapat dikonfigurasi dan mendukung banyak bahasa, termasuk JavaScript, TypeScript, CSS, dan lainnya. Dukungan untuk berkas `.astro` dapat ditambahkan menggunakan [plugin markup_fmt](https://github.com/g-plane/markup_fmt).
