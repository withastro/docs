[Craft CMS](https://craftcms.com/)는 접근 가능한 저작 환경을 갖춘 유연한 오픈 소스 CMS입니다. 자체 프런트엔드가 포함되어 있지만 헤드리스 CMS로 사용하여 Astro 프로젝트에 콘텐츠를 제공할 수도 있습니다.

## 공식 리소스

- 공식 Craft CMS [GraphQL API 가이드](https://craftcms.com/docs/5.x/development/graphql.html) 알아보기
- Craft의 [`headlessMode` 구성 설정](https://craftcms.com/docs/5.x/reference/config/general.html#headlessmode) 공식 문서
