[Sitepins](https://sitepins.com)는 Astro와 같은 최신 프레임워크로 구축된 웹사이트를 위한 Git 기반의 헤드리스 CMS입니다. 깔끔한 WYSIWYG 편집기, 버전 제어 콘텐츠 워크플로우, Astro 및 기타 SSG와의 원활한 통합을 제공합니다.

## 시작하기


1. [Sitepins 계정을 생성](https://app.sitepins.com/register)합니다.

2. Astro 프로젝트가 포함된 Github 리포지토리를 연결합니다.

3. 콘텐츠, 미디어 및 구성 폴더를 구성하고 시각적 편집기에서 편집을 시작합니다.


연결되면 Sitepins는 선택한 폴더에서 콘텐츠를 동기화하고 전체 Git 버전 제어를 통해 콘텐츠를 관리하고 게시할 수 있는 시각적 인터페이스를 제공합니다.
