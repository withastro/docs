:::caution[제거됨]
Astro DB 통합은 v6.4에서 지원 중단되었으며, Astro v7.0부터 완전히 제거되었습니다.

프로젝트에서 이 통합을 사용하고 있었다면, 데이터베이스 기능을 구현하기 위해 서드파티 라이브러리로 마이그레이션하는 것을 권장합니다. 프로젝트 업데이트 방법에 대한 자세한 내용과 권장 사항은 [Astro v7.0 마이그레이션 가이드](/ko/guides/upgrade-to/v7/#제거-astrojsdb)를 참조하세요.
:::
