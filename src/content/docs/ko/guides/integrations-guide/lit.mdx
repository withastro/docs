i18nReady: true

:::caution[더 이상 사용되지 않음]
[Lit](https://lit.dev/) 사용자 정의 요소에 대한 요청 시 렌더링 및 클라이언트 측 하이드레이션을 지원하는 이 Astro 통합은 Astro 5.0에서 더 이상 사용되지 않습니다.
:::

클라이언트 측 스크립트 태그를 추가하여 클라이언트 컴포넌트에서 Lit를 계속 사용할 수 있습니다. 예를 들어:

```astro
<script>
  import "../components/MyTabs";
</script>

<my-tabs title="These are my tabs">...</my-tabs>
```

Lit 통합을 직접 유지 관리하는 데 관심이 있다면 [최근 게시된 버전의 `@astrojs/lit`](https://github.com/withastro/astro/tree/astro%404.13.0/packages/integrations/lit)를 시작점으로 삼아 관련 패키지를 업그레이드할 수 있습니다.
