이제 웹에 작업 중인 사이트가 있으므로 페이지와 게시물을 추가할 차례입니다!

## 앞을 내다보며

이 단계에서는 다음을 수행합니다.
-`.astro` 구문을 사용하여 첫 번째 Astro 페이지 생성
- Markdown (`.md`) 파일이 포함된 블로그 게시물 추가
- `<style>`을 사용하여 개별 페이지 스타일 지정
- 페이지 전체에 글로벌 스타일 적용

 그 과정에서 **`.astro` 파일의 두 섹션**이 함께 작동하여 페이지를 만드는 방법과 페이지에서 변수 및 조건부 렌더링을 사용하는 방법을 배우게 됩니다.
