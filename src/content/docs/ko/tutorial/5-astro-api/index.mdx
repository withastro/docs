이제 블로그 게시물이 있으므로 Astro의 API를 사용하여 파일 작업을 수행할 차례입니다!

## 학습 내용 미리보기

이 단계에서는 색인 페이지, 태그 페이지 및 RSS 피드를 사용하여 블로그를 강화하게 됩니다.

그 과정에서 다음을 사용하는 방법을 배우게 됩니다.
- 프로젝트 파일의 데이터에 액세스하기 위한 `import.meta.glob()`
- 한 번에 여러 페이지 (경로)를 생성하는 `getStaticPaths()`
- RSS 피드를 생성하는 Astro RSS 패키지
