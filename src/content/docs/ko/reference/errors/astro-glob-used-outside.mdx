:::caution[사용되지 않음]
이 오류는 `Astro.glob()`이 제거되면서 Astro v6.0.0에서 함께 제거되었습니다.
:::

> **AstroGlobUsedOutside**: `Astro.glob(GLOB_STR)` can only be used in `.astro` files. `import.meta.glob(GLOB_STR)` can be used instead to achieve a similar result.

## 무엇이 잘못되었나요?

`Astro.glob()`은 오직 `.astro` 파일에서만 사용할 수 있습니다. 대신 다른 파일에서는 [`import.meta.glob()`](https://ko.vite.dev/guide/features.html#glob-import)을 사용하여 동일한 결과를 얻을 수 있습니다.
