> **오류 메시지 예시:**<br/>
The content loader for the collection **blog** returned an entry with an invalid `id`:<br/>
&#123;<br/>
  "id": 1,<br/>
  "title": "Hello, World!"<br/>
&#125;

## 무엇이 잘못되었나요?
콘텐츠 로더가 유효하지 않은 `id`를 반환했습니다.
항목의 `id`가 문자열인지 확인하세요.
자세한 내용은 [콘텐츠 컬렉션 문서](/ko/guides/content-collections/)를 참조하세요.
