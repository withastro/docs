> `Astro.cache` is not available because the cache feature is not enabled.

## 무엇이 잘못되었나요?
`Astro.cache` 또는 `context.cache`를 사용했지만, Astro 설정에서 캐시 기능이 활성화되지 않았을 때 발생합니다.
