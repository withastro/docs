> `query.include` and `query.exclude` cannot be used together.

## 무엇이 잘못되었나요?

메모리 캐시 제공자의 `query.include`와 `query.exclude` 옵션은 서로 동시에 사용할 수 없습니다.
`include`는 캐시 키에 영향을 주는 특정 쿼리 파라미터만 허용할 때 사용하고, `exclude`는 특정 파라미터를 제외할 때 사용합니다.
`include`가 설정되면 그 외의 모든 파라미터는 자동으로 무시됩니다.
