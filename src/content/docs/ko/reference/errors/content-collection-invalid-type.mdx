> **오류 메시지 예시:**<br/>
Invalid collection type "data". Remove the type from your collection definition in your content config file.

## 무엇이 잘못되었나요?

콘텐츠 컬렉션에는 더 이상 `type` 필드가 포함되어서는 안 됩니다. 콘텐츠 설정 파일에서 이 필드를 제거하세요.
자세한 내용은 [Astro 6 마이그레이션 가이드](/ko/guides/upgrade-to/v6/#removed-legacy-content-collections)를 참고하세요.
