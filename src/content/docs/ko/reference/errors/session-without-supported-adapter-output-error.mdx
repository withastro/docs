> **SessionWithoutSupportedAdapterOutputError**: Sessions require an adapter that supports server output. The adapter must set `"server"` in the `buildOutput` adapter feature.

## 무엇이 잘못되었나요?
세션을 사용하려면 어댑터가 서버 출력을 지원해야 합니다.
