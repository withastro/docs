> **PrerenderClientAddressNotAvailable**: `Astro.clientAddress` cannot be used inside prerendered routes

## 무엇이 잘못되었나요?
`Astro.clientAddress` 속성은 사전 렌더링된 경로에서 사용할 수 없습니다.
