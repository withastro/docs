> **FileGlobNotSupported**: Glob patterns are not supported in the `file` loader. Use the `glob` loader instead.

## 무엇이 잘못되었나요?
`file` 로더에는 단일 로컬 파일이 전달되어야 합니다. Glob 패턴은 지원되지 않습니다. 내장된 `glob` 로더에서 여러 로컬 파일의 패턴을 사용하여 항목을 생성하세요.
