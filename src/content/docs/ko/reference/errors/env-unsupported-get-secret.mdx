:::caution[더 이상 사용되지 않음]
이 오류는 이전 버전의 Astro에서 발생하므로 더 이상 사용되지 않습니다. 프로젝트를 최신 버전으로 업그레이드할 수 없다면 [관리되지 않는 이전 문서의 스냅샷](/ko/upgrade-astro/#이전-문서-유지-관리되지-않음)을 참조하여 도움을 받을 수 있습니다.
:::


> **EnvUnsupportedGetSecret**: `astro:env/server` exported function `getSecret` is not supported by your adapter.

## 무엇이 잘못되었나요?
`astro:env/server`에서 내보낸 함수 `getSecret()`이 사용중인 어댑터에서 지원되지 않습니다.
