:::caution[사용되지 않음]
이 오류는 `Astro.glob()`이 제거되면서 Astro v6.0.0에서 함께 제거되었습니다.
:::

> **AstroGlobNoMatch**: `Astro.glob(GLOB_STR)` did not return any matching files.

## 무엇이 잘못되었나요?

`Astro.glob()`이 일치하는 파일을 반환하지 않았습니다. glob 패턴에 오타가 있을 수 있습니다.
