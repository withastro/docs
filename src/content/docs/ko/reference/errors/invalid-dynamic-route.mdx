> **InvalidDynamicRoute**: The INVALID_PARAM param for route ROUTE is invalid. Received **RECEIVED**.

## 무엇이 잘못되었나요?

동적 경로 매개변수가 잘못되었습니다. 이 문제는 `undefined` 매개변수 또는 [나머지 매개변수](/ko/guides/routing/#나머지-매개변수) 누락으로 인해 발생하는 경우가 많습니다.
