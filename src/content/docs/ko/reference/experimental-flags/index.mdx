---
title: 실험적 플래그 구성
i18nReady: true
---

실험적 기능은 Astro 구성 파일에서 플래그를 활성화한 후에만 사용할 수 있습니다.

```js title="astro.config.mjs"
import { defineConfig } from 'astro/config';

export default defineConfig({
    experimental: {
        // 새로운 기능을 시도하기 위해
        // 실험적 플래그를 활성화하세요.
    },
});
```

Astro는 테스트와 피드백을 위해 사용자가 새로운 기능을 조기에 접근할 수 있도록 실험적 플래그를 제공합니다.

이러한 플래그를 통해 문제를 보고하고 의견을 공유함으로써 기능 개발에 참여할 수 있습니다. 이러한 기능들은 안정성이 보장되지 않으며, 기능이 활발히 개발되는 동안에는 작은 `patch` 릴리스에서도 호환성을 깨는 변경사항이 포함될 수 있습니다.

[Astro를 자주 업데이트](/ko/upgrade-astro/#최신-버전으로-업그레이드)하고, [Astro 변경 로그](https://github.com/withastro/astro/blob/main/packages/astro/CHANGELOG.md)의 릴리스 노트를 확인하여 프로젝트 코드에 필요한 변경사항을 파악하는 것을 권장합니다. 실험적 기능 문서는 항상 현재 릴리스된 버전에 대해서만 업데이트됩니다.
