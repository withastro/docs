`astro/hono` 모듈은 Hono 호환 래퍼 형태로 구현된 [고급 라우팅](/ko/guides/routing/#고급-라우팅) 핸들러를 제공합니다.

## `astro/hono`에서 가져오기

```ts
import {
  FetchState,
  astro,
  actions,
  cache,
  i18n,
  middleware,
  pages,
  redirects,
  sessions,
  trailingSlash,
} from "astro/hono";
```

`astro/hono` 모듈은 [`astro/fetch`](/ko/reference/modules/astro-fetch/)와 동일한 핸들러 이름을 내보내지만, 각각 Hono 미들웨어 함수를 반환합니다. 이를 통해 Astro 핸들러를 생태계의 다양한 Hono 미들웨어와 함께 조합하여 사용할 수 있습니다.
