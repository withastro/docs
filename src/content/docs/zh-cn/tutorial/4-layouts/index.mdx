## 本单元目标

在这个单元中，你将构建布局，以在你的页面和博客文章之间共享常见元素和样式。

为了实现这一目标，你将
- 创建可复用的布局组件
- 使用 `<slot />` 将内容传递给你的布局
- 将 Markdown frontmatter 中的数据传递给你的布局
- 嵌套多个布局
