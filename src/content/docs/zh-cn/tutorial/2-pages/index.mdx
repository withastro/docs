现在你在网络上有了一个基础完善的网站，是时候来添加页面和博客文章了！

## 你现在在哪？

你可以在 [GitHub](https://github.com/withastro/blog-tutorial-demo/tree/unit-2/start) 或者 [StackBlitz](https://stackblitz.com/github/withastro/blog-tutorial-demo/tree/unit-2/start?file=src/pages/index.astro) 上找到这个教程当前阶段的代码。

检查你的代码是否与此示例匹配。或者如果你刚刚加入我们，请在 StackBlitz 上 Fork，并从这里开始在浏览器中进行编程。

## 本单元目标

在本单元中，你将
- 使用 `.astro` 语法创建你的第一个 Astro 页面
- 使用 Markdown (`.md`) 文件编写博客文章
- 使用 `<style>` 设置单个页面的样式
- 跨页面应用全局样式

  在此过程中，你将了解`.astro` 文件的两个部分如何协同工作来创建页面，以及如何在页面上使用变量和条件渲染。
