现在你在网络上有了一个基础完善的网站，是时候来添加页面和博客文章了！

## 接下来

在本单元中，你将
- 使用 `.astro` 语法创建你的第一个 Astro 页面
- 使用 Markdown (`.md`) 文件编写博客文章
- 使用 `<style>` 设置单个页面的样式
- 跨页面应用全局样式

  在此过程中，你将了解`.astro` 文件的两个部分如何协同工作来创建页面，以及如何在页面上使用变量和条件渲染。
