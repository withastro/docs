你能将你的 Astro 项目部署到 [Surge](https://surge.sh/)——这是一个为前端开发者而设计，只需要单行命令就可以完成网页发布的平台。

## 如何部署

1. 在终端全局安装 [Surge CLI](https://www.npmjs.com/package/surge)，如果你还没有安装过的话。

    ```shell
    npm install -g surge
    ```

2. 在你的根项目目录下构建你的 Astro 网站。

    ```shell
    npm run build
    ```

3. 通过命令行部署到 Surge。

    ```shell
    surge dist
    ```

    在部署时，你也可以使用 [自定义域名](http://surge.sh/help/adding-a-custom-domain)，即运行`surge dist yourdomain.com`。
