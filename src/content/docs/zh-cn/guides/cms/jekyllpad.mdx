[JekyllPad](https://www.jekyllpad.com) 是一款轻量级、基于浏览器的 CMS，可直接连接到你的 GitHub 仓库。

它提供现代化的 WYSIWYG + Markdown 编辑器，改动会直接提交到你的仓库，并且 100% 在客户端运行，因此你无需搭建服务器或学习 Git 就能管理 Astro 的内容。

它是一个完全在用户浏览器中运行的客户端应用。这意味着数据、OAuth 和安全令牌都保留在用户的浏览器中。无需安装，支持任何移动或桌面浏览器。
