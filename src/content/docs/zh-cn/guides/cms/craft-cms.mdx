[Craft CMS](https://craftcms.com/) 是一款灵活的开源内容管理系统，具有易于访问的创作体验。它拥有自己的前端，当然也可以仅被用作无头 CMS 来向你的 Astro 项目提供内容。

## 官方资源

- 查看官方 Craft CMS [GraphQL API 指南](https://craftcms.com/docs/5.x/development/graphql.html)
- 有关 Craft [`headlessMode` 的配置设置](https://craftcms.com/docs/5.x/reference/config/general.html#headlessmode) 的官方文档
