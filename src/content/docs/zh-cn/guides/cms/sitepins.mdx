[Sitepins](https://sitepins.com) 是一个基于 Git 的无头 CMS，适用于使用 Astro 等现代框架构建的网站。它提供一个简洁的所见即所得编辑器、版本控制的内容工作流以及与 Astro 和其他 SSG 的无缝集成。

## 快速开始


1. 创建一个 [Sitepins](https://app.sitepins.com/register) 账号。

2. 连接包含你的 Astro 项目的 GitHub 仓库。

3. 配置你的内容、媒体和配置文件夹，并在可视化编辑器中开始编辑。


连接后，Sitepins 将同步选定文件夹中的内容，并提供拥有完整 Git 版本控制的可视化界面来管理和发布内容。
