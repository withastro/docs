[Decap CMS](https://decapcms.org/)（前身为 Netlify CMS）是一个基于 Git 的开源内容管理系统。
## 社区资源 

- 使用 [`astro-decap-cms`](https://github.com/advanced-astro/astro-decap-cms) Astro 集成，将 Decap CMS 添加到你的项目中。

- 你可以从 [Astro 博客入门与 Decap CMS](https://github.com/advanced-astro/astro-decap-cms-starter) 开始。

- 博客文章：[使用基于 Git 的 CMS 管理 Astro 站点内容](https://aalam.vercel.app/blog/astro-and-git-cms-netlify) 作者：Aftab Alam

- YouTube教程：[在几分钟内使用 Astro 和 Netlify CMS 创建自定义博客！](https://www.youtube.com/watch?v=3yip2wSRX_4) 作者：Kumail Pirzada
