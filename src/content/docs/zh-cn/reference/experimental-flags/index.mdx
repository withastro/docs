---
title: 配置实验性标志
i18nReady: true
---

实验性功能仅在 Astro 配置文件中启用后有效。

```js title="astro.config.mjs"
import { defineConfig } from 'astro/config';

export default defineConfig({
    experimental: {
        // 启用实验性标志
        // 以尝试新功能
    },
});
```

Astro 提供实验性标志，让用户能够尽早地访问到新功能以进行测试和反馈。

这些标志允许你通过报告问题和分享你的意见来参与到功能开发当中。这些功能无法保证稳定，且在积极开发该功能时，即使在小的 `patch` 版本中也可能存在部分破坏性变更。

我们建议你积极地 [更新 Astro](/zh-cn/upgrade-astro/#升级到最新版本)，并关注 [Astro changelog](https://github.com/withastro/astro/blob/main/packages/astro/CHANGELOG.md) 中的发布说明，它将通知你项目代码所需的任何更改。实验性功能文档将始终仅针对当前发布的版本进行更新。
