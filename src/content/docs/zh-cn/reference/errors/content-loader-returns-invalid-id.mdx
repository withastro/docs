> **错误信息示例：**<br/>
集合 **blog** 的内容加载器返回了一个具有无效 `id` 的条目：<br/>
&#123;<br/>
  "id": 1,<br/>
  "title": "Hello, World!"<br/>
&#125;

## 哪里出了问题？
内容加载器返回了无效的 `id`。
请确保条目的 `id` 为字符串。
有关更多信息，请参阅 [内容集合文档](/zh-cn/guides/content-collections/)。
