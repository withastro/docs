> **NoAdapterInstalledServerIslands**: 无法在缺少适配器的情况下使用服务器群岛。请为你的最终部署环节安装并配置合适的服务器适配器。

## 哪里出了问题？

要使用服务器群岛，存在与服务器端渲染相同的约束，因此需要适配器。

**另见参阅：**
-  [按需渲染](/zh-cn/guides/on-demand-rendering/)
