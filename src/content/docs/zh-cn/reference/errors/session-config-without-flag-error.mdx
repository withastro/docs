> 在没有启用 `experimental.session` 标志的前提下提供了会话（session）配置。

## 哪里出了问题？
在没有启用 `experimental.session` 标志却进行了会话存储配置时，抛出错误。

-  [experimental.session](/zh-cn/reference/experimental-flags/sessions/)
