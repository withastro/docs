> **错误信息示例：**<br/>
**blog** 的加载器返回了无效数据。<br/>
对象缺少必要的属性 "id"。

## 哪里出了问题？
内容集合的加载器返回了无效数据。
内联加载器必须返回具有唯一 ID 字段的对象数组或以 ID 作为键，条目作为值的普通对象。
