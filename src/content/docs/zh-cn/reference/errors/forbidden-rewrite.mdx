> **ForbiddenRewrite**: 当配置 output 为 'server' 时，你尝试将静态路由 'TO' 重写为按需路由 'FROM'。<br/><br/>静态路由 'TO' 是由组件 'COMPONENT' 渲染的，该组件被标记为预渲染。该操作是被禁止的，因为在构建过程中，组件 'COMPONENT' 被编译为一个 HTML 文件，Astro 在运行时无法检索该文件。

## 哪里出了问题？

当配置 output 为 `"server"` 时，`Astro.rewrite()` 不能将静态路由重写为按需路由。
