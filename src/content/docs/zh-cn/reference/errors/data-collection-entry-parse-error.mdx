> `COLLECTION_ENTRY_NAME` 解析失败。

## 哪里发生了错误？

`type: 'data'` 的集合条目必须返回一个包含有效 JSON（对于 `.json` 条目）、YAML（对于 `.yaml` 条目）或 TOML（对于 `.toml` 条目）的对象。
