:::caution[废弃]
从 Astro 4.0 开始弃用。RSS 助手不再存在，而是有一个错误回退。
:::

> **GetStaticPathsRemovedRSSHelper**: RSS 助手已从 `getStaticPaths` 中删除。请改用新的 @astrojs/rss 包。

## 哪里出了问题？
`getStaticPaths` 不再提供生成 RSS 摘要的助手函数。 我们建议迁移到 [@astrojs/rss](/zh-cn/recipes/rss/#设置-astrojsrss) 集成。

**请参阅：**
-  [添加 RSS 摘要](/zh-cn/recipes/rss/)
