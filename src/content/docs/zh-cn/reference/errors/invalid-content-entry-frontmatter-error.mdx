> **错误信息示例：**<br/>
**blog** → **post.md** 的 frontmatter 与集合模式不匹配。<br/>
需要 "title"。<br/>
"date" 必须是有效的日期。

## 哪里发生了错误？

在 `src/content/` 目录下的 Markdown 或 MDX 条目与其集合模式不匹配。
确保所有必需的字段都存在，并且所有字段都是正确的类型。
请检查 `src/content/config.*` 文件中的集合模式。
更多信息，请参阅[内容集合](/zh-cn/guides/content-collections/)
