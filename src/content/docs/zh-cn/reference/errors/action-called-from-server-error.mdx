> **ActionCalledFromServerError**: 在服务器页面或端点中调用了 action，但没有使用 `Astro.callAction()`。该包装器必须用于从服务器代码调用 action。

## 哪里出了问题？

在服务器页面或端点中调用了 action，但没有使用 `Astro.callAction()`。

-  [`Astro.callAction()` 参考](/zh-cn/reference/api-reference/#callaction)
