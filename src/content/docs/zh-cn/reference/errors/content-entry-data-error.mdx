> **错误信息示例：**<br/>
**blog** → **post** data 不匹配集合 schema。<br/>
"title" 是必需的。<br/>
"date" 必须是一个有效日期。

## 哪里出了问题？
内容条目和它的集合 schema 不匹配。
请确保所有必要字段都存在，且字段类型准确无误。
你可以对照 `src/content.config.*` 文件中的集合 schema 来进行检查。
有关更多信息，请参阅 [内容集合文档](/zh-cn/guides/content-collections/)。
