> **ActionsWithoutServerOutputError**: 创建可调用的后端函数需要一个服务器。要将路由部署到服务器，请在你的 Astro 配置中添加一个适配器，并将路由配置为按需渲染。

## 出了什么问题？
你的项目必须有服务器输出，才能用 Actions 创建后端函数。
