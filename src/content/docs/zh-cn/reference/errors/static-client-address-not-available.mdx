> **StaticClientAddressNotAvailable**: `Astro.clientAddress` 仅在服务器端渲染时可用。

## 哪里出了问题？

只有在启用[服务端渲染](/zh-cn/guides/on-demand-rendering/)时，`Astro.clientAddress` 属性才可用。

如要在静态模式下获取用户的 IP 地址，可以在[客户端脚本](/zh-cn/guides/client-side-scripts/) 中使用类似于 [Ipify](https://www.ipify.org/) 等不同的 API，或者可以使用托管在服务提供商处的无服务器 (serverless) 函数来获取用户的 IP。

**请参阅：**

- [在项目中启用 SSR](/zh-cn/guides/on-demand-rendering/)
- [Astro.clientAddress](/zh-cn/reference/api-reference/#clientaddress)
