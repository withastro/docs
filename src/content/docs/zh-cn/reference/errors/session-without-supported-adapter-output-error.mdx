该错误已在 Astro 5.7 中被移除，会话功能已不再是实验性功能。
:::

> **SessionWithoutSupportedAdapterOutputError**: 会话（session）需要一个支持服务器输出的适配器。该适配器必须在其 `buildOutput` 功能中设置为 `"server"`。

## 哪里出了问题？
你的适配器必须支持服务器输出以使用会话。
