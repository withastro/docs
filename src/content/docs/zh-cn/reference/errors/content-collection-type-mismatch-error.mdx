> **ContentCollectionTypeMismatchError**：集合 COLLECTION 包含了 EXPECTED_TYPE 类型的条目，但是它被配置为 ACTUAL_TYPE 类型的集合。

## 哪里发生了错误？

旧版的内容集合必须包含所配置的类型。默认类型为 `type: 'content'`。试着为数据集合添加 `type: 'data'` 的类型。

- [旧版内容集合](/zh-cn/guides/upgrade-to/v5/#升级现有的集合)
