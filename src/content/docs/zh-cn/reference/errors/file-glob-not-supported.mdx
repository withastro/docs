> **FileGlobNotSupported**：在 `file` 加载器中，不支持 Glob 模式导入。请使用 `glob` 加载器替代。

## 哪里出了问题？

`file` 加载器必须传入单个本地文件。其不支持 Glob 模式。使用内置的 `glob` 加载器以从多个本地文件的模式中创建条目。

-  [Astro 内置的加载器](/zh-cn/guides/content-collections/#内置的加载器)
