> **CantUseAstroConfigModuleError**: 无法导入模块 "MODULE_NAME"，因为实验性功能已禁用。在你的 `astro.config.mjs` 中启用 `experimental.serializeConfig`

## 哪里出了问题？
如果不启用实验性功能，则无法使用模块 `astro:config`。
