<p>

**Type :** `boolean`<br />
**Par défaut :** `false`<br />
<Since v="4.14.0" />
</p>

Active les fonctions d'Intellisense (par exemple la complétion de code, les conseils rapides) pour les entrées de votre collection de contenu dans les éditeurs compatibles.

Lorsqu'elle est activée, cette fonctionnalité génère et ajoute des schémas JSON dans le répertoire `.astro` de votre projet. Ces fichiers peuvent être utilisés par le serveur de langage Astro pour fournir une Intellisense à l'intérieur des fichiers de contenu (`.md`, `.mdx`, `.mdoc`).

```js
{
  experimental: {
    contentIntellisense: true,
  },
}
```

Pour utiliser cette fonctionnalité avec l'extension Astro VS Code, vous devez également activer l'option `astro.content-intellisense` dans les paramètres de VS Code. Pour les éditeurs utilisant directement le serveur de langue Astro, passez le paramètre d'initialisation `contentIntellisense: true` pour activer cette fonctionnalité.
