:::caution[Déprécié]
Déprécié depuis Astro 4.0. L'aide RSS n'existe plus avec un retour d'erreur.
:::


> **GetStaticPathsRemovedRSSHelper**: The RSS helper has been removed from `getStaticPaths`. Try the new @astrojs/rss package instead.

## Qu'est-ce qui a mal tourné ? \{#what-went-wrong\}
`getStaticPaths` n'expose plus d'aide pour générer un flux RSS. Nous recommandons de migrer vers l'intégration [@astrojs/rss](/fr/guides/rss/#setting-up-astrojsrss) à la place.
