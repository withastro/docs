> Route returned a `RETURNED_VALUE`. Only a Response can be returned from Astro files.

## Qu'est-ce qui a mal tourné ?
Seules les instances de [Response](https://developer.mozilla.org/fr-FR/docs/Web/API/Response) peuvent être renvoyées dans les fichiers Astro.
```astro title="pages/login.astro"
---
return new Response(null, {
 status: 404,
 statusText: 'Not found'
});

// Pour les redirections, Astro.redirect renvoie également une instance de Response
return Astro.redirect('/login');
---
```
