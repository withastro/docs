> `Astro.cache` is not available because the cache feature is not enabled.

## Qu'est-ce qui a mal tourné ?
Émise lorsque `Astro.cache` ou `context.cache` est utilisé mais que la fonctionnalité de cache n'a pas été activée dans la configuration d'Astro.
