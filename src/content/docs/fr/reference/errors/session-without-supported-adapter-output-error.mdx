:::caution[Obsolète]
Cette erreur a été supprimée dans Astro 5.7, lorsque la fonctionnalité Sessions a cessé d'être expérimentale.
:::

> **SessionWithoutSupportedAdapterOutputError**: Sessions require an adapter that supports server output. The adapter must set `"server"` in the `buildOutput` adapter feature.

## Qu'est-ce qui a mal tourné ?
Votre adaptateur doit prendre en charge la sortie du serveur pour utiliser les sessions.
