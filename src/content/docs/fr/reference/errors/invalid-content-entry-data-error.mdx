> **Example error message:**<br/>
**blog** → **post** frontmatter does not match collection schema.<br/>
"title" is required.<br/>
"date" must be a valid date.

## Qu'est-ce qui a mal tourné ?
Une entrée de contenu ne correspond pas à son schéma de collection.
Assurez-vous que tous les champs obligatoires sont présents et que tous les champs utilisent le bon type.
Vous pouvez vérifier le schéma de collection dans votre fichier `src/content.config.*`.
Consultez la [documentation des collections de contenu](/fr/guides/content-collections/) pour plus d'informations.
