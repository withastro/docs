> **ContentSchemaContainsSlugError**: A content collection schema should not contain `slug` since it is reserved for slug generation. Remove this from your COLLECTION_NAME collection schema.

## Qu'est-ce qui a mal tourné ?
Un schéma de collection de contenu hérité ne doit pas contenir le champ `slug`. Celui-ci est réservé par Astro pour générer les slugs des entrées. Supprimez `slug` de votre schéma. Vous pouvez toujours utiliser des slugs personnalisés dans votre frontmatter.
