> **Exemples de messages d'erreur :**<br/>
InvalidComponentArgs: Invalid arguments passed to `<MyAstroComponent>` component.

## Qu'est-ce qui a mal tourné ?
Les composants Astro ne peuvent pas être rendus manuellement via un appel de fonction, comme `Component()` ou `{items.map(Component)}`. Préférez la syntaxe de composant `<Composant />` ou `{items.map(item => <Composant {...item} />)}`.
