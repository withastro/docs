> **ContentCollectionTypeMismatchError** : COLLECTION contains EXPECTED_TYPE entries, but is configured as a ACTUAL_TYPE collection.

## Qu'est-ce qui a mal tourné ?
Les collections de contenu héritées doivent contenir des entrées du type configuré. Par défaut, les collections ont `type: 'content'`. Essayez d'ajouter `type: 'data'` à la configuration de votre collection pour les collections de données.
