:::caution[Obsolète]
Cette erreur a été supprimée dans Astro 5.7, lorsque la fonctionnalité Sessions a cessé d'être expérimentale.
:::

> Session config was provided without enabling the `experimental.session` flag

## Qu'est-ce qui a mal tourné ?
Émis lorsque le stockage de session est configuré mais que l'option `experimental.session` n'est pas activée.
