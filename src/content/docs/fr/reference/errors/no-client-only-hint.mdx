> **NoClientOnlyHint**: Unable to render `COMPONENT_NAME`. When using the `client:only` hydration strategy, Astro needs a hint to use the correct renderer.

## Qu'est-ce qui a mal tourné ?
Les composants `client:only` ne sont pas exécutés sur le serveur, de ce fait Astro ne sait pas (et ne peut pas deviner) quel moteur de rendu utiliser et requiert une indication. Par exemple :

```astro
<SomeReactComponent client:only="react" />
```
