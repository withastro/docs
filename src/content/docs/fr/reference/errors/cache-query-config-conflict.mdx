> `query.include` and `query.exclude` cannot be used together.

## Qu'est-ce qui a mal tourné ?
Les options `query.include` et `query.exclude` du fournisseur de cache mémoire sont mutuellement exclusives.
Utilisez `include` pour autoriser des paramètres de requête spécifiques qui affectent la clé du cache, ou `exclude` pour
bloquer des paramètres. Lorsque `include` est définie, tous les autres paramètres sont automatiquement ignorés.
