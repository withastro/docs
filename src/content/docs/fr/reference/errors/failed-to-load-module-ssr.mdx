> **FailedToLoadModuleSSR**: Could not import `IMPORT_NAME`.

## Qu'est-ce qui a mal tourné ?
Astro n'a pas pu importer le fichier demandé. Souvent, cela est dû à un chemin d'importation erroné (soit parce que le fichier n'existe pas, soit parce qu'il y a une faute de frappe dans le chemin).

Ce message peut également apparaître lorsqu'un type est importé sans spécifier qu'il s'agit d'une [importation de type](/fr/guides/typescript/#importations-de-types).
