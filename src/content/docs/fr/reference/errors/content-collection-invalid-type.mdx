> **Exemple de message d'erreur :**<br/>
Invalid collection type "data". Remove the type from your collection definition in your content config file.

## Qu'est-ce qui a mal tourné ?
Les collections de contenu ne doivent plus comporter de champ `type`. Supprimez ce champ de votre fichier de configuration de contenu.
Consultez le [guide de migration Astro 6](/fr/guides/upgrade-to/v6/#suppression-collections-de-contenu-héritées) pour plus d'informations.
