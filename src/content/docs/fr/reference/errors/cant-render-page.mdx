> **CantRenderPage**: Astro cannot find any content to render for this route. There is no file or redirect associated with this route.

## Qu'est-ce qui a mal tourné ?
Astro n'a pas pu trouver de fichier associé avec du contenu en essayant de restituer la route. Il s'agit d'une erreur d'Astro et non d'une erreur de l'utilisateur. Si le redémarrage du serveur de développement ne résout pas le problème, veuillez créer un ticket.
