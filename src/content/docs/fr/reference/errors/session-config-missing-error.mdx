:::caution[Dépréciée]
Cette erreur a été supprimée dans Astro 5.7, lorsque la fonctionnalité Sessions a cessé d'être expérimentale.
:::

> The `experimental.session` flag was set to `true`, but no storage was configured. Either configure the storage manually or use an adapter that provides session storage

## Qu'est-ce qui a mal tourné ?
Émise lorsque le stockage de session est activé mais non configuré.
