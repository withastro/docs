:::caution[Dépréciée]
Les collections qui n'existent pas n'entraînent plus d'erreur. Un avertissement est donné à la place.
:::

> A collection queried via `getCollection()` does not exist.

## Qu'est-ce qui a mal tourné ?
Lors de l'interrogation d'une collection, assurez-vous qu'un répertoire de collection portant le nom demandé existe dans `src/content/`.
