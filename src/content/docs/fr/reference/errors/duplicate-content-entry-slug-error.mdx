> `COLLECTION_NAME` contains multiple entries with the same slug: `SLUG`. Slugs must be unique.

## Qu'est-ce qui a mal tourné ?
Les entrées de collection de contenu doivent avoir des slugs uniques. Les doublons sont souvent causés par la propriété `slug` du frontmatter.
