> **Exemple de messages d'erreur :**<br/>
The loader for **blog** returned invalid data.<br/>
Object is missing required property "id".

## Qu'est-ce qui a mal tourné ?
Le chargeur d'une collection de contenu a renvoyé des données non valides.
Les chargeurs incorporés au document doivent renvoyer un tableau d'objets avec des champs d'ID uniques ou un objet simple avec des ID comme noms de propriétés et des entrées comme valeurs.
