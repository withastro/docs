> **Exemple de messages d'erreur :**<br/>
can not read an implicit mapping pair; a colon is missed<br/>
unexpected end of the stream within a double quoted scalar<br/>
can not read a block mapping entry; a multiline key may not be an implicit key

## Qu'est-ce qui a mal tourné ?

Astro a rencontré une erreur lors de l'analyse du frontmatter de votre fichier Markdown.
Cela est souvent dû à une erreur de syntaxe, telle que l'absence de deux-points ou d'un guillemet final manquant.
