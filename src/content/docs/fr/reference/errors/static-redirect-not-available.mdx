> **StaticRedirectNotAvailable**: Redirects are only available when using `output: 'server'` or `output: 'hybrid'`. Update your Astro config if you need SSR features.

## Qu'est-ce qui a mal tourné ?
La fonction `Astro.redirect` n'est disponible que lorsque [Server-side-rendering](/fr/guides/on-demand-rendering/) est activé.

Pour rediriger un site web statique, l'attribut [meta refresh attribute](https://developer.mozilla.org/fr-FR/docs/Web/HTML/Element/meta) peut être utilisé. Certains hébergeurs proposent également des redirections basées sur la configuration (ex : [Netlify redirects](https://docs.netlify.com/routing/redirects/)).
