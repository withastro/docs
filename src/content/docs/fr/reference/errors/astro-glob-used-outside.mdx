> **AstroGlobUsedOutside**: `Astro.glob(GLOB_STR)` can only be used in `.astro` files. `import.meta.glob(GLOB_STR)` can be used instead to achieve a similar result.

## Qu'est-ce qui a mal tourné ?
`Astro.glob()` n'est utilisable que dans les fichiers `.astro`. Vous pouvez utiliser [`import.meta.glob()`](https://vite.dev/guide/features.html#glob-import) à la place pour obtenir le même résultat.
