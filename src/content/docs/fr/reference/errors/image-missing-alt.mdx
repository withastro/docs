> **ImageMissingAlt**: Image missing "alt" property. "alt" text is required to describe important images on the page.

## Qu'est-ce qui a mal tourné ?
La propriété `alt` vous permet de fournir un texte alternatif descriptif aux utilisateurs de lecteurs d'écran et d'autres technologies d'assistance. Afin de garantir l'accessibilité de vos images, le composant `Image` exige qu'un `alt` soit spécifié.

Si l'image est simplement décorative (c'est-à-dire qu'elle ne contribue pas à la compréhension de la page), définissez `alt=""` pour que les lecteurs d'écran sachent qu'ils doivent ignorer l'image.
