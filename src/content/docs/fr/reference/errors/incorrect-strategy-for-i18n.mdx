> **MissingMiddlewareForInternationalization**: The function `FUNCTION_NAME` can only be used when the `i18n.routing.strategy` is set to `"manual"`.

## Qu'est-ce qui n'a pas fonctionné ?
Certaines fonctions d'internationalisation ne sont disponibles que lorsque le routage i18n d'Astro est désactivé par le paramètre de configuration `i18n.routing: "manual"`.
