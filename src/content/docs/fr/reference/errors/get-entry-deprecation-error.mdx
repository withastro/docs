> **GetEntryDeprecationError**: The `METHOD` function is deprecated and cannot be used to query the "COLLECTION" collection. Use `getEntry` instead.

## Qu'est-ce qui a mal tourné ?
Les fonctions `getDataEntryById` et `getEntryBySlug` sont obsolètes et ne peuvent pas être utilisées avec les collections utilisant les couches de contenu. Utilisez plutôt la fonction `getEntry`.
