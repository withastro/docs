:::caution[Dépréciée]
Cette erreur n'est plus spécifique à Markdown et a été remplacée par `ImageNotFound`.
:::

> Could not find requested image `IMAGE_PATH` at `FULL_IMAGE_PATH`.

## Qu'est-ce qui a mal tourné ?
Astro n'a pas pu trouver une image que vous avez incluse dans votre contenu Markdown. En général, il s'agit simplement d'une faute de frappe dans le chemin d'accès.

Les images en Markdown sont relatives au fichier courant. Pour faire référence à une image située dans le même dossier que le fichier `.md`, le chemin doit commencer par `./`
