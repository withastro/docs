> **InvalidImageService**: There was an error loading the configured image service. Please see the stack trace for more information.

## Qu'est-ce qui a mal tourné ?
Une erreur s'est produite lors du chargement du service d'images configuré. Cette erreur peut être due à différents facteurs, comme le fait que votre service d'images n'exporte pas correctement un objet compatible dans son exportation par défaut, ou un chemin d'accès incorrect.

Si vous pensez que votre service est correctement configuré et que cette erreur est erronée, veuillez [ouvrir un ticket](https://astro.build/issues/).
