> `COLLECTION_NAME` → `ENTRY_ID` has an invalid slug. `slug` must be a string.

## Qu'est-ce qui a mal tourné ?
Une entrée de collection a un `slug` invalide. Ce champ est réservé pour générer des slugs d'entrée, et doit être une chaîne de caractères lorsqu'il est présent.
