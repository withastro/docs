La référence suivante est une liste complète des erreurs que vous pouvez rencontrer lors de l'utilisation d'Astro. Pour obtenir une assistance supplémentaire, y compris les pièges courants, veuillez également consulter notre [Guide de dépannage](/fr/guides/troubleshooting/).

## Erreurs d'Astro

- [**UnknownCompilerError**](/fr/reference/errors/unknown-compiler-error/)<br/>Erreur inconnue du compilateur.
- [**StaticRedirectNotAvailable**](/fr/reference/errors/static-redirect-not-available/)<br/>`Astro.redirect` n'est pas disponible en mode statique.
- [**ClientAddressNotAvailable**](/fr/reference/errors/client-address-not-available/)<br/>`Astro.clientAddress` n'est pas disponible dans l'adaptateur actuel.
- [**StaticClientAddressNotAvailable**](/fr/reference/errors/static-client-address-not-available/)<br/>`Astro.clientAddress` n'est pas disponible en mode statique.
- [**NoMatchingStaticPathFound**](/fr/reference/errors/no-matching-static-path-found/)<br/>Aucun chemin statique n'a été trouvé pour le chemin demandé.
- [**OnlyResponseCanBeReturned**](/fr/reference/errors/only-response-can-be-returned/)<br/>Type non valide renvoyé par la page Astro.
- [**MissingMediaQueryDirective**](/fr/reference/errors/missing-media-query-directive/)<br/>Valeur manquante pour la directive `client:media`.
- [**NoMatchingRenderer**](/fr/reference/errors/no-matching-renderer/)<br/>Aucun moteur de rendu correspondant n'a été trouvé.
- [**NoClientEntrypoint**](/fr/reference/errors/no-client-entrypoint/)<br/>Aucun point d'entrée client n'est spécifié dans le moteur de rendu.
- [**NoClientOnlyHint**](/fr/reference/errors/no-client-only-hint/)<br/>Indication manquante sur la directive `client:only`.
- [**InvalidGetStaticPathParam**](/fr/reference/errors/invalid-get-static-path-param/)<br/>Valeur invalide renvoyée par un chemin `getStaticPaths`.
- [**InvalidGetStaticPathsEntry**](/fr/reference/errors/invalid-get-static-paths-entry/)<br/>Entrée invalide dans la valeur de retour de getStaticPath
- [**InvalidGetStaticPathsReturn**](/fr/reference/errors/invalid-get-static-paths-return/)<br/>Valeur invalide renvoyée par getStaticPaths.
- [**GetStaticPathsRemovedRSSHelper**](/fr/reference/errors/get-static-paths-removed-rsshelper/)<br/>getStaticPaths RSS helper n'est plus disponible.
- [**GetStaticPathsExpectedParams**](/fr/reference/errors/get-static-paths-expected-params/)<br/>Propriété params manquante sur la route `getStaticPaths`.
- [**GetStaticPathsInvalidRouteParam**](/fr/reference/errors/get-static-paths-invalid-route-param/)<br/>Valeur non valide pour le paramètre de route `getStaticPaths`.
- [**GetStaticPathsRequired**](/fr/reference/errors/get-static-paths-required/)<br/>`getStaticPaths()` est nécessaire pour les itinéraires dynamiques.
- [**ReservedSlotName**](/fr/reference/errors/reserved-slot-name/)<br/>Nom du Slot invalide.
- [**NoAdapterInstalled**](/fr/reference/errors/no-adapter-installed/)<br/>Impossible d'utiliser le rendu côté serveur sans adaptateur.
- [**NoMatchingImport**](/fr/reference/errors/no-matching-import/)<br/>Aucune importation trouvée pour le composant.
- [**InvalidPrerenderExport**](/fr/reference/errors/invalid-prerender-export/)<br/>Exportation de prérendu invalide.
- [**InvalidComponentArgs**](/fr/reference/errors/invalid-component-args/)<br/>Arguments de composant non valides.
- [**PageNumberParamNotFound**](/fr/reference/errors/page-number-param-not-found/)<br/>Le numéro de page param est introuvable.
- [**ImageMissingAlt**](/fr/reference/errors/image-missing-alt/)<br/>Propriété alt manquante.
- [**InvalidImageService**](/fr/reference/errors/invalid-image-service/)<br/>Erreur lors du chargement du service d'image.
- [**MissingImageDimension**](/fr/reference/errors/missing-image-dimension/)<br/>Dimenssions d'image manquantes
- [**UnsupportedImageFormat**](/fr/reference/errors/unsupported-image-format/)<br/>Format d'image non supporté
- [**UnsupportedImageConversion**](/fr/reference/errors/unsupported-image-conversion/)<br/>Converion d'image non supportée.
- [**PrerenderDynamicEndpointPathCollide**](/fr/reference/errors/prerender-dynamic-endpoint-path-collide/)<br/>Le point de terminaison dynamique généré entre en collision avec une autre route.
- [**ExpectedImage**](/fr/reference/errors/expected-image/)<br/>S'attendait à ce que src soit une image.
- [**ExpectedImageOptions**](/fr/reference/errors/expected-image-options/)<br/>Options d'image attendues.
- [**IncompatibleDescriptorOptions**](/fr/reference/errors/incompatible-descriptor-options/)<br/>Impossible de définir à la fois les `densités` et les `largeurs`
- [**ImageNotFound**](/fr/reference/errors/image-not-found/)<br/>Image non trouvée.
- [**NoImageMetadata**](/fr/reference/errors/no-image-metadata/)<br/>Impossible de traiter les métadonnées de l'image.
- [**MarkdownImageNotFound**](/fr/reference/errors/markdown-image-not-found/)<br/>Image non trouvée.
- [**ResponseSentError**](/fr/reference/errors/response-sent-error/)<br/>Impossible d'établir la réponse.
- [**MiddlewareNoDataOrNextCalled**](/fr/reference/errors/middleware-no-data-or-next-called/)<br/>Le middleware n'a pas retourné de réponse ou appelé `next`.
- [**MiddlewareNotAResponse**](/fr/reference/errors/middleware-not-aresponse/)<br/>The middleware a renvoyé quelque chose qui n'est pas un objet `Response`.
- [**LocalsNotAnObject**](/fr/reference/errors/locals-not-an-object/)<br/>La valeur attribuée à `locals` est refusée.
- [**LocalImageUsedWrongly**](/fr/reference/errors/local-image-used-wrongly/)<br/>Les images importées par ESM doivent être transmises telles quelles.
- [**AstroGlobUsedOutside**](/fr/reference/errors/astro-glob-used-outside/)<br/>Astro.glob() utilisé en dehors d'un fichier Astro
- [**AstroGlobNoMatch**](/fr/reference/errors/astro-glob-no-match/)<br/>Astro.glob() ne correspond à aucun fichier.
- [**RedirectWithNoLocation**](/fr/reference/errors/redirect-with-no-location/)<br/>Une redirection doit recevoir un emplacement avec l'en-tête `Location`.
- [**InvalidDynamicRoute**](/fr/reference/errors/invalid-dynamic-route/)<br/>Route dynamique invalide.
- [**MissingSharp**](/fr/reference/errors/missing-sharp/)<br/>N'a pas pu trouver Sharp.
- [**UnknownViteError**](/fr/reference/errors/unknown-vite-error/)<br/>Erreur Vite inconnue.
- [**FailedToLoadModuleSSR**](/fr/reference/errors/failed-to-load-module-ssr/)<br/>Impossible d'importer le fichier.
- [**InvalidGlob**](/fr/reference/errors/invalid-glob/)<br/>Pattern globale invalide.
- [**FailedToFindPageMapSSR**](/fr/reference/errors/failed-to-find-page-map-ssr/)<br/>Astro n'a pas pu trouver la bonne page à rendre

## Erreurs de Collection de Contenu

- [**UnknownContentCollectionError**](/fr/reference/errors/unknown-content-collection-error/)<br/>Erreur de collecte de contenu inconnu.
- [**InvalidContentEntryFrontmatterError**](/fr/reference/errors/invalid-content-entry-frontmatter-error/)<br/>Le frontmatter du contenu ne correspond pas au schéma.
- [**InvalidContentEntrySlugError**](/fr/reference/errors/invalid-content-entry-slug-error/)<br/>Le Slug du contenu n'est pas valide.
- [**ContentSchemaContainsSlugError**](/fr/reference/errors/content-schema-contains-slug-error/)<br/>Le schéma de contenu ne doit pas contenir `slug`.
- [**CollectionDoesNotExistError**](/fr/reference/errors/collection-does-not-exist-error/)<br/>La collection n'existe pas.
- [**MixedContentDataCollectionError**](/fr/reference/errors/mixed-content-data-collection-error/)<br/>Le contenu et les données ne peuvent pas être dans la même collection.
- [**ContentCollectionTypeMismatchError**](/fr/reference/errors/content-collection-type-mismatch-error/)<br/>La collection contient des entrées d'un type différent.
- [**DataCollectionEntryParseError**](/fr/reference/errors/data-collection-entry-parse-error/)<br/>L'analyse de l'entrée de la collection de données a échoué.
- [**DuplicateContentEntrySlugError**](/fr/reference/errors/duplicate-content-entry-slug-error/)<br/>Entrée Slug du contenu est dupliqué
- [**UnsupportedConfigTransformError**](/fr/reference/errors/unsupported-config-transform-error/)<br/>Transformation non prise en charge dans la configuration du contenu.
