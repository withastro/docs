Le module `astro/hono` fournit des gestionnaires de [routage avancé](/fr/guides/routing/#routage-avancé) construits comme des enveloppes compatibles avec Hono.

## Importations depuis `astro/hono`

```ts
import {
  FetchState,
  astro,
  actions,
  cache,
  i18n,
  middleware,
  pages,
  redirects,
  sessions,
  trailingSlash,
} from "astro/hono";
```

Le module `astro/hono` exporte les mêmes noms de gestionnaires que [`astro/fetch`](/fr/reference/modules/astro-fetch/), mais chacun renvoie une fonction de middleware Hono. Cela vous permet de mélanger les gestionnaires Astro avec n'importe quel middleware Hono de l'écosystème.

<ReadMore>Découvrez comment [utiliser Hono](/fr/guides/routing/#utilisation-avec-hono) avec Astro dans le guide de routage avancé.</ReadMore>
