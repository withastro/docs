:::caution[Supprimée]
L'intégration Astro DB a été dépréciée dans la version 6.4 et a été supprimée depuis la v7.0 d'Astro.

Si vous utilisiez cette intégration dans votre projet, nous vous recommandons de migrer vers une bibliothèque tierce pour les fonctionnalités de base de données. Consultez le [guide de migration vers Astro v7.0](/fr/guides/upgrade-to/v7/#removed-astrojsdb) pour plus de détails et des recommandations sur la manière de mettre à jour votre projet.
:::
