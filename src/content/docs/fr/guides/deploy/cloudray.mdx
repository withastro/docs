Vous pouvez déployer votre projet Astro à l'aide de [CloudRay](https://cloudray.io), une plateforme centralisée qui vous aide à gérer vos serveurs, à organiser les scripts Bash et à automatiser les tâches de déploiement sur les machines virtuelles et les serveurs cloud.

:::note
CloudRay lui-même n'héberge pas votre site. Au lieu de cela, il fournit des outils d'automatisation pour exécuter des scripts de déploiement sur votre propre infrastructure (par exemple, des serveurs Ubuntu) à l'aide d'un agent connecté.
:::

## Prérequis 

Pour commencer, vous aurez besoin de :

- Un [compte CloudRay](https://app.cloudray.io)
- Le code de votre application enregistré dans un dépôt [GitHub](https://github.com/)

## Comment déployer via le tableau de bord CloudRay

Le déploiement avec CloudRay implique généralement trois étapes principales :

<Steps>
1. Installez l'[agent CloudRay](https://cloudray.io/docs/agent) sur votre serveur pour enregistrer votre machine en toute sécurité et activer l'automatisation à distance.

2. Dans votre tableau de bord CloudRay, écrivez un script Bash réutilisable qui clone votre dépôt Astro, installe les dépendances, crée votre site et configure un serveur Web. Définissez toutes les valeurs spécifiques au dépôt à l’aide des [groupes de variables de CloudRay](https://cloudray.io/docs/variable-groups).

3. Utilisez l’interface Runlog de CloudRay pour exécuter votre script sur votre serveur connecté et surveiller le déploiement en temps réel.
</Steps>
