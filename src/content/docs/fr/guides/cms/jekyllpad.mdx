[JekyllPad](https://www.jekyllpad.com) est un CMS léger, fonctionnant dans le navigateur, qui se connecte directement à votre dépôt GitHub.

Il fournit un éditeur WYSIWYG + Markdown moderne, enregistre les modifications directement dans votre dépôt et fonctionne à 100% côté client, vous permettant ainsi de gérer le contenu d'Astro sans configurer de serveurs ni apprendre Git.

Il s'agit d'une application côté client qui s'exécute intégralement dans le navigateur de l'utilisateur. Cela signifie que les données, les jetons d'authentification OAuth et de sécurité restent tous dans le navigateur de l'utilisateur. Aucune installation n'est requise ; il fonctionne avec tous les navigateurs mobiles et de bureau.
