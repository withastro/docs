[Sitepins](https://sitepins.com) est un CMS headless basé sur Git pour les sites web créés avec des frameworks modernes comme Astro. Il offre un éditeur WYSIWYG propre, un processus d'édition contrôlé par version et une intégration transparente avec Astro et d'autres SSG.

## Mise en route


1. [Créez un compte Sitepins](https://app.sitepins.com/register).

2. Connectez votre dépôt GitHub qui contient votre projet Astro.

3. Configurez vos dossiers de contenu, de médias et de configuration et commencez à éditer dans l'éditeur visuel.


Une fois connecté, Sitepins synchronisera votre contenu à partir du dossier sélectionné et fournira une interface visuelle pour gérer et publier du contenu avec un contrôle de version Git complet.
