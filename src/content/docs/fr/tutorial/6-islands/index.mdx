Maintenant que vous avez un blog entièrement fonctionnel, ajoutons quelques îles interactives à votre site !

## Où allons-nous ?

Dans cette unité, vous utiliserez les **îles Astro** pour intégrer des composants de framework frontend dans votre site Astro.

Vous allez :
- Ajouter un framework UI, Preact, à votre projet Astro
- Utiliser Preact pour créer un composant de salutation interactif
- Apprendre quand vous pourriez _ne pas_ choisir les îles pour l'interactivité
