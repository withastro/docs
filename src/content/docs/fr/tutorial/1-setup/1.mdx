## Obtenez les outils de développement dont vous aurez besoin

### Terminal

Vous utiliserez une **ligne de commande (terminal)** pour créer votre projet Astro et exécuter des commandes clés pour construire, développer et tester votre site.

Vous pouvez accéder à la ligne de commande via un programme de terminal local adapté à votre système d'exploitation. Les applications courantes incluent **Terminal** (MacOS/Linux), **Command Prompt** (Windows) et **Termux** (Android). L'une de ces applications devrait déjà être installée sur votre machine.

### Node.js

Pour que Astro fonctionne sur votre système, vous devrez également avoir [**Node.js**](https://nodejs.org/fr/) installé, de préférence en version `v18.14.1` ou ultérieure.

Pour vérifier si vous avez déjà une version compatible installée, exécutez la commande suivante dans votre terminal :

```sh
node -v

// Exemple de résultat
v18.14.1
```

Si la commande renvoie un numéro de version supérieur à `v18.14.1`, vous êtes prêt !

Si la commande renvoie un message d'erreur tel que `Command 'node' not found` ou un numéro de version inférieur à `v18.14.1`, vous devrez [installer une version compatible de Node.js](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm).

### Éditeur de code

De plus, vous devrez télécharger et installer un **éditeur de code** pour écrire votre code.

:::tip[nous utiliserons...]
Ce tutoriel utilisera **VS Code**, mais vous pouvez utiliser n'importe quel éditeur adapté à votre système d'exploitation.
:::

1. [Téléchargez et installez VS Code](https://code.visualstudio.com/#alt-downloads) ou un autre éditeur de code de votre choix.
