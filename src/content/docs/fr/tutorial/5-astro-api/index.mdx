Maintenant que vous avez quelques billets de blog, il est temps d'utiliser l'API d'Astro pour travailler avec vos fichiers !

## Ce que la suite vous réserve

Dans cette unité, vous allez donner un coup de fouet à votre blog avec une page d'index, des pages d'étiquettes et un flux RSS.

En chemin, vous apprendrez à utiliser :
- `import.meta.glob()` pour accéder aux données à partir des fichiers de votre projet
- `getStaticPaths()` pour créer plusieurs pages (routes) en une seule fois
- Le paquet RSS d'Astro pour créer un flux RSS
