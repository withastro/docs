<details>
<summary>Comment est ce que j'utilise les cases a cocher en bas de chaque page?</summary>

Vous les cochez!
 
A la fin de chaque page, vous trouverez une liste de cases a cocher, de taches que vous etes deja en mesure de faire. Cocher les cases correspondantes pour visualiser votre progres sur le trackeur du tutoriel

Utiliser le trackeur est optionnel, mais il permet de vous rappeller les etapes deja realisee lorsque vous parcourez le tutoriel en plusieurs visites. Vous pouvez aussi laisser certaines cases a cocher vierges pour vous rappeler des modules que vous voulez relire plus tard.

(Cette information est sauvegardee seulement dans le stockage local de votre naviguateur etn'est disponible nulle part ailleurs. Aucune information n'est ni envoyee a, ni sauvegardee par Astro.)
</details>

<details>
<summary> Le module 1 est constitue de sujets que je connais deja. Puis je les passer?</summary>

Vous pouvez utiliser les lecons dans le [Module 1](/en/tutorial/1-setup/) pour vous rassurer que vous avez tout le necessaire pour votre environnement de developement ainsi que les comptes dont vous aurez besoin pour finaliser le tutoriel. Ce tutoriel vous guidera dans le processus de creation d'une nouveau projet Astro, que vous sauvegarderez sur Github et deploierez sur Netlify.

Si vous [creer un nouveau, projet Astro vide](/en/install/auto/) et etes a l'aise avec votre setup, libre a vous de continuer directement au [Module 2](/en/tutorial/2-pages/) ou vous comencerez a creer de nouvelles pages dans votre projet.
</details>
