Você pode usar [Fleek](http://fleek.xyz/) para publicar um site estático Astro em sua rede descentralizada de ponta otimizada.

Este guia fornece um passo a passo completo para publicar seu site Astro na Fleek usando a interfacee a CLI Fleek.

## Configuração do Projeto

Seu projeto Astro pode ser publicado na Fleek como um site estático.

## Como publicar

Você pode publicar na Fleek através da interface do website ou usando a CLI Fleek (interface de linha de comando).

### Publicando pela Interface Web

<Steps>
1. Crie uma conta [Fleek](https://app.fleek.xyz).

2. Envie seu código para um repositório Git online (GitHub).

3. Importe seu projeto na Fleek.

4. Fleek detectará automaticamente o Astro e você poderá configurar as definições corretas.

5. Sua aplicação será publicada!
</Steps>


### Fleek CLI

<Steps>
1. Instale a CLI Fleek.

    ```bash
    # Você precisa ter o Nodejs >= 18.18.2
    npm install -g @fleek-platform/cli
    ```
2. Faça login na sua conta Fleek a partir do seu terminal.

    ```bash
    fleek login
    ```

3. Execute o comando de build para gerar os arquivos estáticos. Por padrão, esses arquivos estarão localizados no diretório `dist/`.

    ```bash
    npm run build
    ```

4. Inicialize seu projeto. Isso gerará um arquivo de configuração.

    ```bash
    fleek sites init
    ```

5. Você será solicitado a criar um novo Site Fleek ou usar um já existente. Dê um nome ao site e selecione o diretório onde seu projeto está localizado.

6. Publique seu site.

    ```bash
    fleek sites deploy
    ```

</Steps>
