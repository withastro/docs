Agora que você tem algumas postagens do blog, vamos usar a API do Astro para trabalhar com seus arquivos!

## Onde estamos agora?

Você pode encontrar o código para este ponto no tutorial no [GitHub](https://github.com/withastro/blog-tutorial-demo/tree/unit-5/start) ou
[StackBlitz](https://stackblitz.com/github/withastro/blog-tutorial-demo/tree/unit-5/start?file=src/pages/index.astro).

Compare seu código contra esse exemplo, ou se você acabou de se juntar a nós, faça um fork no StackBlitz para começar a codificar junto do seu navegador a partir de agora.

## Para onde vamos?

Nesta unidade, você irá incrementar o seu blog com uma página de índice, uma página de tags e um feed RSS. 

Durante o caminho, você irá aprender como utilizar:
- `Astro.glob()` para acessar dados de arquivos no seu projeto
- `getStaticPaths()` para criar múltiplas páginas (rotas) de uma vez
- O pacote Astro RSS para criar um feed RSS
