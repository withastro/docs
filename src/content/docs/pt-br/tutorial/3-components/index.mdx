Agora que você tem arquivos `.astro` e `.md` gerando páginas inteiras no seu website, vamos fazer e reutilizar pequenos pedaços de HTML com componentes Astro!

## Onde estamos agora?

Você pode encontrar o código para este ponto no tutorial no [GitHub](https://github.com/withastro/blog-tutorial-demo/tree/unit-3/start) ou
[StackBlitz](https://stackblitz.com/github/withastro/blog-tutorial-demo/tree/unit-3/start?file=src/pages/index.astro).

Compare seu código contra esse exemplo, ou se você acabou de se juntar a nós, faça um fork no StackBlitz para começar a codificar junto do seu navegador a partir de agora.

## Para onde vamos?

Nesta unidade, você irá aprender como criar **componentes Astro** para reutilizar código entre elementos comuns através do seu website. 

Você irá construir:
- Um componente de Navegação que apresenta um menu de links para suas páginas 
- Um componente de Rodapé para incluir no fim de cada página
- Um componente de Rede Social, usado no rodapé, para links de páginas de perfil
- Um componente interativo Hamburger para mostrar a Navegação em dispositivos móveis

Durante o caminho, você utilizará CSS e JavaScript para construir um design responsivo que reage a tamanhos de tela e a interação do usuário.
