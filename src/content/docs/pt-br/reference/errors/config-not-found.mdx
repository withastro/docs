> **ConfigNotFound**: Impossível resolver `--config "ARQUIVO_CONFIG"`. O arquivo existe?

## O que deu errado?

O arquivo de configuração especificado utilizando `--config` não pôde ser encontrado. Certifique-se de que ele existe ou de que o caminho está correto.
