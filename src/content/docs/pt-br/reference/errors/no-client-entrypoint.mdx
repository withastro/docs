> **NoClientEntrypoint**: Componente `NOME_COMPONENTE` tem uma diretiva `client:DIRETIVA_CLIENTE`, mas nenhum entrypoint do cliente foi fornecido por `NOME_RENDERIZADOR`.

## O que deu errado?

Astro tentou hidratar um componente no cliente, mas o renderizador utilizado não fornece um entrypoint do cliente para utilizar para hidratação.
