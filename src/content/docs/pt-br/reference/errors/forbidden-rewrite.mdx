> **ForbiddenRewrite**: Você tentou reescrever a rota sob demanda 'FROM' com uma rota estática 'TO', usando a saída 'server'. <br/><br/>A rota estática 'TO' é processada pelo componente
'COMPONENT', que está marcado como pré-processado. Essa é uma operação proibida porque durante a construção o componente 'COMPONENT' é compilado em um
arquivo HTML, que não pode ser obtido em tempo de execução pelo Astro.

## O que deu errado?
`Astro.rewrite()` não pode ser usado para reescrever uma rota sob demanda com uma rota estática ao usar a saída `"server"`.
