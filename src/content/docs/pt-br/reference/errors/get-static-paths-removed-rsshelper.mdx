:::caution[Descontinuado]
Descontinuado desde o Astro 4.0. O utilitário de RSS não existe mais com um erro de fallback.
:::

> **GetStaticPathsRemovedRSSHelper**: O utilitário de RSS foi removido de `getStaticPaths`. Tente o novo pacote @astrojs/rss no lugar.

## O que deu errado?

`getStaticPaths` não expõe mais um utilitário para gerar um feed RSS. Nós recomendamos migrar para a integração [@astrojs/rss](/pt-br/recipes/rss/#setting-up-astrojsrss) no lugar.
