> **NoMatchingStaticPathFound**: Um padrão de rota `getStaticPaths()` foi correspondido, mas nenhum caminho estático correspondente foi encontrado para o caminho requisitado `NOME_CAMINHO`.

## O que deu errado?

Uma [rota dinâmica](/pt-br/guides/routing/#rotas-dinâmicas) foi correspondida, mas nenhum caminho correspondente foi encontrado para o parâmetros requisitados. Isso é geralmente causado por um erro de digitação no caminho requisitado ou gerado.
