> **FailedToLoadModuleSSR**: Não foi possível importar `NOME_IMPORT`.

## O que deu errado?

O Astro não podê importar o arquivo pedido. Frequentemente, isso é causado pelo caminho da importação estar errado (seja pelo arquivo não existir ou por um erro na digitação do caminho)

Essa mensagem também pode aparecer quando um tipo é importado sem especificar de que é uma [importação de tipo](/pt-br/guides/typescript/#type-imports).
