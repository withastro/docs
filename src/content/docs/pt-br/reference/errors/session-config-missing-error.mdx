> A opção `experimental.session` foi definida como `true`, mas nenhum armazenamento foi configurado. Configure o armazenamento manualmente ou use um adaptador que forneça armazenamento de sessão

## O que deu errado?
Lançado quando armazenamento de sessão está habilitado mas não configurado.
