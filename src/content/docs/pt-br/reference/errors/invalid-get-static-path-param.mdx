> **InvalidGetStaticPathParam**: Parâmetros inválidos dados ao caminho de `getStaticPaths`. Esperava um `object`, recebeu um `TIPO_PARÂMETRO`

## O que deu errado?

A propriedade `params` no valor retornado de `getStaticPaths` (um array de objetos) também deve ser um objeto.

```astro title="pages/blog/[id].astro"
---
export async function getStaticPaths() {
	return [
		{ params: { slug: "blog" } },
		{ params: { slug: "sobre" } }
	];
}
---
```
