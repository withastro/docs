> **InvalidFrontmatterInjectionError**: Um plugin remark ou rehype tentou injetar frontmatter inválido. Garanta que "astro.frontmatter" é definido como um objeto JSON válido que não é `null` nem `undefined`.

## O que deu errado?

Um plugin remark ou rehype tentou injetar frontmatter inválido. Isso ocorre quando "astro.frontmatter" é definido como `null`, `undefined` ou como um objeto JSON inválido.
