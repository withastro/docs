> **SessionWithoutSupportedAdapterOutputError**: Sessões exigem um adaptador que suporte saída do servidor. O adaptador deve definir `"server"` na funcionalidade `buildOutput` do adaptador.

## O que deu errado?
Seu adaptador deve suportar saída do servidor para usar sessões.
