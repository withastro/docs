> **InvalidGlob**: Padrão de glob inválido: `PADRÃO_GLOB`. Padrões de glob devem começar com './', '../' ou '/'.

## O que deu errado?

Astro encontrou um padrão de glob inválido. Isso é geralmente causado pelo padrão de glob não ser um caminho de arquivo válido.
