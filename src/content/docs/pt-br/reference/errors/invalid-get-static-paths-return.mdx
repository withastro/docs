> **InvalidGetStaticPathsReturn**: Tipo inválido retornado por `getStaticPaths`. Esperava uma `array`, recebeu um `TIPO_RETORNO`

## O que deu errado?

O valor retornado por `getStaticPaths` deve ser um array de objetos.

```ts title="pages/blog/[id].astro"
export async function getStaticPaths() {
	return [ // <-- Array
		{ params: { slug: "blog" } },
		{ params: { slug: "about" } }
	];
}
```
