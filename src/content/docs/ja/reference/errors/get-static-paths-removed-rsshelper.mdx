> **GetStaticPathsRemovedRSSHelper**: RSSヘルパーは`getStaticPaths`から削除されました。代わりに新しい@astrojs/rssパッケージを試してみてください。

## 何が問題か？

`getStaticPaths`はRSSフィードを生成するためのヘルパーを公開しなくなりました。代わりに[@astrojs/rss](/ja/guides/rss/#astrojsrssの準備)インテグレーションに移行することをおすすめします。
