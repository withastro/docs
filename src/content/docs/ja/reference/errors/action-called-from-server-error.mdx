> **ActionCalledFromServerError**: サーバーページまたはエンドポイントから`Astro.callAction()`を使わずにActionが呼び出されています。サーバーコードからActionを呼び出す場合は、このラッパーを使用する必要があります。

## 何が問題か？
サーバーページまたはエンドポイントから`Astro.callAction()`を使わずにActionが呼び出されています。
