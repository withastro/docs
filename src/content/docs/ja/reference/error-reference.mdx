以下のリファレンスは、Astroを使用している際に遭遇する可能性のあるエラーの完全なリストです。よくある落とし穴など、その他のサポートが必要であれば、[トラブルシューティングガイド](/ja/guides/troubleshooting/)も参照してください。

## Astroエラー

- [**UnknownCompilerError**](/ja/reference/errors/unknown-compiler-error/)<br/>Unknown compiler error.
- [**StaticRedirectNotAvailable**](/ja/reference/errors/static-redirect-not-available/)<br/>`Astro.redirect` is not available in static mode.
- [**ClientAddressNotAvailable**](/ja/reference/errors/client-address-not-available/)<br/>`Astro.clientAddress` is not available in current adapter.
- [**StaticClientAddressNotAvailable**](/ja/reference/errors/static-client-address-not-available/)<br/>`Astro.clientAddress` is not available in static mode.
- [**NoMatchingStaticPathFound**](/ja/reference/errors/no-matching-static-path-found/)<br/>No static path found for requested path.
- [**OnlyResponseCanBeReturned**](/ja/reference/errors/only-response-can-be-returned/)<br/>Invalid type returned by Astro page.
- [**MissingMediaQueryDirective**](/ja/reference/errors/missing-media-query-directive/)<br/>Missing value for `client:media` directive.
- [**NoMatchingRenderer**](/ja/reference/errors/no-matching-renderer/)<br/>No matching renderer found.
- [**NoClientEntrypoint**](/ja/reference/errors/no-client-entrypoint/)<br/>No client entrypoint specified in renderer.
- [**NoClientOnlyHint**](/ja/reference/errors/no-client-only-hint/)<br/>Missing hint on `client:only` directive.
- [**InvalidGetStaticPathParam**](/ja/reference/errors/invalid-get-static-path-param/)<br/>Invalid value returned by a `getStaticPaths` path.
- [**InvalidGetStaticPathsEntry**](/ja/reference/errors/invalid-get-static-paths-entry/)<br/>Invalid entry inside getStaticPath's return value
- [**InvalidGetStaticPathsReturn**](/ja/reference/errors/invalid-get-static-paths-return/)<br/>Invalid value returned by getStaticPaths.
- [**GetStaticPathsRemovedRSSHelper**](/ja/reference/errors/get-static-paths-removed-rsshelper/)<br/>getStaticPaths RSS helper is not available anymore.
- [**GetStaticPathsExpectedParams**](/ja/reference/errors/get-static-paths-expected-params/)<br/>Missing params property on `getStaticPaths` route.
- [**GetStaticPathsInvalidRouteParam**](/ja/reference/errors/get-static-paths-invalid-route-param/)<br/>Invalid value for `getStaticPaths` route parameter.
- [**GetStaticPathsRequired**](/ja/reference/errors/get-static-paths-required/)<br/>`getStaticPaths()` function required for dynamic routes.
- [**ReservedSlotName**](/ja/reference/errors/reserved-slot-name/)<br/>Invalid slot name.
- [**NoAdapterInstalled**](/ja/reference/errors/no-adapter-installed/)<br/>Cannot use Server-side Rendering without an adapter.
- [**NoMatchingImport**](/ja/reference/errors/no-matching-import/)<br/>No import found for component.
- [**InvalidPrerenderExport**](/ja/reference/errors/invalid-prerender-export/)<br/>Invalid prerender export.
- [**InvalidComponentArgs**](/ja/reference/errors/invalid-component-args/)<br/>Invalid component arguments.
- [**PageNumberParamNotFound**](/ja/reference/errors/page-number-param-not-found/)<br/>Page number param not found.
- [**ImageMissingAlt**](/ja/reference/errors/image-missing-alt/)<br/>Missing alt property.
- [**InvalidImageService**](/ja/reference/errors/invalid-image-service/)<br/>Error while loading image service.
- [**MissingImageDimension**](/ja/reference/errors/missing-image-dimension/)<br/>Missing image dimensions
- [**UnsupportedImageFormat**](/ja/reference/errors/unsupported-image-format/)<br/>Unsupported image format
- [**PrerenderDynamicEndpointPathCollide**](/ja/reference/errors/prerender-dynamic-endpoint-path-collide/)<br/>Prerendered dynamic endpoint has path collision.
- [**ExpectedImage**](/ja/reference/errors/expected-image/)<br/>Expected src to be an image.
- [**ExpectedImageOptions**](/ja/reference/errors/expected-image-options/)<br/>Expected image options.
- [**MarkdownImageNotFound**](/ja/reference/errors/markdown-image-not-found/)<br/>Image not found.
- [**ResponseSentError**](/ja/reference/errors/response-sent-error/)<br/>Unable to set response.
- [**MiddlewareNoDataOrNextCalled**](/ja/reference/errors/middleware-no-data-or-next-called/)<br/>The middleware didn't return a response or call `next`.
- [**MiddlewareNotAResponse**](/ja/reference/errors/middleware-not-aresponse/)<br/>The middleware returned something that is not a `Response` object.
- [**LocalImageUsedWrongly**](/ja/reference/errors/local-image-used-wrongly/)<br/>ESM imported images must be passed as-is.
- [**LocalsNotAnObject**](/ja/reference/errors/locals-not-an-object/)<br/>Value assigned to `locals` is not accepted.
- [**AstroGlobUsedOutside**](/ja/reference/errors/astro-glob-used-outside/)<br/>Astro.glob() used outside of an Astro file.
- [**AstroGlobNoMatch**](/ja/reference/errors/astro-glob-no-match/)<br/>Astro.glob() did not match any files.
- [**RedirectWithNoLocation**](/ja/reference/errors/redirect-with-no-location/)<br/>A redirect must be given a location with the `Location` header.
- [**InvalidDynamicRoute**](/ja/reference/errors/invalid-dynamic-route/)<br/>Invalid dynamic route.
- [**MissingSharp**](/ja/reference/errors/missing-sharp/)<br/>Could not find Sharp.
- [**UnknownViteError**](/ja/reference/errors/unknown-vite-error/)<br/>Unknown Vite Error.
- [**FailedToLoadModuleSSR**](/ja/reference/errors/failed-to-load-module-ssr/)<br/>Could not import file.
- [**InvalidGlob**](/ja/reference/errors/invalid-glob/)<br/>Invalid glob pattern.
- [**FailedToFindPageMapSSR**](/ja/reference/errors/failed-to-find-page-map-ssr/)<br/>Astro couldn't find the correct page to render
