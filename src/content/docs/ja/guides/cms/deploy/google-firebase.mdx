Googleのアプリ開発プラットフォーム[Firebase](https://firebase.google.com/)が提供するサービス[Firebase Hosting](https://firebase.google.com/products/hosting)で、Astroサイトをデプロイできます。

## デプロイ方法

1. [firebase-tools](https://www.npmjs.com/package/firebase-tools)がインストールされていることを確認してください。

2. `firebase.json`と`.firebaserc`をプロジェクトのルートに作成し、次の内容を記入します。

   `firebase.json`:

   ```json
   {
     "hosting": {
       "public": "dist",
       "ignore": []
     }
   }
   ```

   `.firebaserc`:

   ```json
   {
     "projects": {
       "default": "<YOUR_FIREBASE_ID>"
     }
   }
   ```

3. `npm run build`の実行後、`firebase deploy`コマンドを使用してデプロイします。
