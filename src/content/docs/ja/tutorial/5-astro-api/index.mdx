## 概要

このユニットでは、インデックスページ、タグページ、RSSフィードを使ってブログを強化します。

その途中、以下の使い方を学びます。
- プロジェクト内にあるファイルのデータにアクセスするための`import.meta.glob()`
- 一度に複数のページ（ルート）を作成するための`getStaticPaths()`
- RSSフィードを作成するためのAstro RSSパッケージ
