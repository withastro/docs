當開發伺服器執行時，Astro 會在你本機瀏覽器預覽的每個頁面底部加上一條開發工具列。

這條工具列包含多種實用工具，能在開發期間協助你為網站除錯、檢視網站狀況，還可以透過整合目錄中的更多開發工具列應用程式[加以擴充](#擴充開發工具列)。你甚至可以使用[開發工具列 API](/zh-tw/reference/dev-toolbar-app-reference/) 來[建立自己的工具列應用程式](/zh-tw/recipes/making-toolbar-apps/)！

這條工具列預設啟用，當你將滑鼠移至頁面底部時就會出現。它只是個開發工具，不會出現在你發佈的網站上。

## 內建的應用程式

### Astro 選單

Astro 選單應用程式讓你輕鬆存取目前專案的各種資訊，並提供前往其他資源的連結。它最明顯的用處是提供一鍵前往 Astro 文件、GitHub 儲存庫以及 Discord 伺服器的管道。

這個應用程式也包含「Copy debug info」按鈕，會執行 [`astro info`](/zh-tw/reference/cli-reference/#astro-info) 指令，並把輸出複製到剪貼簿。當你請求協助或回報問題時，這個功能會很實用。

### Inspect

Inspect 應用程式提供目前頁面上任何[群島](/zh-tw/concepts/islands/)的資訊。它會顯示傳入每座群島的屬性，以及用來算繪它們的客戶端指令。

### Audit

Audit 應用程式會自動在目前頁面上執行一系列稽核，檢查最常見的效能與無障礙問題。發現問題時，工具列上就會出現一個紅點。點擊這個應用程式會彈出稽核結果清單，並直接在頁面中標示相關的元素。

:::note
開發工具列所執行的基本效能與無障礙稽核，並無法取代 [Pa11y](https://pa11y.org/) 或 [Lighthouse](https://developers.google.com/web/tools/lighthouse) 等專門工具，或是更棒的——真人審核！

開發工具列的目標是在開發期間提供一種快速且方便的方式，讓你捕捉常見問題，不需要切換到另一個工具。
:::

### Settings

Settings 應用程式讓你設定開發工具列的選項，例如詳細紀錄、停用通知，以及調整它在你螢幕上的擺放位置。

## 擴充開發工具列

Astro 整合可以為開發工具列新增應用程式，讓你用針對專案的客製化工具擴充它。你可以在[整合目錄中找到更多可安裝的開發工具列應用程式](https://astro.build/integrations/?search=&categories%5B%5D=toolbar)，或透過 [Astro 選單](#astro-選單)來尋找。

在你的專案中安裝額外的開發工具列應用程式整合，做法就和任何其他 [Astro 整合](/zh-tw/guides/integrations/)一樣，依照各自的安裝說明即可。

## 停用開發工具列

每個網站預設都會啟用開發工具列。你可以依需求在個別專案或個別使用者停用它。

### 依專案停用

要為所有在同一個專案上工作的人停用開發工具列，請在 [Astro 設定檔](/zh-tw/reference/configuration-reference/#devtoolbarenabled)中設定 `devToolbar: false`。

```js title="astro.config.mjs" ins={4-6}
import { defineConfig } from "astro/config";

export default defineConfig({
  devToolbar: {
    enabled: false
  }
});
```

要再次啟用開發工具列，只要從你的設定檔中移除這幾行，或把 `enabled: true` 設回來即可。

### 依使用者停用

若只想為自己在特定專案停用開發工具列，請執行 [`astro preferences`](/zh-tw/reference/cli-reference/#astro-preferences) 指令。

```shell
astro preferences disable devToolbar
```

若要為目前機器上的使用者在所有 Astro 專案中停用開發工具列，執行 `astro-preferences` 時請加上 `--global` 旗標：

```shell
astro preferences disable --global devToolbar
```

之後可以用以下指令重新啟用開發工具列：

```shell
astro preferences enable devToolbar
```
