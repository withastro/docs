透過 rehype 外掛，你可以辨識並修改 Markdown 檔案中指向外部網站的連結。這個範例會在每個外部連結結尾加上圖示，讓訪客知道他們即將離開你的網站。

## 前置需求

- 一個使用 Markdown 作為內容頁面的 Astro 專案。

## 操作指南

<Steps>
1. 安裝 `rehype-external-links` 外掛。

    <PackageManagerTabs>
        <Fragment slot="npm">
        ```shell
        npm install rehype-external-links
        ```
        </Fragment>
        <Fragment slot="pnpm">
        ```shell
        pnpm add rehype-external-links
        ```
        </Fragment>
        <Fragment slot="yarn">
        ```shell
        yarn add rehype-external-links
        ```
        </Fragment>
      </PackageManagerTabs>

2. 在 `astro.config.mjs` 檔案中匯入此外掛。

	將 `rehypeExternalLinks` 連同一個含有 content 屬性的選項物件，傳入 `rehypePlugins` 陣列。如果你想在連結結尾加上純文字，把這個屬性的 `type` 設為 `text`；如果想改加 HTML，則把屬性 `type` 設為 `raw`。

    ```ts
    // ...
    import rehypeExternalLinks from 'rehype-external-links';

    export default defineConfig({
      // ...
      markdown: {
        rehypePlugins: [
          [
            rehypeExternalLinks,
            {
              content: { type: 'text', value: ' 🔗' }
            }
          ],
        ]
      },
    });
    ```
  
    :::note
      `content` 屬性的值[並不會反映在無障礙樹中](https://developer.mozilla.org/en-US/docs/Web/CSS/content#accessibility_concerns)。因此最好在連結周圍的內容裡明確說明這是外部連結，而不要只依賴圖示。
    :::
</Steps>
