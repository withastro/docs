Astro 建置於 Vite 之上，同時支援 Vite 與 Rollup 外掛。這份操作指南使用一個 Rollup 外掛，讓你可以在 Astro 中匯入 YAML（`.yml`）檔案。

## 操作指南

<Steps>
1. 安裝 `@rollup/plugin-yaml`：

    <PackageManagerTabs>
        <Fragment slot="npm">
        ```shell
        npm install @rollup/plugin-yaml --save-dev
        ```
        </Fragment>
        <Fragment slot="pnpm">
        ```shell
        pnpm add @rollup/plugin-yaml --save-dev
        ```
        </Fragment>
        <Fragment slot="yarn">
        ```shell
        yarn add @rollup/plugin-yaml --dev
        ```
        </Fragment>
    </PackageManagerTabs>

2. 在 `astro.config.mjs` 中匯入此外掛，並加入到 Vite 的 plugins 陣列：

    ```js title="astro.config.mjs" ins={2,5-7}
    import { defineConfig } from 'astro/config';
    import yaml from '@rollup/plugin-yaml';

    export default defineConfig({
      vite: {
        plugins: [yaml()]
      }
    });
    ```

3. 最後，你可以使用 `import` 語句來匯入 YAML 資料：

    ```js
    import yml from './data.yml';
    ```

    :::note
    雖然現在可以在 Astro 專案中匯入 YAML 資料，但你的編輯器不會為匯入的資料提供型別。要加入型別，請在專案的 `src` 目錄中建立或找出既有的 `*.d.ts` 檔案，並加入以下內容：
    ```ts title="src/files.d.ts"
    // Specify the file extension you want to import
    declare module "*.yml" {
      const value: any; // Add type definitions here if desired
      export default value;
    }
    ```
    這樣編輯器就能為你的 YAML 資料提供型別提示。
    :::
</Steps>
