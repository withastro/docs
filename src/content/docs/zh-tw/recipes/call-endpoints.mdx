端點可用於提供多種類型的資料。這份操作指南會在頁面的元件腳本中呼叫伺服器端點來顯示一段問候語，無需額外發出 fetch 請求。

## 前置需求

- 已啟用 [SSR](/zh-tw/guides/on-demand-rendering/)（output: 'server'）的專案

## 操作指南

<Steps>
1. 在 `src/pages/api/hello.ts` 新檔案中建立一個會回傳資料的端點：

    ```ts title="src/pages/api/hello.ts"
    import type { APIRoute } from 'astro'

    export const GET: APIRoute = () => {
      return new Response(
        JSON.stringify({
          greeting: 'Hello',
        }),
      )
    }
    ```

2. 在任意一個 Astro 頁面中，從該端點匯入 `GET()` 方法。以 [`Astro` 全域物件](/zh-tw/reference/api-reference/)呼叫，傳入請求情境，並在頁面中使用其回應：

    ```astro title="src/pages/index.astro"
    ---
    import { GET } from './api/hello.ts'

    let response = await GET(Astro)
    const data = await response.json()
    ---

    <h1>{data.greeting} world!</h1>
    ```
</Steps>
