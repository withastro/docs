Mientras el servidor de desarrollo está en ejecución, Astro incluye una barra de herramientas de desarrollo en la parte inferior de cada página en la vista previa local del navegador.

Esta barra de herramientas incluye una serie de herramientas útiles para depurar e inspeccionar tu sitio durante el desarrollo y se puede [extender con más aplicaciones para la barra de herramientas](#extender-la-barra-de-herramientas-de-desarrollo) que se encuentran en el directorio de integraciones. ¡Incluso puedes [crear tus propias aplicaciones para la barra de herramientas](/es/recipes/making-toolbar-apps/) usando la [API de la barra de herramientas de desarrollo](/es/reference/dev-toolbar-app-reference/)!

Esta barra de herramientas está habilitada por defecto y aparece al pasar el cursor sobre la parte inferior de la página. Es solo una herramienta de desarrollo y no aparecerá en tu sitio publicado.

## Aplicaciones integradas

### Menú de Astro

La aplicación Menú de Astro proporciona acceso fácil a diversa información sobre el proyecto actual y enlaces a recursos adicionales. Notablemente, proporciona acceso con un solo clic a la documentación de Astro, al repositorio de GitHub y al servidor de Discord.

Esta aplicación también incluye un botón «Copiar información de depuración» que ejecutará el comando [`astro info`](/es/reference/cli-reference/#astro-info) y copiará la salida a tu portapapeles. Esto puede ser útil cuando se pide ayuda o se reportan problemas.

### Inspeccionar

La aplicación Inspeccionar proporciona información sobre cualquier [island](/es/concepts/islands/) en la página actual. Te mostrará las propiedades pasadas a cada island y la directiva de cliente que se está usando para renderizarlos.

### Auditoría

La aplicación Auditoría ejecuta automáticamente una serie de auditorías en la página actual, verificando los problemas de rendimiento y accesibilidad más comunes. Cuando se encuentra un problema, aparecerá un punto rojo en la barra de herramientas. Al hacer clic en la aplicación, aparecerá una lista de resultados de la auditoría y resaltará los elementos relacionados directamente en la página.

:::note
Las auditorías básicas de rendimiento y accesibilidad realizadas por la barra de herramientas de desarrollo no reemplazan a las herramientas dedicadas como [Pa11y](https://pa11y.org/) o [Lighthouse](https://developers.google.com/web/tools/lighthouse), o incluso mejor, ¡los humanos!

La barra de herramientas de desarrollo tiene como objetivo proporcionar una forma rápida y sencilla de detectar problemas comunes durante el desarrollo, sin necesidad de cambiar de contexto a una herramienta diferente.
:::

### Configuración

La aplicación Configuración te permite configurar opciones para la barra de herramientas de desarrollo, como el registro detallado, la desactivación de notificaciones y el ajuste de su ubicación en la pantalla.

## Extender la barra de herramientas de desarrollo

Las integraciones de Astro pueden añadir nuevas aplicaciones a la barra de herramientas de desarrollo, lo que te permite extenderla con herramientas personalizadas específicas para tu proyecto. Puedes encontrar [más aplicaciones para la barra de herramientas para instalar en el directorio de integraciones](https://astro.build/integrations/?search=&categories%5B%5D=toolbar) o usando el [Menú de Astro](#menú-de-astro).

Instala integraciones adicionales de aplicaciones para la barra de herramientas de desarrollo en tu proyecto como cualquier otra [integración de Astro](/es/guides/integrations/), según sus propias instrucciones de instalación.

## Deshabilitar la barra de herramientas de desarrollo

La barra de herramientas de desarrollo está habilitada por defecto para cada sitio. Puedes optar por deshabilitarla para proyectos individuales y/o usuarios según sea necesario.

### Por proyecto

Para deshabilitar la barra de herramientas de desarrollo para todos los que trabajan en un proyecto, establece `devToolbar: false` en el [archivo de configuración de Astro](/es/reference/configuration-reference/#devtoolbarenabled).

```js title="astro.config.mjs" ins={4-6}
import { defineConfig } from "astro/config";

export default defineConfig({
  devToolbar: {
    enabled: false
  }
});
```

Para volver a habilitar la barra de herramientas de desarrollo, elimina estas líneas de tu configuración o establece `enabled: true`.

### Por usuario

Para deshabilitar la barra de herramientas de desarrollo para ti mismo en un proyecto específico, ejecuta el comando [`astro preferences`](/es/reference/cli-reference/#astro-preferences).

```shell
astro preferences disable devToolbar
```

Para deshabilitar la barra de herramientas de desarrollo en todos los proyectos de Astro para un usuario en la máquina actual, añade la opción `--global` al ejecutar `astro-preferences`:

```shell
astro preferences disable --global devToolbar
```

La barra de herramientas de desarrollo se puede habilitar posteriormente con:

```shell
astro preferences enable devToolbar
```
