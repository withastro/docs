Puedes usar [Fleek](http://fleek.xyz/) para desplegar un sitio Astro estático en su red descentralizada optimizada para el borde.

Esta guía proporciona un recorrido completo para desplegar tu sitio Astro en Fleek utilizando la interfaz de usuario y la CLI de Fleek.

## Configuración del proyecto

Tu proyecto Astro se puede desplegar en Fleek como un sitio estático.

## Cómo desplegar

Puedes desplegar en Fleek a través de la interfaz de usuario del sitio web o utilizando la CLI (interfaz de línea de comandos) de Fleek.

### Despliegue a través de la interfaz de la plataforma

<Steps>
1. Crea una cuenta en [Fleek](https://app.fleek.xyz).

2. Sube tu código a tu repositorio Git en línea (GitHub).

3. Importa tu proyecto a Fleek.

4. Fleek detectará automáticamente Astro y luego podrás configurar los ajustes correctos.

5. ¡Tu aplicación está desplegada!
</Steps>

### CLI de Fleek

<Steps>
1. Instala la CLI de Fleek.

    ```bash
    # Necesitas tener Nodejs >= 18.18.2
    npm install -g @fleek-platform/cli
    ```

2. Inicia sesión en tu cuenta de Fleek desde tu terminal.

    ```bash
    fleek login
    ```

3. Ejecuta el comando de construcción para generar los archivos estáticos. Por defecto, estos se ubicarán en el directorio `dist/`.

    ```bash
    npm run build
    ```

4. Inicializa tu proyecto. Esto generará un archivo de configuración.

    ```bash
    fleek sites init
    ```

5. Se te pedirá que crees un nuevo sitio Fleek o que uses uno existente. Dale un nombre al sitio y selecciona el directorio donde se encuentra tu proyecto.

6. Despliega tu sitio.

    ```bash
    fleek sites deploy
    ```
</Steps>
