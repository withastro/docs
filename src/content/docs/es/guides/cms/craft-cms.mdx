[Craft CMS](https://craftcms.com/) es un CMS flexible de código abierto con una experiencia de edición accesible. Incluye su propio front-end, pero también puede ser usado como CMS headless para proporcionar contenido a tu proyecto de Astro.

## Recursos oficiales

- Echa un vistazo a la [guía oficial de la API GraphQL de Craft CMS](https://craftcms.com/docs/5.x/development/graphql.html)
- La documentación oficial sobre la [configuración `headlessMode` de Craft](https://craftcms.com/docs/5.x/reference/config/general.html#headlessmode)
