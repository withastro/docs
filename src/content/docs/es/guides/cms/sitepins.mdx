[Sitepins](https://sitepins.com) es un CMS headless basado en Git para sitios web creados con frameworks modernos como Astro. Ofrece un editor WYSIWYG limpio, un flujo de trabajo de contenido con control de versiones y una integración perfecta con Astro y otros SSG.

## Cómo empezar

1. [Crea una cuenta en Sitepins](https://app.sitepins.com/register).

2. Conecta tu repositorio de GitHub que contiene tu proyecto de Astro.

3. Configura tus carpetas de contenido, medios y configuración, y empieza a editar en el editor visual.

Una vez conectado, Sitepins sincronizará tu contenido desde la carpeta seleccionada y proporcionará una interfaz visual para gestionar y publicar contenido con control de versiones de Git completo.
