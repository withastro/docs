Ahora que ya tienes un sitio web en funcionamiento, ¡vamos a añadir páginas y posts!

## ¿Dónde estamos ahora?

You can find the code at this point in the tutorial on [GitHub](https://github.com/withastro/blog-tutorial-demo/tree/unit-2/start) or
[StackBlitz](https://stackblitz.com/github/withastro/blog-tutorial-demo/tree/unit-2/start?file=src/pages/index.astro).

Check your code against this example, or if you've just joined us, fork on StackBlitz to start coding along in your browser from here.

## ¿Adónde vamos?

En esta unidad
- Crea tus primeras páginas Astro con la sintaxis `.astro
- Añadir posts con archivos Markdown (`.md`)
- Estilar una página individual con `<style>`.
- Aplicar estilos globales a todas las páginas

En el camino, aprenderás cómo las **dos secciones de un archivo `.astro`** trabajan juntas para crear una página, y cómo usar variables y renderizado condicional en tus páginas.
