Ahora que ya tienes un sitio web en funcionamiento, ¡es hora de añadir páginas y publicaciones!

## Mirando hacia el futuro

En esta unidad
- Crearás tus primeras páginas Astro con la sintaxis `.astro
- Añadirás publicaciones con archivos Markdown (`.md`)
- Estilizarás una página individual con `<style>`.
- Aplicarás estilos globales a todas las páginas

En el camino, aprenderás cómo las **dos secciones de un archivo `.astro`** trabajan juntas para crear una página, y cómo usar variables y renderizado condicional en tus páginas.
