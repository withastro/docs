## Mirando hacia el futuro

En esta unidad, crearás plantillas para compartir elementos y estilos comunes en tus páginas y publicaciones de blog.

Para ello:

- Crea componentes de plantillas reutilizables
- Pasa contenido a tus plantillas con `<slot />`
- Pasa datos de Markdown frontmatter a tus plantillas
- Anida varias plantillas
