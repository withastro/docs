Aquí, conectarás tu repositorio de GitHub a Netlify. Netlify utilizarás ese proyecto para construir y desplegar tu sitio en vivo en la web cada vez que confirmes un cambio en tu código. 

:::tip[Usaremos...]
Este tutorial utilizará **Netlify**, pero te invitamos a utilizar tu servicio de alojamiento preferido para desplegar tu sitio en Internet.
:::

## Crea un nuevo sitio Netlify

<Steps>
1. Crea una cuenta gratuita en [Netlify](https://netlify.com) si aún no tienes una.

    Anota tu nombre de usuario. Verás tu panel de control y cualquiera de los sitios que crees en `https://app.netlify.com/teams/username`.

2. Haz clic en <kbd>Add new site</kbd> > <kbd>Import an existing project</kbd>.

    Se te pedirá que te conectes a un proveedor de Git. Elige GitHub y sigue los pasos que aparecen en pantalla para autenticar tu cuenta de GitHub. A continuación, elige el repositorio GitHub de tu proyecto de Astro de la lista proporcionada.

3. En el último paso, Netlify te mostrará la configuración del sitio de tu aplicación. Los valores predeterminados deben ser correctos para tu proyecto Astro, así que puedes desplazarte hacia abajo y hacer clic en <kbd>Deploy site</kbd>.
</Steps>

Enhorabuena, ¡tienes un sitio web de Astro!

## Cambia el nombre del proyecto

En la página de resumen de tu sitio en Netlify, verás el nombre de tu proyecto generado aleatoriamente, y la URL de tu sitio web de la forma `https://project-name-123456.netlify.app`. Puede cambiar el nombre de tu proyecto por algo más memorable y esto actualizará automáticamente tu URL.

## Visita tu nuevo sitio web

Haz clic en la URL en la configuración de tu sitio o escríbela en una ventana del navegador para ver tu nuevo sitio web.
