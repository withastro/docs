<p>
**Tipo:** `boolean`<br />
**Predeterminado:** `false`<br />
<Since v="4.14.0" />
</p>

Habilita las funciones de Intellisense (por ejemplo, autocompletado de código, sugerencias rápidas) para las entradas de tu colección de contenido en editores compatibles.

Cuando está habilitada, esta función generará y agregará esquemas JSON al directorio `.astro` en tu proyecto. Estos archivos pueden ser utilizados por el servidor de lenguaje Astro para proporcionar Intellisense dentro de los archivos de contenido (`.md`, `.mdx`, `.mdoc`).

```js
{
  experimental: {
    contentIntellisense: true,
  },
}
```

Para usar esta función con la extensión de Astro VS Code, también debes habilitar la opción `astro.content-intellisense` en tu configuración de VS Code. Para los editores que utilizan el servidor de lenguaje Astro directamente, pasa el parámetro de inicialización `contentIntellisense: true` para habilitar esta función.
