> **ActionsUsedWithForGetError**: La acción NOMBRE_DE_ACCIÓN fue llamada desde un formulario usando una solicitud GET, pero solo se admiten solicitudes POST. Esto ocurre a menudo si `method="POST"` no está presente en el formulario.

## ¿Qué salió mal?
La Acción fué llamada desde un formulario usando una solicitud GET, pero solo se admiten solicitudes POST. Esto ocurre a menudo si `method="POST"` no está presente en el formulario.
