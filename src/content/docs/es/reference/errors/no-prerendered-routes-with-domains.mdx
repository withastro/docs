> **NoPrerenderedRoutesWithDomains**: Las páginas estáticas aún no son compatibles con múltiples dominios. Para habilitar esta función, debes deshabilitar el prerenderizado para la página COMPONENT

## ¿Qué salió mal?
Las páginas estáticas aún no son compatibles con múltiples dominios. Si deseas habilitar esta función, debes deshabilitar el prerenderizado.
