> **SessionWithoutSupportedAdapterOutputError**: Las sesiones requieren un adaptador que admita la salida del servidor. El adaptador debe establecer `"server"` en la característica `buildOutput` del adaptador.

## ¿Qué salió mal?
Tu adaptador debe admitir la salida del servidor para usar sesiones.
