> **ForbiddenRewrite**: Intentaste reescribir la ruta de servidor 'DESDE' con la ruta estática 'PARA', cuando se utiliza la salida 'server'. <br/><br/>La ruta estática 'PARA' es renderizada por el componente
'COMPONENTE', qué es marcado como prerenderizado. Esta es una operación prohibida porque durante la construcción del componente 'COMPONENTE' se compila a un
archivo HTML, que no se puede recuperar en tiempo de ejecución por Astro.

## ¿Qué salió mal?
`Astro.rewrite()` no puede ser usado para reescribir una ruta de servidor con una ruta estática cuando se utiliza la salida `"server"`.
