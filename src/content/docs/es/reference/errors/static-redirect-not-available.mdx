> **StaticRedirectNotAvailable**: Los redireccionamientos solo están disponibles cuando se usa `output: 'server'` o `output: 'hybrid'`. Actualiza tu configuración de Astro si necesitas funciones de SSR. (E03001)

## ¿Qué salió mal?

La función `Astro.redirect` solo está disponible cuando [el renderizado en el servidor](/es/guides/server-side-rendering/) está habilitada.

Para redirigir en un sitio web estático, se puede usar el [atributo meta de actualización](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/meta). Ciertos hosts también proporcionan redireccionamientos basados en configuración (por ejemplo, [redireccionamientos de Netlify](https://docs.netlify.com/routing/redirects/)).
