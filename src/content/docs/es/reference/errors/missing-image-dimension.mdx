> Falta el atributo de ancho y alto para `IMAGE_URL`. Cuando se usan imágenes remotas, ambas dimensiones son necesarias para evitar el desplazamiento acumulativo de la capa (CLS).

## ¿Qué salió mal?
Para imágenes remotas, no se pueden inferir automaticamente el `width` y `height` del archivo original. Para evitar el Desplazamiento del Acumulativo del Diseño (CLS), especifica estas dos propiedades o establece `inferSize` en `true` para obtener las dimensiones originales de una imagen remota.

Si tu imagen está dentro de tu carpeta `src`, probablemente quieras importarla en su lugar. Consulta [la guía de importaciones para obtener más información](/es/guides/imports/#other-assets).
