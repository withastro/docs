> Ocurrió un error al determinar el peso y el estilo desde el archivo tipográfico local.

## ¿Qué salió mal?
No se puede determinar el peso y el estilo desde el archivo tipográfico. En su lugar, actualiza la configuración de tu familia y define `weight` y `style` manualmente.
