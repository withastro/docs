> **ExpectedNotESMImage**: Una imagen importada con ESM no se puede pasar directamente a `getImage()`. En su lugar, pasa un objeto con la imagen en la propiedad `src`.

## ¿Qué salió mal?
Una imagen importada con ESM no se puede pasar directamente a `getImage()`. En su lugar, pasa un objeto con la imagen en la propiedad `src`.

```diff
import { getImage } from "astro:assets";
import myImage from "../assets/my_image.png";
- const optimizedImage = await getImage( myImage );
+ const optimizedImage = await getImage({ src: myImage });
```
