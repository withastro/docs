> **FileGlobNotSupported**: Los patrones glob no son compatibles en el cargador `file`. Usa el cargador `glob` en su lugar.

## ¿Qué salió mal?
Al cargador `file` debes pasarle un único archivo local. Los patrones glob no son compatibles. Usa el cargador integrado `glob` para crear entradas a partir de patrones de múltiples archivos locales.
