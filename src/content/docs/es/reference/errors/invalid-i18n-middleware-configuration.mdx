> **InvalidI18nMiddlewareConfiguration**: La opción `redirectToDefaultLocale` solo puede habilitarse cuando `prefixDefaultLocale` también está en `true`; de lo contrario, las redirecciones pueden causar bucles infinitos. Habilita `prefixDefaultLocale` para seguir usando `redirectToDefaultLocale`, o asegúrate de que ambas opciones estén en `false`.

## ¿Qué salió mal?
Se detectó una configuración inválida en el middleware de i18n.
