> **Ejemplo de mensaje de error:**<br/>
El cargador de contenido para la collección **blog** devolvió una entrada con un `id` inválido:<br/>
&#123;<br/>
  "id": 1,<br/>
  "title": "Hola, Mundo!"<br/>
&#125;

## ¿Qué salió mal?
Un cargador de contenido devolvió una entrada con un `id` inválido.
Asegúrate de que el `id` de la entrada sea una cadena.
Consulta la [documentación de colecciones de contenido](/es/guides/content-collections/) para obtener más información.
