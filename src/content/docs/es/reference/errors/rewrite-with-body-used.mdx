> **RewriteWithBodyUsed**: Astro.rewrite() no puede ser usado si el cuerpo de la solicitud ya se ha leído. Si necesitas leer el cuerpo, primero clona la solicitud.

## ¿Qué salió mal?
`Astro.rewrite()` no puede ser usado si el cuerpo de la solicitud ya se ha leído. Si necesitas leer el cuerpo, primero clona la solicitud. Por ejemplo:

```js
const data = await Astro.request.clone().formData();

Astro.rewrite("/target")
```
