:::caution[Obsoleto]
Este error se eliminó en Astro v6.0.0 junto con la eliminación de `Astro.glob()`.
:::

> **AstroGlobUsedOutside**: `Astro.glob(GLOB_STR)` solo puede usarse en archivos `.astro`. En su lugar, puedes usar `import.meta.glob(GLOB_STR)` para lograr un resultado similar.

## ¿Qué salió mal?
`Astro.glob()` solo puede usarse en archivos `.astro`. Puedes usar [`import.meta.glob()`](https://es.vite.dev/guide/features#importaciones-glob) para conseguir el mismo resultado.
