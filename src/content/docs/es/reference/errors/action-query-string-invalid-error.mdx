:::caution[Obsoleto]  
Este error proviene de una versión antigua de Astro y ya no está en uso. Si no puedes actualizar tu proyecto a una versión más reciente, puedes consultar [instantáneas no mantenidas de documentación antigua](/es/upgrade-astro/#documentación-antigua-sin-mantenimiento) para obtener ayuda.  
:::  

> **ActionQueryStringInvalidError**: El servidor recibió la cadena de consulta `?_astroAction=NOMBRE_DE_ACCIÓN`, pero no pudo encontrar una acción con ese nombre. Si cambiaste el nombre de una acción durante el desarrollo, elimina este parámetro de la URL y actualiza la página.

## ¿Qué salió mal?
El servidor recibió la cadena de consulta `?_astroAction=name`, pero no pudo encontrar una acción con ese nombre. Usa la propiedad `.queryString` de la función de acción para obtener la URL de `action` del formulario.
