> No se pudo encontrar la imagen solicitada `IMAGE_PATH` en `FULL_IMAGE_PATH`.

## ¿Qué salió mal?
Astro no pudo encontrar una imagen que incluiste en tu contenido Markdown. Normalmente, esto se debe a un error tipográfico en la ruta.

Las imágenes en Markdown son relativas al archivo actual. Para referirse a una imagen que se encuentra en la misma carpeta que el archivo `.md`, la ruta debe empezar por `./`.
