> **RewriteEncounteredAnError**: La ruta RUTA que intentaste renderizar no existe, o emitió un error durante la fase de renderización. STACK ? STACK : ''.

## ¿Qué salió mal?
El usuario intentó reescribir utilizando una ruta que no existe, o emitió un error de tiempo de ejecución durante su fase de renderización.
