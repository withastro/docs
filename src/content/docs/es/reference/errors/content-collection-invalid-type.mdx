> **Mensaje de error de ejemplo:**<br/>
Tipo de colección no válido "data". Elimina el campo type de la definición de tu colección en el archivo de configuración de contenido.

## ¿Qué salió mal?
Las colecciones de contenido ya no deben tener un campo `type`. Elimina este campo de tu archivo de configuración de contenido.
Consulta la [guía de migración de Astro 6](/es/guides/upgrade-to/v6/#removed-legacy-content-collections) para más información.
