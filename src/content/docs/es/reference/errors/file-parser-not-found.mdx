> **FileParserNotFound**: No se encontró ningún analizador para 'FILE_NAME'. Pase una función de analizador (por ejemplo, `parser: csv`) al cargador `file`.

## ¿Qué salió mal?
El cargador `file` no puede determinar qué analizador utilizar. Proporcione un analizador personalizado (por ejemplo, `csv-parse`) para crear una colección a partir de su tipo de archivo.
