> **ActionsReturnedInvalidDataError**: El controlador de la Acción devolvió datos no válidos. Los controladores deben devolver tipos de datos serializables como objetos, arrays, cadenas y números. Error de análisis: ERROR

## ¿Qué salió mal?  
El controlador de la acción devolvió datos no válidos. Los controladores deben devolver tipos de datos serializables y no pueden devolver un objeto Response.
