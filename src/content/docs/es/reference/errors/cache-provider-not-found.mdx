> No se pudo resolver el proveedor de caché `PROVIDER`. Asegúrate de que el paquete esté instalado.

## ¿Qué salió mal?
Se lanza cuando no se puede resolver el proveedor de caché configurado. Normalmente significa que el paquete no está instalado o que la ruta de importación es incorrecta.
