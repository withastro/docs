> `query.include` y `query.exclude` no pueden usarse juntos.

## ¿Qué salió mal?
Las opciones `query.include` y `query.exclude` del proveedor de caché en memoria son mutuamente excluyentes.
Usa `include` para permitir explícitamente parámetros de consulta que afectan a la clave de caché, o `exclude` para bloquear parámetros.
Cuando se establece `include`, todos los demás parámetros se ignoran automáticamente.
