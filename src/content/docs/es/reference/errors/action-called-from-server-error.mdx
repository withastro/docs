> **ActionCalledFromServerError**: Se llamó una acción desde una página o endpoint del servidor sin usar `Astro.callAction()`. Debes usar este wrapper para llamar acciones desde código de servidor.

## ¿Qué salió mal?
Se llamó una acción desde una página o endpoint del servidor sin usar `Astro.callAction()`.
