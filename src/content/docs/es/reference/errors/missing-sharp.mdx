> **MissingSharp**: No se pudo encontrar Sharp. Instala Sharp (`sharp`) manualmente en tu proyecto o migra a otro servicio de imagen.

## ¿Qué salió mal?

Sharp es el servicio de imagen predeterminado utilizado para `astro:assets`. Cuando se utiliza un [administrador de paquetes estricto](https://pnpm.io/pnpm-vs-npm#npms-flat-tree) como pnpm, Sharp debe instalarse manualmente en tu proyecto para utilizar el procesamiento de imágenes.

Si no estás usando `astro:assets` para procesar imágenes y no deseas instalar Sharp, puedes configurar el siguiente servicio de imagen para que no realice ningún procesamiento:

```js
import { defineConfig, passthroughImageService } from "astro/config";
export default defineConfig({
 image: {
   service: passthroughImageService(),
 },
});
```
