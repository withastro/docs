> **NoAdapterInstalledServerIslands**: No se puede utilizar islas de servidor sin un adaptador. Instala y configura el adaptador adecuado para tu despliegue final.

## ¿Qué salió mal?
Para utilizar las islas de servidor, existen las mismas restricciones que para el renderizado del lado del servidor, por lo que se necesita un adaptador.
