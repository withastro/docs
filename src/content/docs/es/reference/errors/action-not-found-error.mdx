> **ActionNotFoundError**: El servidor recibió una solicitud para una acción llamada `NOMBRE_DE_ACCIÓN`, pero no pudo encontrar una coincidencia. Si renombraste una acción, verifica que hayas actualizado tu archivo `actions/index` y el código que la llama para que coincidan.

## ¿Qué salió mal?
El servidor recibió una solicitud para una acción pero no pudo encontrar una coincidencia con el mismo nombre.
