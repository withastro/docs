> La bandera `experimental.session` se estableció en `true`, pero no se ha configurado el almacenamiento de sesión. Configura manualmente el almacenamiento de sesión o utilices un adaptador que proporcione el almacenamiento de sesión

## ¿Qué salió mal?
Se lanza cuando el almacenamiento de sesión está habilitado pero no se ha configurado.
