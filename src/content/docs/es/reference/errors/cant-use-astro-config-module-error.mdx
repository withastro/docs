:::caution[Obsoleto]
Este error pertenece a una version anterior de Astro y ya no se utiliza. Si no puedes actualizar tu proyecto a una version mas reciente, consulta las [instantaneas sin mantenimiento de la documentacion antigua](/es/upgrade-astro/#documentación-antigua-sin-mantenimiento).
:::

> **CantUseAstroConfigModuleError**: No se puede importar el módulo "NOMBRE_DE_MÓDULO" porque la característica experimental está deshabilitada. Habilita `experimental.serializeConfig` en tu archivo `astro.config.mjs`

## ¿Qué salió mal?
No puedes usar el módulo `astro:config` sin habilitar la característica experimental.
