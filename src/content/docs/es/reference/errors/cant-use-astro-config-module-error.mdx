> **CantUseAstroConfigModuleError**: No se puede importar el módulo "NOMBRE_DE_MÓDULO" porque la característica experimental está deshabilitada. Habilita `experimental.serializeConfig` en tu archivo `astro.config.mjs`

## ¿Qué salió mal?
No puedes usar el módulo `astro:config` sin habilitar la característica experimental.
