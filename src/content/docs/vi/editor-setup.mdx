Tùy chỉnh trình soạn thảo code để cải thiện trải nghiệm phát triển Astro và mở khóa các tính năng mới.

## VS Code

[VS Code](https://code.visualstudio.com/) là trình soạn thảo code phổ biến cho web developer, được xây dựng bởi Microsoft. Engine VS Code cũng cung cấp năng lượng cho các trình soạn thảo code in-browser phổ biến như [GitHub Codespaces](https://github.com/features/codespaces) và [Gitpod](https://gitpod.io/).

Astro hoạt động với bất kỳ trình soạn thảo code nào. Tuy nhiên, VS Code là trình soạn thảo được chúng tôi khuyến nghị cho các dự án Astro. Chúng tôi duy trì một [Astro VS Code Extension](https://marketplace.visualstudio.com/items?itemName=astro-build.astro-vscode) chính thức mở khóa một số tính năng chính và cải thiện trải nghiệm developer cho các dự án Astro.

- Syntax highlighting cho file `.astro`.
- Thông tin kiểu TypeScript cho file `.astro`.
- [VS Code Intellisense](https://code.visualstudio.com/docs/editor/intellisense) cho code completion, gợi ý và nhiều hơn nữa.

Để bắt đầu, cài đặt [Astro VS Code Extension](https://marketplace.visualstudio.com/items?itemName=astro-build.astro-vscode) ngay hôm nay.

## Zed

[Zed](https://zed.dev/) là trình soạn thảo code mã nguồn mở đã thêm hỗ trợ cho Astro trong phiên bản 0.123.2. Bạn có thể cài đặt [Astro extension](https://github.com/zed-extensions/astro) trong tab Extensions của IDE. Extension này bao gồm các tính năng như syntax highlighting, code completion và formatting.

## JetBrains IDEs

[Webstorm](https://www.jetbrains.com/webstorm/) là một JavaScript và TypeScript IDE đã thêm hỗ trợ cho Astro Language Server trong phiên bản 2024.2. Bản cập nhật này mang đến các tính năng như syntax highlighting, code completion và formatting.

Cài đặt plugin chính thức thông qua [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/20959-astro) hoặc bằng cách tìm kiếm "Astro" trong tab Plugins của IDE. Bạn có thể bật/tắt language server trong `Settings | Languages & Frameworks | TypeScript | Astro`.

Để biết thêm thông tin về hỗ trợ Astro trong Webstorm, xem [tài liệu Webstorm Astro chính thức](https://www.jetbrains.com/help/webstorm/astro.html).

## Các Trình soạn thảo code khác

Cộng đồng tuyệt vời của chúng tôi duy trì một số extension cho các trình soạn thảo phổ biến khác, bao gồm:

- [VS Code Extension trên Open VSX](https://open-vsx.org/extension/astro-build/astro-vscode) <span style="margin: 0.25em;"><Badge text="Official" /></span> - Astro VS Code Extension chính thức, có sẵn trên Open VSX registry cho các nền tảng mở như [VSCodium](https://vscodium.com/)
- [Nova Extension](https://extensions.panic.com/extensions/sciencefidelity/sciencefidelity.astro/) <span style="margin: 0.25em;"><Badge class="neutral-badge" text="Community" /></span> - Cung cấp syntax highlighting và code completion cho Astro bên trong Nova
- [Vim Plugin](https://github.com/wuelnerdotexe/vim-astro) <span style="margin: 0.25em;"><Badge class="neutral-badge" text="Community" /></span> - Cung cấp syntax highlighting, indentation và code folding support cho Astro bên trong Vim hoặc Neovim
- Neovim [LSP](https://github.com/neovim/nvim-lspconfig/blob/master/doc/configs.md#astro) và [TreeSitter](https://github.com/virchau13/tree-sitter-astro) Plugins <span style="margin: 0.25em;"><Badge class="neutral-badge" text="Community" /></span> - Cung cấp syntax highlighting, treesitter parsing và code completion cho Astro bên trong Neovim
- Emacs - Xem hướng dẫn [Cấu hình Emacs và Eglot](https://medium.com/@jrmjrm/configuring-emacs-and-eglot-to-work-with-astro-language-server-9408eb709ab0) <span style="margin: 0.25em;"><Badge class="neutral-badge" text="Community" /></span> để hoạt động với Astro
- [Astro syntax highlighting cho Sublime Text](https://packagecontrol.io/packages/Astro) <span style="margin: 0.25em;"><Badge class="neutral-badge" text="Community" /></span> - Gói Astro cho Sublime Text, có sẵn trên Sublime Text package manager.

## Trình soạn thảo In-Browser

Ngoài các trình soạn thảo local, Astro cũng hoạt động tốt trên các trình soạn thảo hosted in-browser, bao gồm:

- [StackBlitz](https://stackblitz.com/) và [CodeSandbox](https://codesandbox.io/) - trình soạn thảo online chạy trong trình duyệt của bạn, với hỗ trợ syntax highlighting tích hợp cho file `.astro`. Không cần cài đặt hoặc cấu hình!
- [GitHub.dev](https://github.dev/) - cho phép bạn cài đặt Astro VS Code extension như một [web extension](https://code.visualstudio.com/api/extension-guides/web-extensions), giúp bạn truy cập chỉ một số tính năng của extension đầy đủ. Hiện tại, chỉ hỗ trợ syntax highlighting.
- [IDX](https://idx.dev) và [Gitpod](https://gitpod.io/) - môi trường dev đầy đủ trên cloud có thể cài đặt Astro VS Code Extension chính thức từ Open VSX.

## Các công cụ khác

### ESLint

[ESLint](https://eslint.org/) là một linter phổ biến cho JavaScript và JSX. Để hỗ trợ Astro, có thể cài đặt [một plugin được cộng đồng duy trì](https://github.com/ota-meshi/eslint-plugin-astro).

Xem [User Guide của dự án](https://ota-meshi.github.io/eslint-plugin-astro/user-guide/) để biết thêm thông tin về cách cài đặt và thiết lập ESLint cho dự án của bạn.

### Stylelint

[Stylelint](https://stylelint.io/) là một linter phổ biến cho CSS. [Cấu hình Stylelint được cộng đồng duy trì](https://github.com/ota-meshi/stylelint-config-html) cung cấp hỗ trợ Astro.

Hướng dẫn cài đặt, tích hợp editor và thông tin bổ sung có thể được tìm thấy trong README của dự án.

### Prettier

[Prettier](https://prettier.io/) là một formatter phổ biến cho JavaScript, HTML, CSS và nhiều hơn nữa. Nếu bạn đang sử dụng [Astro VS Code Extension](https://marketplace.visualstudio.com/items?itemName=astro-build.astro-vscode) hoặc [Astro language server trong editor khác](#other-code-editors), code formatting với Prettier đã được bao gồm.

Để thêm hỗ trợ formatting file `.astro` bên ngoài editor (ví dụ: CLI) hoặc bên trong các editor không hỗ trợ editor tooling của chúng tôi, cài đặt [Astro Prettier plugin chính thức](https://github.com/withastro/prettier-plugin-astro).

<Steps>
1. Cài đặt `prettier` và `prettier-plugin-astro`.

    <PackageManagerTabs>
      <Fragment slot="npm">
      ```shell
      npm install --save-dev --save-exact prettier prettier-plugin-astro
      ```
      </Fragment>
      <Fragment slot="pnpm">
      ```shell
      pnpm add --save-dev --save-exact prettier prettier-plugin-astro
      ```
      </Fragment>
      <Fragment slot="yarn">
      ```shell
      yarn add --dev --exact prettier prettier-plugin-astro
      ```
      </Fragment>
    </PackageManagerTabs>

2. Tạo file cấu hình `.prettierrc` (hoặc `.prettierrc.json`, `.prettierrc.mjs`, hoặc [các định dạng được hỗ trợ khác](https://prettier.io/docs/configuration)) ở root của dự án và thêm `prettier-plugin-astro` vào đó.

    Trong file này, cũng chỉ định thủ công parser cho file Astro.

    ```json title=".prettierrc"
    {
      "plugins": ["prettier-plugin-astro"],
      "overrides": [
        {
          "files": "*.astro",
          "options": {
            "parser": "astro",
          }
        }
      ]
    }
    ```

3. Tùy chọn, cài đặt các Prettier plugin khác cho dự án của bạn và thêm chúng vào file cấu hình. Các plugin bổ sung này có thể cần được liệt kê theo thứ tự cụ thể. Ví dụ, nếu bạn sử dụng Tailwind, `prettier-plugin-tailwindcss` phải là [Prettier plugin cuối cùng trong plugins array](https://github.com/tailwindlabs/prettier-plugin-tailwindcss#compatibility-with-other-prettier-plugins).

    ```json title=".prettierrc"
    {
      "plugins": [
        "prettier-plugin-astro",
        "prettier-plugin-tailwindcss"
      ],
      "overrides": [
        {
          "files": "*.astro",
          "options": {
            "parser": "astro"
          }
        }
      ]
    }
    ```

4. Chạy lệnh sau trong terminal để format file của bạn.

    <PackageManagerTabs>
      <Fragment slot="npm">
      ```shell
      npx prettier . --write
      ```
      </Fragment>
      <Fragment slot="pnpm">
      ```shell
      pnpm exec prettier . --write
      ```
      </Fragment>
      <Fragment slot="yarn">
      ```shell
      yarn exec prettier . --write
      ```
      </Fragment>
    </PackageManagerTabs>
</Steps>

Xem [README của Prettier plugin](https://github.com/withastro/prettier-plugin-astro/blob/main/README.md) để biết thêm thông tin về các tùy chọn được hỗ trợ, cách thiết lập Prettier trong VS Code và nhiều hơn nữa.
