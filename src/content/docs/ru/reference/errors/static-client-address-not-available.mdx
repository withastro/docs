> **StaticClientAddressNotAvailable**: `Astro.clientAddress` доступен только при использовании `output: 'server'` или `output: 'hybrid'`. Обновите конфигурацию Astro, если вам нужны функции SSR.

## Что пошло не так?

Свойство `Astro.clientAddress` доступно только при включенном [SSR](/ru/guides/on-demand-rendering/).

Для получения IP-адреса пользователя в статическом режиме можно использовать различные API, такие как [Ipify](https://www.ipify.org/), в [скрипте на стороне клиента](/ru/guides/client-side-scripts/) или получить IP пользователя с помощью бессерверной функции, размещенной у вашего хостинг-провайдера.
