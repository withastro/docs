> **MixedContentDataCollectionError**: **COLLECTION_NAME** содержит смешанные записи контента и данных. Все записи должны быть одного типа.

## Что пошло не так?

Коллекция контента не может содержать смешанные записи контента и данных. Вы должны хранить записи в отдельных коллекциях по типу.
