> **FailedToLoadModuleSSR**: Не удалось импортировать `IMPORT_NAME`.

## Что пошло не так?

Astro не удалось импортировать запрошенный файл. Зачастую это происходит из-за неправильного пути импорта (либо потому, что файл не существует, либо в пути допущена опечатка).

Это сообщение также может появиться, когда тип импортируется без указания, что это [импорт типа](/ru/guides/typescript/#type-imports).
