> Невозможно выполнить рендеринг `COMPONENT_NAME`. В вашем файле `astro.config.mjs` настроено `RENDERER_COUNT` рендерера(ов), но ни один из них не смог выполнить серверный рендер для `COMPONENT_NAME`.

## What went wrong?
Ни одна из установленных интеграций не смогла отрендерить импортированный вами компонент. Убедитесь, что установлена соответствующая интеграция для типа компонента, который вы пытаетесь включить на вашей странице.

Для файлов JSX / TSX можно использовать [@astrojs/react](/ru/guides/integrations-guide/react/), [@astrojs/preact](/ru/guides/integrations-guide/preact/) или [@astrojs/solid-js](/ru/guides/integrations-guide/solid-js/). Для файлов Vue и Svelte можно использовать интеграцию [@astrojs/vue](/ru/guides/integrations-guide/vue/) и [@astrojs/svelte](/ru/guides/integrations-guide/svelte/) соответственно.
