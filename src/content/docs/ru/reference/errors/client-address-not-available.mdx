> **ClientAddressNotAvailable**: `Astro.clientAddress` недоступен в адаптере `ADAPTER_NAME`. Подайте запрос на адаптер, чтобы добавить поддержку.

## Что пошло не так?

К сожалению, используемый вами адаптер не поддерживает `Astro.clientAddress`.
