> **GetStaticPathsExpectedParams**: Отсутствующее или пустое обязательное свойство `params` в маршруте `getStaticPaths`.

## Что пошло не так?

Каждому маршруту, указанному в `getStaticPaths`, требуется наличие свойства `params`, указывающего параметры пути, необходимые для сопоставления маршрута.

Например, следующий код:
```astro title="pages/blog/[id].astro"
---
export async function getStaticPaths() {
	return [
		{ params: { id: '1' } }
	];
}
---
```
Создаст следующий маршрут: `site.com/blog/1`.
