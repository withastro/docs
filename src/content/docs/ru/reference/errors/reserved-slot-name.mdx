> **ReservedSlotName**: Невозможно создать слот с именем `SLOT_NAME`. `SLOT_NAME` — это зарезервированное имя слота. Пожалуйста, обновите имя этого слота.

## Что пошло не так?

Некоторые слова не могут быть использованы для имен слотов, так как уже используются внутри системы.
