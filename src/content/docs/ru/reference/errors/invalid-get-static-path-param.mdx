> **InvalidGetStaticPathParam**: Неверные параметры, переданные пути `getStaticPaths`. Ожидался `object`, получен `PARAM_TYPE`

## Что пошло не так?

Свойство `params` возвращаемого значения `getStaticPaths` (массив объектов) также должно быть объектом.

```astro title="pages/blog/[id].astro"
---
export async function getStaticPaths() {
	return [
		{ params: { slug: "blog" } },
		{ params: { slug: "about" } }
	];
}
---
```
