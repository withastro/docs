> **ContentCollectionTypeMismatchError**: COLLECTION содержит записи EXPECTED_TYPE, но настроена как коллекция ACTUAL_TYPE.

## Что пошло не так?

Коллекции контента должны содержать записи заданного типа. По умолчанию коллекции имеют тип `type: 'content'`. Попробуйте добавить `type: 'data'` в конфигурацию коллекции для коллекций данных.
