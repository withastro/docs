> **InvalidGetStaticPathsReturn**: Неверный тип, возвращаемый `getStaticPaths`. Ожидался `массив`, получен `RETURN_TYPE`

## Что пошло не так?

Возвращаемое значение `getStaticPaths` должно быть массивом объектов.

```ts title="pages/blog/[id].astro"
export async function getStaticPaths() {
	return [ // <-- Array
		{ params: { slug: "blog" } },
		{ params: { slug: "about" } }
	];
}
```
