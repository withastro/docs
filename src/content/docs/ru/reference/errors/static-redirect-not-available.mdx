Устарело с версии 2.6.
:::

> **StaticRedirectNotAvailable**: Перенаправления доступны только при использовании `output: 'server'` или `output: 'hybrid'`. Обновите конфигурацию Astro, если вам нужны функции SSR.

## Что пошло не так?
Функция `Astro.redirect` доступна только при включенном [SSR](/ru/guides/on-demand-rendering/).

Для перенаправления на статическом сайте можно использовать [атрибут meta refresh](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/meta). Некоторые хостеры также предоставляют перенаправления на основе конфигурации (например, [перенаправления Netlify](https://docs.netlify.com/routing/redirects/)).
