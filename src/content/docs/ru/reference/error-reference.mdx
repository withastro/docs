Ниже представлен полный список ошибок, с которыми вы можете столкнуться при использовании Astro. Если вам нужна помощь в решении типичных проблем, загляните в наше [Руководство по устранению неполадок](/ru/guides/troubleshooting/).

## Ошибки Astro

- [**UnknownCompilerError**](/ru/reference/errors/unknown-compiler-error/) — Неизвестная ошибка компилятора.
- [**ClientAddressNotAvailable**](/ru/reference/errors/client-address-not-available/) — `Astro.clientAddress` недоступен в текущем адаптере.
- [**PrerenderClientAddressNotAvailable**](/ru/reference/errors/prerender-client-address-not-available/) — `Astro.clientAddress` нельзя использовать внутри предварительно отрендеренных маршрутов.
- [**NoMatchingStaticPathFound**](/ru/reference/errors/no-matching-static-path-found/) — Не найден статический путь для запрошенного адреса.
- [**OnlyResponseCanBeReturned**](/ru/reference/errors/only-response-can-be-returned/) — Неверный тип данных, возвращаемый страницей Astro.
- [**NoAdapterInstalled**](/ru/reference/errors/no-adapter-installed/) — Нельзя использовать SSR без установленного адаптера.
- [**ImageMissingAlt**](/ru/reference/errors/image-missing-alt/) — У изображения отсутствует обязательное свойство "alt".
- [**ImageNotFound**](/ru/reference/errors/image-not-found/) — Изображение не найдено.
- [**InvalidDynamicRoute**](/ru/reference/errors/invalid-dynamic-route/) — Неверный динамический маршрут.
- [**RouteNotFound**](/ru/reference/errors/route-not-found/) — Маршрут не найден.

## Ошибки Markdown

- [**MarkdownFrontmatterParseError**](/ru/reference/errors/markdown-frontmatter-parse-error/) — Ошибка парсинга метаданных Markdown.
- [**MdxIntegrationMissingError**](/ru/reference/errors/mdx-integration-missing-error/) — Отсутствует интеграция MDX.

## Ошибки коллекций контента

- [**InvalidContentEntryDataError**](/ru/reference/errors/invalid-content-entry-data-error/) — Данные записи контента не соответствуют схеме.
- [**DuplicateContentEntrySlugError**](/ru/reference/errors/duplicate-content-entry-slug-error/) — Дублирующийся слаг (slug) записи контента.

## Ошибки действий (Actions)

- [**ActionNotFoundError**](/ru/reference/errors/action-not-found-error/) — Действие не найдено.
- [**ActionsWithoutServerOutputError**](/ru/reference/errors/actions-without-server-output-error/) — Действия требуют включенного серверного вывода.
