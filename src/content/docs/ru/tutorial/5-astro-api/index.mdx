Теперь, когда у вас есть несколько записей в блоге, пришло время использовать API Astro для работы с файлами!

## Забегая вперёд

В этом модуле вы дополните свой блог индексной страницей, страницами тегов и RSS-каналом.

Попутно вы узнаете, как использовать:
- `import.meta.glob()` для доступа к данным из файлов в вашем проекте
- `getStaticPaths()` для создания нескольких страниц (маршрутов) одновременно
- Пакет Astro RSS для создания RSS-ленты
