Теперь, когда у вас есть полностью функционирующий блог, давайте добавим некоторые интерактивные острова на ваш сайт!

## Забегая вперёд

В этом модуле вы будете использовать **острова Astro** для внедрения компонентов фреймворка в ваш сайт на Astro.

Вы сможете:
- Добавить UI-фреймворк, Preact, в ваш Astro-проект
- Использовать Preact для создания интерактивного компонента приветствия
- Узнать, когда можно _не_ выбирать острова для интерактивности
