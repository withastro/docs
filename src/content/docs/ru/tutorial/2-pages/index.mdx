Теперь, когда у вас есть работающий сайт, пришло время добавить страницы и посты!

## Забегая вперёд

В этом модуле вы:
- Создадите свои первые страницы Astro с синтаксисом `.astro`
- Добавите посты в блоге с помощью файлов Markdown (`.md`)
- Стилизуете отдельную страницу с помощью `<style>`
- Примените глобальные стили ко всем страницам

 Попутно вы узнаете, как **две секции файла `.astro`** работают вместе для создания страницы, а также как использовать переменные и условный рендеринг на ваших страницах.
