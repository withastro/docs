Здесь вы подключите ваш репозиторий на GitHub к Netlify. Netlify будет использовать этот проект для сборки и развёртывания вашего сайта в реальном времени на веб-сервере каждый раз, когда вы вносите изменения в свой код.

:::tip[Мы будем использовать…]
В этом учебном пособии мы будем использовать **Netlify**, но вы можете использовать любой другой выбранный вами хостинг-сервис для развёртывания вашего сайта в интернете.
:::

## Создайте новый сайт на Netlify

<Steps>
1. Создайте бесплатную учётную запись на [Netlify](https://netlify.com), если у вас ещё её нет.

    Запишите свое имя пользователя. Вы можете просмотреть свою панель инструментов и любые созданные вами сайты по адресу `https://app.netlify.com/teams/username`

2. Нажмите <kbd>Add new site</kbd> (Добавить новый сайт) > <kbd>Import an existing project</kbd> (Импортировать существующий проект).

    Вам будет предложено подключиться к Git-провайдеру. Выберите GitHub и следуйте инструкциям на экране, чтобы аутентифицировать свою учётную запись GitHub. Затем выберите репозиторий вашего проекта Astro на GitHub из предоставленного списка.

3. На последнем этапе Netlify покажет вам настройки вашего сайта. Параметры по умолчанию должны быть правильными для вашего проекта Astro, поэтому вы можете прокрутить страницу вниз и нажать <kbd>Deploy site</kbd> (Развернуть сайт).
</Steps>

Поздравляем, у вас есть сайт Astro!

## Измените название проекта

На странице обзора вашего сайта в Netlify вы увидите случайно сгенерированное название проекта и URL вашего сайта в формате `https://project-name-123456.netlify.app`. Вы можете изменить название вашего проекта на более запоминающееся, и это автоматически обновит ваш URL.

## Посетите ваш новый сайт

Щелкните по URL-адресу в настройках сайта или введите его в окно браузера, чтобы просмотреть ваш новый сайт.
