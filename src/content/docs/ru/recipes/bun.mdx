[Bun](https://bun.sh/) — это современная среда выполнения JavaScript и набор инструментов «все в одном».

:::caution
Использование Bun с Astro может иметь свои нюансы, так как некоторые интеграции могут работать не так, как ожидается. Сверяйтесь с [официальным руководством Bun по работе с Astro](https://bun.sh/guides/ecosystem/astro).
:::

## Создание нового проекта с Bun

Для создания проекта используйте следующую команду:

```bash
bun create astro my-astro-project
```

## Установка зависимостей

Если вы пропустили этот шаг при создании проекта, выполните:

```bash
bun install
```

## Типизация для Bun

Для поддержки типов Bun в редакторе установите пакет `@types/bun`:

```bash
bun add -d @types/bun
```

## Разработка и сборка

Команды для запуска сервера разработки и создания итоговой сборки:

```bash
# Запуск сервера
bun run dev

# Сборка проекта
bun run build

# Предварительный просмотр сборки
bun run preview
```

## Тестирование

Bun содержит встроенный и быстрый инструмент для тестирования, совместимый с Jest, который запускается командой `bun test`. Также вы можете использовать любые другие [инструменты для тестирования Astro](/ru/guides/testing/).
