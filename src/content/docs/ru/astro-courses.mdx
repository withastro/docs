Хотите начать изучение Astro с помощью курса или руководства?

Вы можете освоить основы Astro с помощью нашего [официального руководства «Создание блога»](/ru/tutorial/0-introduction/), либо изучить нашу подборку рекомендованного образовательного контента по Astro.


## Партнёры по обучению


:::tip[Поддержите Astro во время обучения]
Используйте партнёрские ссылки Astro для получения скидок у наших образовательных партнёров, и часть вашей покупки пойдёт непосредственно на финансирование разработки проекта Astro с открытым исходным кодом!
:::
