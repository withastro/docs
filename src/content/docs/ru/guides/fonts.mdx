Это руководство покажет вам, как добавить веб-шрифты в ваш проект и использовать их в компонентах.

:::tip[Fonts API (экспериментально)]
Узнайте об [экспериментальном Fonts API](/ru/reference/experimental-flags/fonts/), который позволяет использовать шрифты из файловой системы и от различных провайдеров через унифицированный и типобезопасный интерфейс.
:::

## Использование локального файла шрифта

В этом примере мы добавим шрифт с использованием файла `DistantGalaxy.woff`.

<Steps>
1. Поместите файл шрифта в папку `public/fonts/`.
2. Добавьте правило `@font-face` в ваш CSS. Это можно сделать в глобальном `.css` файле или блоке `<style is:global>`.

    ```css
    @font-face {
      font-family: 'DistantGalaxy';
      src: url('/fonts/DistantGalaxy.woff') format('woff');
      font-weight: normal;
      font-style: normal;
      font-display: swap;
    }
    ```

3. Используйте имя семейства шрифтов из правила `@font-face` для стилизации элементов:

    ```astro title="src/pages/example.astro"
    <style>
    h1 {
      font-family: 'DistantGalaxy', sans-serif;
    }
    </style>
    ```
</Steps>

## Использование Fontsource

Проект [Fontsource](https://fontsource.org/) упрощает использование Google Fonts и других открытых шрифтов.

<Steps>
1. Найдите нужный шрифт в [каталоге Fontsource](https://fontsource.org/). Мы будем использовать [Twinkle Star](https://fontsource.org/fonts/twinkle-star).
2. Установите пакет шрифта:

    <PackageManagerTabs>
      <Fragment slot="npm">
      ```shell
      npm install @fontsource/twinkle-star
      ```
      </Fragment>
      <Fragment slot="pnpm">
      ```shell
      pnpm add @fontsource/twinkle-star
      ```
      </Fragment>
      <Fragment slot="yarn">
      ```shell
      yarn add @fontsource/twinkle-star
      ```
      </Fragment>
    </PackageManagerTabs>

3. Импортируйте пакет в компоненте макета:

    ```astro title="src/layouts/BaseLayout.astro"
    ---
    import '@fontsource/twinkle-star';
    ---
    ```

4. Используйте имя шрифта в CSS:

    ```css
    h1 {
      font-family: "Twinkle Star", cursive;
    }
    ```
</Steps>

## Регистрация шрифтов в Tailwind

Если вы используете [Tailwind](/ru/guides/styling/#tailwind), вы можете зарегистрировать шрифт в файле `src/styles/global.css`:

```css title="src/styles/global.css"
@import 'tailwindcss';

@theme {
  --font-sans: 'Inter', 'sans-serif'; 
}
```

Теперь стандартный класс Tailwind `font-sans` будет использовать выбранный вами шрифт.
